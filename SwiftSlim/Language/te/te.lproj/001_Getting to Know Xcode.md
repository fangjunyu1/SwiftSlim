# Xcode ను పరిచయం చేసుకోవడం

## Xcode అంటే ఏమిటి

Xcode అనేది Apple అధికారికంగా అందించే సమగ్ర అభివృద్ధి వాతావరణం (IDE, Integrated Development Environment). ఇది iOS, macOS, watchOS, tvOS వంటి Apple ప్లాట్‌ఫార్మ్‌ల కోసం అప్లికేషన్లు అభివృద్ధి చేయడానికి ఉపయోగించబడుతుంది.

![Swift](../../RESOURCE/001_xcode.png)

మొత్తం బోధనా ప్రక్రియలో మనం కోడ్ రాయడానికి, ప్రోగ్రామ్‌లు నడపడానికి Xcode నే ఉపయోగిస్తాము.

ఈ పాఠంలో, Xcode ను ఎలా డౌన్‌లోడ్ చేయాలి, మొదటి iOS అప్లికేషన్‌ను ఎలా సృష్టించాలి, అలాగే Xcode ఇంటర్‌ఫేస్ నిర్మాణాన్ని ప్రాథమికంగా ఎలా అర్థం చేసుకోవాలి అన్నది నేర్చుకుంటాము.

## Xcode డౌన్‌లోడ్ చేయడం

ప్రస్తుతం Xcode ను ఇన్‌స్టాల్ చేయడానికి రెండు విధానాలు ఉన్నాయి:

1、[Mac App Store](https://apps.apple.com/cn/app/xcode/id497799835?mt=12) ద్వారా డౌన్‌లోడ్ చేయడం

2、[Apple Developer అధికారిక వెబ్‌సైట్](https://developer.apple.com/xcode/) ద్వారా డౌన్‌లోడ్ చేయడం

ఈ రెండు మార్గాల్లో కూడా Xcode ను డౌన్‌లోడ్ చేయవచ్చు. తేడా ఏమిటంటే, Apple Developer అధికారిక వెబ్‌సైట్‌లో అధికారిక విడుదలలతో పాటు [Beta](https://developer.apple.com/download/all/?q=Xcode) వెర్షన్లు, చరిత్రాత్మక పాత వెర్షన్లు కూడా లభిస్తాయి.

మీరు కేవలం అభ్యాసం కోసం నేర్చుకుంటున్నట్లయితే, అధికారిక స్థిర వెర్షన్ ఉపయోగించడం మంచిది. Beta వెర్షన్‌లు ప్రధానంగా కొత్త ఫీచర్లను పరీక్షించడానికి ఉపయోగిస్తారు; అవి స్థిరంగా లేకపోవచ్చు.

## మొదటి ప్రాజెక్ట్ సృష్టించడం

Xcode ను తెరిచిన తర్వాత, ప్రారంభ స్క్రీన్ కనిపిస్తుంది.

![Swift](../../RESOURCE/001_xcode1.png)

ఎడమవైపు సాధారణంగా ఉపయోగించే మూడు ఎంపికలు ఉంటాయి:

* Create New Project —— కొత్త ప్రాజెక్ట్ సృష్టించు
* Clone Git Repository —— Git రిపోజిటరీని క్లోన్ చేయు
* Open Existing Project —— ఇప్పటికే ఉన్న ప్రాజెక్ట్‌ను తెరువు

కుడివైపు, ఇటీవల తెరిచిన Xcode ప్రాజెక్ట్‌లు కనిపిస్తాయి.

మనము "Create New Project" ఎంపికను ఎంచుకుంటాము.

### ప్రాజెక్ట్ టెంప్లేట్‌ను ఎంచుకోవడం

![Swift](../../RESOURCE/001_xcode2.png)

ఇప్పుడు ప్రాజెక్ట్ టెంప్లేట్ ఎంపిక స్క్రీన్‌లోకి వెళ్తాము. ఇక్కడ వివిధ ప్లాట్‌ఫార్మ్‌లు (iOS, macOS మొదలైనవి) మరియు వేర్వేరు రకాల ప్రాజెక్ట్ టెంప్లేట్‌లు కనిపిస్తాయి.

సాధారణ టెంప్లేట్‌ల వివరణ:

* App —— అత్యంత ప్రాథమిక అప్లికేషన్ రకం (సిఫార్సు చేయబడుతుంది)
* Document App —— డాక్యుమెంట్ మోడ్ ఆధారిత అప్లికేషన్
* Game —— గేమ్ అభివృద్ధి కోసం
* Framework —— మళ్లీ వినియోగించగల మాడ్యూల్

ప్రారంభ దశలో, మనం కేవలం "iOS" - "App" టెంప్లేట్‌ను మాత్రమే ఎంచుకుంటాము, ఎందుకంటే ఇది ఇంటర్‌ఫేస్ ఆధారిత అప్లికేషన్ రకం.

తర్వాత "Next" పై క్లిక్ చేయండి.

### ప్రాజెక్ట్ సమాచారం నింపడం

ఇప్పుడు ప్రాజెక్ట్ సమాచారం నింపే స్క్రీన్‌లోకి వెళ్తాము. ఇక్కడ ప్రతీ ఫీల్డ్‌కు అనుగుణంగా సమాచారం నింపాలి.

![Swift](../../RESOURCE/001_xcode3.png)

ప్రతి ఫీల్డ్ యొక్క అర్థం:

* Product Name —— ప్రాజెక్ట్ పేరు. ఉదాహరణకు: SwiftSlimTest
* Team —— అభివృద్ధి బృందం. మీకు చెల్లింపు Apple Developer ఖాతా లేకపోతే, దీనిని ఎంచుకోకపోయినా సరిపోతుంది
* Organization Identifier —— సంస్థ గుర్తింపు. సాధారణంగా రివర్స్ డొమైన్ రూపాన్ని ఉపయోగిస్తారు. ఉదాహరణకు:

  * com.yourname
  * com.yourcompany
  * మీకు డొమైన్ లేకపోతే, మీ ఇంగ్లీష్ పేరును గుర్తింపుగా ఉపయోగించవచ్చు
* Bundle Identifier —— అప్లికేషన్‌కు చెందిన ఏకైక గుర్తింపు. ఇది Organization Identifier మరియు Product Name ఆధారంగా స్వయంచాలకంగా రూపొందుతుంది
* Interface —— ఇంటర్‌ఫేస్ టెక్నాలజీ, SwiftUI ను ఎంచుకోండి
* Language —— అభివృద్ధి భాష, Swift ను ఎంచుకోండి
* Testing System —— డిఫాల్ట్‌గా టెస్ట్ టార్గెట్‌లు (Unit Tests) సృష్టించబడతాయి; ప్రారంభ దశలో దీన్ని పట్టించుకోనవసరం లేదు
* Storage —— స్థానిక డేటా నిల్వ ఫ్రేమ్‌వర్క్. ఇది SwiftData లేదా Core Data ను ఆటోమేటిక్‌గా సమీకరించగలదు; ప్రారంభ దశలో దీన్నీ పట్టించుకోనవసరం లేదు

సమాచారం నింపిన తర్వాత, "Next" పై క్లిక్ చేయండి.

### ప్రాజెక్ట్‌ను సేవ్ చేయడం

![Swift](../../RESOURCE/001_xcode4.png)

సేవ్ చేయడానికి సరైన ఫోల్డర్‌ను ఎంచుకోండి.

"Create" బటన్‌పై క్లిక్ చేసిన తర్వాత, Xcode స్వయంచాలకంగా ప్రాజెక్ట్ నిర్మాణాన్ని రూపొందిస్తుంది.

## Xcode ప్రాజెక్ట్ నిర్మాణం

Xcode ప్రాజెక్ట్‌ను సేవ్ చేసిన తర్వాత, Finder లో రూపొందిన ప్రాజెక్ట్ ఫోల్డర్ కనిపిస్తుంది.

![Swift](../../RESOURCE/001_xcode5.png)

* Assets.xcassets —— చిత్రాలు, App చిహ్నాలు మరియు ఇతర వనరులను నిల్వ చేయడానికి ఉపయోగించే ఫోల్డర్

* ContentView.swift —— డిఫాల్ట్‌గా రూపొందే SwiftUI view ఫైల్. తరువాత మనం ఇక్కడే ఇంటర్‌ఫేస్ కోడ్ రాస్తాము.

* SwiftSlimTestApp.swift —— అప్లికేషన్ ప్రవేశ ఫైల్ (App Entry Point). ఇది అప్లికేషన్ ప్రారంభానికి బాధ్యత వహిస్తుంది; ప్రస్తుతానికి దీన్ని అర్థం చేసుకోవాల్సిన అవసరం లేదు.

* SwiftSlimTest.xcodeproj —— Xcode ప్రాజెక్ట్ ఫైల్. ఈ ఫైల్‌పై డబుల్ క్లిక్ చేస్తే ప్రాజెక్ట్ తెరుచుకుంటుంది.

## Xcode ఇంటర్‌ఫేస్‌ను అర్థం చేసుకోవడం

ప్రాజెక్ట్‌ను తెరవడానికి .xcodeproj ఫైల్‌పై డబుల్ క్లిక్ చేయండి. తరువాత ఎడమవైపు Navigator ప్రాంతంలో ContentView.swift ఫైల్‌ను ఎంచుకోండి.

Xcode ఇంటర్‌ఫేస్ ప్రధానంగా ఐదు ప్రాంతాలుగా విభజించబడింది:

* Navigator —— నావిగేషన్ ప్రాంతం; ఫైల్ నిర్మాణం చూడటానికి, శోధించడానికి ఉపయోగిస్తారు
* Editor —— ఎడిటింగ్ ప్రాంతం; Swift లేదా SwiftUI కోడ్ రాయడానికి ఉపయోగిస్తారు
* Canvas —— కేన్వాస్ ప్రాంతం; SwiftUI ప్రివ్యూ కోసం ఉపయోగిస్తారు
* Inspector —— గుణల పరిశీలన ప్రాంతం; ఫైల్ లక్షణాలను చూడటానికి, మార్చటానికి ఉపయోగిస్తారు
* Debug Area —— డీబగ్ ప్రాంతం; లాగ్ అవుట్‌పుట్ చూడటానికి ఉపయోగిస్తారు

![Swift](../../RESOURCE/001_xcode6.png)

గమనిక: Canvas ప్రధానంగా SwiftUI view preview కోసం ఉపయోగించబడుతుంది. Xcode అధికారిక నిర్మాణ దృష్టిలో ఇది Editor కు అనుబంధమైన preview panel మాత్రమే. SwiftUI view ఫైల్‌ను తెరిచినప్పుడు మాత్రమే Canvas ను ప్రారంభించవచ్చు; ఇతర రకాల ఫైల్‌లలో preview ఫీచర్ కనిపించదు.

తరువాతి పాఠాల్లో మనం ప్రధానంగా Editor ప్రాంతంలో Swift మరియు SwiftUI కోడ్ రాస్తాము.

మొదటిసారి Xcode ప్రాజెక్ట్‌ను తెరిచినప్పుడు, Inspector మరియు Debug Area డిఫాల్ట్‌గా దాచబడి ఉండవచ్చు. వాటిని పై కుడి మూల మరియు కింది కుడి మూల బటన్లపై క్లిక్ చేసి చూపించవచ్చు.

![Swift](../../RESOURCE/001_xcode7.png)

సూచన: Inspector ప్రాంతం ప్రధానంగా ఫైల్ లక్షణాలను చూడటానికి మరియు మార్చటానికి ఉపయోగించబడుతుంది. వాస్తవ అభివృద్ధిలో సాధారణంగా పెద్ద ఎడిటింగ్ స్థలం పొందడానికి ఈ ప్రాంతాన్ని దాచిపెడతారు.

## సారాంశం

ఈ పాఠంలో, Xcode ను ఎలా డౌన్‌లోడ్ చేయాలి, మొదటి Xcode ప్రాజెక్ట్‌ను ఎలా సృష్టించాలి, అలాగే Xcode యొక్క ప్రాథమిక ఇంటర్‌ఫేస్ అమరికను ఎలా అర్థం చేసుకోవాలి అన్నది నేర్చుకున్నాము.

ఇక కిందివి మీ స్వతంత్రంగా ప్రయత్నించే పనులు:

* Navigator ప్రాంతంలో ఫైళ్లను సృష్టించడం, తొలగించడం ప్రయత్నించండి。
* Editor ప్రాంతంలోని కోడ్‌ను చూసి, దాని నిర్మాణాన్ని గమనించండి。
* Canvas ప్రాంతంలోని బటన్లను చూసి, preview ను zoom in, zoom out చేసి చూడండి。

తదుపరి పాఠంలో, మనం కొన్ని సరళమైన కోడ్‌లు రాయడం ప్రారంభిస్తాము, తద్వారా Xcode గురించి మరింత లోతుగా అర్థం చేసుకుంటాము。
