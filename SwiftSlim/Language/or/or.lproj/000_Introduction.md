# ପ୍ରସ୍ତାବନା

## Swift କ’ଣ?

Swift ଶିଖିବା ପୂର୍ବରୁ, ଆମେ ପ୍ରଥମେ Swift ପ୍ରୋଗ୍ରାମିଙ୍ଗ ଭାଷା ବିଷୟରେ ସାରକଥାରେ ଜାଣିନେବା। Swift ହେଉଛି Apple ଦ୍ୱାରା ବିକଶିତ ଏକ ଆଧୁନିକ ପ୍ରୋଗ୍ରାମିଙ୍ଗ ଭାଷା, ଯାହା ପ୍ରଧାନତଃ iOS、macOS、iPadOS、watchOS ପରି Apple ପ୍ଲାଟଫର୍ମଗୁଡ଼ିକରେ ଆପ୍ଲିକେଶନ ତିଆରି କରିବା ପାଇଁ ବ୍ୟବହୃତ ହୁଏ।

![Swift](../../RESOURCE/000_swift.png)

ଅନ୍ୟାନ୍ୟ ପ୍ରୋଗ୍ରାମିଙ୍ଗ ଭାଷାସହିତ ତୁଳନା କଲେ, Swift ର syntax ଅଧିକ ସରଳ ଏବଂ ସୁରକ୍ଷା ଓ ପାଠଯୋଗ୍ୟତା ଉପରେ ଅଧିକ ଜୋର ଦିଏ। SwiftUI ସହିତ ଏହାକୁ ଶିଖିବା ଓ ବୁଝିବା ଅଧିକ ସହଜ ହୁଏ। ନିଶ୍ଚୟ, ମୁଁ ପୂର୍ବରୁ ଅନ୍ୟ କୌଣସି ପ୍ରୋଗ୍ରାମିଙ୍ଗ ଭାଷା ଶିଖିନଥିଲି, ସେହିପରିସ୍ଥିତିରେ ଏହା ସମ୍ଭବତଃ HTML ଠାରୁ କିଛିଟା କଷ୍ଟକର ଲାଗିପାରେ।

Swift କେବଳ Apple ପ୍ଲାଟଫର୍ମ ପାଇଁ ଆପ୍ଲିକେଶନ ଡିଭେଲପ କରିବାରେ ସୀମିତ ନୁହେଁ। Swift community, Swift କୁ [Android](https://www.swift.org/blog/nightly-swift-sdk-for-android/) ପ୍ଲାଟଫର୍ମକୁ ପୋର୍ଟ କରିବା ପାଇଁ ଚେଷ୍ଟା କରୁଛି, ଏବଂ ଏହାକୁ [server](https://github.com/awslabs/swift-aws-lambda-runtime) ଭଳି ଅନ୍ୟ ପରିବେଶରେ ମଧ୍ୟ ଚଳାଇବାକୁ ଚାହୁଁଛି। ଦୀର୍ଘମିଆଦି ଉନ୍ନତିର ଦୃଷ୍ଟିକୋଣରୁ ଦେଖିଲେ, Swift ର ବିକାଶ ପାଇଁ ଅଧିକ ସ୍ଥାନ ଅଛି, ଏବଂ ଭବିଷ୍ୟତରେ Flutter、React Native ପରି cross-platform ପ୍ରୋଗ୍ରାମିଙ୍ଗ ଭାଷା ହେବାର ସୁଯୋଗ ମଧ୍ୟ ଅଛି।

## ଶିକ୍ଷାର ବିଷୟବସ୍ତୁ

ଏହି tutorial ର ମୁଖ୍ୟ କେନ୍ଦ୍ରବିନ୍ଦୁ Swift ଓ SwiftUI ର ମୌଳିକ ଜ୍ଞାନ।

ଆମେ ସରଳ variables ଓ data types ଠାରୁ ଆରମ୍ଭ କରିବୁ, SwiftUI ବ୍ୟବହାର କରି interface ତିଆରି କରିବୁ, data କିପରି store ଓ read କରାଯାଏ ସେଥିକି ବୁଝିବୁ, ଏବଂ ଶେଷରେ ସମ୍ଭବତଃ SwiftData、iCloud synchronization ପରି ଅଧିକ ଉନ୍ନତ ବିଷୟବସ୍ତୁ ମଧ୍ୟ ଅଧ୍ୟୟନ କରିବୁ।

ପ୍ରତ୍ୟେକ ଅଧ୍ୟାୟ ଏକ ପ୍ରାକ୍ତିକ ଉଦାହରଣକୁ କେନ୍ଦ୍ର କରି ଗଠିତ ହେବ, ଯାହାଦ୍ୱାରା ବାସ୍ତବ ପ୍ରୟୋଗରେ Swift ଜ୍ଞାନ ଶିଖାଯିବ।

କିଛି tutorial ପ୍ରଥମେ ବହୁତ ସମସ୍ତ syntax ଓ API ଗୁଡ଼ିକୁ ଏକାଠି ବୁଝାଇ, ପରେ project practice କୁ ପ୍ରବେଶ କରେ। ଆରମ୍ଭିକ ଶିକ୍ଷାର୍ଥୀଙ୍କ ପାଇଁ ଏହା ଏମିତି, ଯେପରି ଜଣେ ଶିଶୁ dictionary ରୁ ଅକ୍ଷର ଶିଖୁଛି — ବ୍ୟବହାରିକ ପରିପ୍ରେକ୍ଷ୍ୟ ଛଡ଼ା ବହୁତ ବିଷୟ ସ୍ମରଣ କରିବା, ଶେଷରେ କେବଳ ଆଗ୍ରହକୁ କମେଇ ଦେଏ।

ଏହି tutorial ରେ ଆମେ ବାସ୍ତବ development scenario ମଧ୍ୟରେ ଜ୍ଞାନ ଶିଖିବୁ, ଅର୍ଥାତ୍ ଆବଶ୍ୟକତା ଠାରୁ ସମ୍ପର୍କିତ ଜ୍ଞାନ ବିନ୍ଦୁକୁ ଆଣିବୁ।

ଏହି tutorial ସମାପ୍ତ କରିବା ପରେ, ଆପଣ ନିଜେ ଏକ ମୌଳିକ ଆପ୍ଲିକେଶନ ଡିଭେଲପ ଓ ଚଳାଇ ପାରିବେ, ଏହାକୁ ନିଜ iOS / macOS ଡିଭାଇସ୍ ଉପରେ ଇନସ୍ଟଲ କରିପାରିବେ, ଏବଂ ଆପଣଙ୍କ ଆପ୍କୁ App Store ରେ ମଧ୍ୟ ପ୍ରକାଶ କରିପାରିବେ।

## ଶୂନ୍ୟ ଭିତ୍ତିରୁ କିପରି ଶିଖିବେ?

ମୁଁ ମଧ୍ୟ ଆରମ୍ଭରେ ଏକଦମ ଶୂନ୍ୟ ଭିତ୍ତିରୁ ଆରମ୍ଭ କରିଥିଲି। ପ୍ରଥମେ କେବଳ Youtube ର କିଛି [Swift ଶିକ୍ଷାଭିଡିଓ](https://www.youtube.com/watch?v=4SaddkGm5yA&list=PLliocbKHJNwuLHJlVv6q5tKJxJltjBspC&index=3) ଦେଖିଥିଲି, ଏବଂ ମନେ ହେଲା iOS ଆପ୍ ଡିଭେଲପ କରିବା ଏତେ ଜଟିଳ ନୁହେଁ, ଯାହା ମୋର ଆଗ୍ରହକୁ ଉତ୍ତେଜିତ କଲା। ତାପରେ ମୁଁ ଆଉ ଅଧିକ Swift ଜ୍ଞାନ ଶିଖିବାର ଚେଷ୍ଟା କରିଲି। ଯେପରିକି [Swift Playgrounds](https://apps.apple.com/cn/app/swift-playgrounds/id1496833156?mt=12) ର programming game ଅନୁଭବ କରିବା, [Swift official tutorial documentation](https://docs.swift.org/swift-book/documentation/the-swift-programming-language/) ପଢ଼ିବା, ଏବଂ [100 Days Of SwiftUI](https://www.hackingwithswift.com/100/swiftui) ଶିଖିବା।

ଏବେ ପଛକୁ ଦେଖିଲେ, ମୁଖ୍ୟତଃ [100 Days Of SwiftUI](https://www.hackingwithswift.com/100/swiftui) ମୋତେ ଏକ ଶିକ୍ଷା ଢାଞ୍ଚା ଦେଇଥିଲା, ଯାହା ମୋତେ ପଦ୍ଧତିଗତ ଭାବରେ SwiftUI ଶିଖିବାର ସୁବିଧା ଦେଇଥିଲା। କିନ୍ତୁ, tutorial ଟି ଇଂରାଜୀରେ ଥିବାରୁ ଏବଂ ମୋର programming ଭିତ୍ତି ନଥିବାରୁ, ବୁଝିବା ପ୍ରକ୍ରିୟାରେ ଅତ୍ୟନ୍ତ ଅଧିକ କଷ୍ଟ ହୋଇଥିଲା। ସେଥିପାଇଁ, ମୁଁ [ChatGPT](https://chatgpt.com/) ଏବଂ [Claude](https://claude.ai/?redirect=claude.com) ପରି AI ଟୁଲ୍ ବ୍ୟବହାର କରି ଜ୍ଞାନ ବିନ୍ଦୁଗୁଡ଼ିକୁ ବୁଝିଥିଲି।

ଶିକ୍ଷା ପ୍ରକ୍ରିୟାରେ, କିଛି ଏମିତି ବିଷୟ ନିଶ୍ଚୟ ଆସେ ଯାହା ବୁଝିବାକୁ ଅଳ୍ପ କଷ୍ଟକର ଲାଗେ। ମୋତେ ମନେ ଅଛି, UserDefaults ଶିଖୁଥିବା ବେଳେ ମୁଁ କେବଳ ଏତିକି ଜାଣୁଥିଲି ଯେ data କିପରି save କରିବା, କିନ୍ତୁ ଏହାକୁ manually read କରିବାକୁ ପଡ଼େ ବୋଲି ଜାଣୁନଥିଲି। ଏପର୍ଯ୍ୟନ୍ତ ମୁଁ ମୋର code ରେ ସମସ୍ୟା ଅଛି ବୋଲି ସନ୍ଦେହ କରିଥିଲି। ଏତେ ସରଳ ଏକ ଜ୍ଞାନ ବିନ୍ଦୁ ମଧ୍ୟ ମୋର କିଛି ଦିନ ନଷ୍ଟ କରିଦେଇଥିଲା।

ଅନ୍ୟ ଉଦାହରଣ ଭାବେ, closures、main thread、generics ପରି ବିଷୟଗୁଡ଼ିକୁ ଆରମ୍ଭିକ ଶିକ୍ଷା ପର୍ଯ୍ୟାୟରେ ମୁଁ ଗଭୀର ଭାବରେ ବୁଝିପାରିନଥିଲି। ପରେ ବାସ୍ତବ development ରେ ପୁଣି ସେଗୁଡ଼ିକୁ ମିଳିଲେ, ଆଉଥରେ ଏହାମାନଙ୍କୁ ଶିଖିବା ପାଇଁ ନୂଆ କରି ସମୟ ଦେବାକୁ ପଡ଼ିଥିଲା।

ସେହିପରି, ମୋର ପରାମର୍ଶ ହେଉଛି:

**1、ମୌଳିକ ଜ୍ଞାନ ପୂରଣ କରନ୍ତୁ**

ଚୀନରେ ଏକ କହାବତ ଅଛି: “ସମସ୍ତ କାମର ଆରମ୍ଭ କଷ୍ଟକର।” Swift ଶୂନ୍ୟ ଭିତ୍ତିରୁ ଶିଖୁଥିବା ନୂତନ ଶିକ୍ଷାର୍ଥୀଙ୍କ ପାଇଁ, ଶିକ୍ଷା ପ୍ରକ୍ରିୟାରେ ଅନେକ କଠିନ ଲାଗୁଥିବା ବିଷୟ ଆସିପାରେ —— ବହୁତ ସମୟରେ ସେଗୁଡ଼ିକ ବାସ୍ତବରେ ତେତେ ଜଟିଳ ନୁହେଁ, ବରଂ ଏହି ଜ୍ଞାନର ପିରାମିଡର ତଳ ଭାଗରେ ଥିବା କିଛି ଅଧିକ ମୌଳିକ ଧାରଣା ସହିତ ପରିଚୟ ନଥିବାରୁ ସେମିତି ଲାଗେ।

ଏପରି ପରିସ୍ଥିତିରେ, [ChatGPT](https://chatgpt.com/) ଓ [Claude](https://claude.ai/?redirect=claude.com) ପରି AI tools ର ସହାୟତା ନେଇ, ତୁରନ୍ତ ଅଭାବ ଥିବା ମୌଳିକ ଜ୍ଞାନକୁ ପୂରଣ କରିପାରିବେ।

**2、ପ୍ରତ୍ୟେକ ଜ୍ଞାନ ବିନ୍ଦୁକୁ ବୁଝନ୍ତୁ**

ଶିଖୁଥିବା ସମୟରେ, ସର୍ବପ୍ରଧାନ କଥା ହେଉଛି ପ୍ରତ୍ୟେକ ଜ୍ଞାନ ବିନ୍ଦୁକୁ ସମ୍ଭବ ଥିବା ମାତ୍ରାରେ ବୁଝିବା, ଯଦିଓ ତାହା ପାଇଁ କିଛି ଦିନ ସମୟ ଲାଗିଯାଏ।

ଯଦି ଅବୁଝା ବିଷୟକୁ ଛାଡ଼ିଦେବେ, ପରବର୍ତ୍ତୀ development ରେ ବହୁତ ସମ୍ଭାବନା ଅଛି ସେହି ବିଷୟ ପୁଣି ଆସିବ, ଏବଂ ଆପଣଙ୍କ development progress କୁ ବାଧା ଦେବ। ଶେଷରେ ଆଉଥରେ ଏହାକୁ ଶିଖିବା ପାଇଁ ସମୟ ଲଗାଇବାକୁ ପଡ଼ିବ। ତେଣୁ ପରେ ସୁଧାରିବାଠାରୁ, ଶିକ୍ଷା ପର୍ଯ୍ୟାୟରେ ଥିବାବେଳେ ଏହାକୁ ସଠିକ ଭାବରେ ବୁଝିନେବା ଭଲ।

ପ୍ରତ୍ୟେକ ଜ୍ଞାନ ବିନ୍ଦୁର ମୂଳ ସ୍ୱରୂପ ହେଉଛି ଏକ ଉପକରଣ —— ଯେତେ ଅଧିକ ଉପକରଣ ଆପଣ ଆତ୍ମସାତ୍ କରିବେ, ସମସ୍ୟା ସମାଧାନ କରିବାର କ୍ଷମତା ସେତେ ଶକ୍ତିଶାଳୀ ହେବ।

ଏହା ଘର ବନାଇବା ପରି। ଯଦି ଆପଣଙ୍କ ହାତରେ କେବଳ ଗୋଟିଏ ଉପକରଣ ଥାଏ, ତେବେ ଯଦିଓ ବହୁତ କାମକୁ ଯେମିତି ସେମିତି ସାରିହେବ, ଉଚ୍ଚ ଗୁଣୋତ୍ତର ଆବଶ୍ୟକତା ଆସିଲେ ଆପଣ ଅସହାୟ ହୋଇପଡ଼ିବେ। ଅଧିକ ଉପକରଣ ଆତ୍ମସାତ୍ କରିଲେ ମାତ୍ର ଆପଣ ଅଧିକ ମଜବୁତ ଘର ତିଆରି କରିପାରିବେ।

**3、ନିରନ୍ତର ଶିକ୍ଷା ଜାରି ରଖନ୍ତୁ**

ଶିକ୍ଷା ପାଇଁ ଦୀର୍ଘକାଳୀନ ଅଟଳତା ଦରକାର। ସତରେ କେହି କେହି ମାସ ଧରି ଗମ୍ଭୀର ଭାବରେ ଗୋଟିଏ tutorial ଶିଖିବା ପାଇଁ ସମୟ ଦେବାକୁ ଇଚ୍ଛୁକ ହୁଅନ୍ତି। ଅଳ୍ପ ସମୟର ଅତ୍ୟଧିକ ଚେଷ୍ଟାଠାରୁ, ନିୟମିତ ଭାବରେ ସମୟ ଦେଇ ଶିଖିବା ଅଧିକ ଜରୁରୀ।

ଯଦି ପ୍ରତିଦିନ କେବଳ ଏକ ପ୍ରତିଶତ କରି ସମାପ୍ତ କରିବେ, ତେବେ ଶତଦିନରେ ସମୁଦାୟ ଶିକ୍ଷା ଲକ୍ଷ୍ୟ ସମାପ୍ତ ହୋଇପାରିବ।

## ଶିକ୍ଷା ସ୍ୱପ୍ନ

ଏହି course ବହୁତ ଅଧିକ Swift ଓ SwiftUI ଜ୍ଞାନ ପଢ଼ାଇବା ପାଇଁ ନୁହେଁ; ବରଂ ଆରମ୍ଭିକ ଶିକ୍ଷାର୍ଥୀଙ୍କୁ Swift ବୋଲିଥିବା ଏହି programming ପଥରେ ପ୍ରବେଶ କରାଇବା ପାଇଁ। ଏହା ଟିକେ building blocks ପରି —— tutorial କେବଳ ଯୋଡ଼ିବାର ପ୍ରକ୍ରିୟା ବୁଝାଏ, ବାକୀ ସମୟରେ ଆପଣଙ୍କୁ ନିଜର blocks ଦ୍ୱାରା ନିଜ ଗଠନ ତିଆରି କରିବାକୁ ପଡ଼ିବ।

ଏହି project [GitHub](https://github.com/fangjunyu1/SwiftSlim) ରେ open source ଭାବେ ଉପଲବ୍ଧ, ଏବଂ ମାଗଣାରେ ଡାଉନଲୋଡ୍ ଓ ବ୍ୟବହାର କରିପାରିବେ। ସେଥିସହିତ, ଅନୁଭବ ପାଇଁ App Store version ମଧ୍ୟ ଦିଆଯାଇଛି।

ଆମର ଆଶା ହେଉଛି, ଯେମାନଙ୍କ ପାଖରେ ପ୍ରାୟୋଗିକ ପ୍ରାବଳ୍ୟ ନଥିଲେ ମଧ୍ୟ ନିଜ career path କୁ ପରିବର୍ତ୍ତନ କରିବାର ଇଚ୍ଛା ଅଛି, ସେମାନଙ୍କୁ programming ଓ app development ଶିଖିବା ମାଧ୍ୟମରେ ନୂଆ ସମ୍ଭାବନା ଦେବା।

## ବିଷୟବସ୍ତୁ ଅଦ୍ୟତନ

ଏହି tutorial, ବର୍ତ୍ତମାନର Swift ଓ SwiftUI ସଂସ୍କରଣକୁ ଆଧାର କରି ଲେଖାଯାଇଛି।

Apple ପ୍ଲାଟଫର୍ମ ଓ tools ର ଅଦ୍ୟତନ ସହିତ, କିଛି API ପରିବର୍ତ୍ତନ ହୋଇପାରେ। କୌଣସି ସମସ୍ୟା ମିଳିଲେ କିମ୍ବା ବିଷୟବସ୍ତୁ ଅଦ୍ୟତନ ଦରକାର ହେଲେ, official documentation କିମ୍ବା open-source repository ର ନବୀନତମ version ଦେଖନ୍ତୁ।
