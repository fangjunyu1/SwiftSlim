# 변수와 상수

이번 수업에서는 Swift의 변수와 상수를 중심으로, 자주 쓰이는 데이터 타입과 기본 연산자도 함께 배웁니다.

이 내용은 프로그래밍에서 가장 기초적인 지식이며, 이후 SwiftUI를 배우기 위한 중요한 기반이기도 합니다.

## 생활 속의 변수와 상수

우리는 생활 속 사물을 통해 변수와 상수를 이해할 수 있습니다.

생활 속의 변수:

- TV에서는 프로그램마다 재생되는 내용이 다릅니다
- 매일의 날씨는 모두 다릅니다
- 시계는 매초 변합니다

이런 것들의 공통점은: 계속 변한다는 것입니다.

만약 TV가 영원히 하나의 화면만 보여 주고, 날씨가 늘 맑고, 시계 바늘이 영원히 움직이지 않는다면, 이런 것들은 상수라고 볼 수 있습니다.

하나는 바뀔 수 있고, 다른 하나는 바뀌지 않습니다.

## 변수와 상수 이해하기

App 개발에서는 보통 사용자가 입력하거나 저장해야 할 정보가 있습니다.

예를 들어:

- 계정 이름
- 생일
- 연락처
- 주소

이런 정보는 저장되고, 다시 화면에 표시됩니다.

예를 들어 사용자가 App 안에 다음과 같은 이름을 입력했다고 합시다:

```
FangJunyu
```

우리는 이 이름을 저장해 두어야 App 안에서 다시 표시할 수 있습니다.

이 저장 과정을 간단히 "서랍에 물건을 넣는 것"이라고 생각해 볼 수 있습니다.

이름을 저장한다는 것은, 이름을 하나의 서랍 안에 넣는 것과 비슷합니다.

저장해야 할 내용은 많을 수 있고, 서랍도 많을 수 있습니다. 각 서랍 안에 무엇이 들어 있는지 알기 위해서는 서랍에 이름을 붙여야 합니다.

예:

```
name
```

이 예에서 `name`은 이름표이고, `FangJunyu`는 저장되는 정보입니다.

![Var](../../RESOURCE/007_var.png)

**Swift에서 데이터를 저장하려면 반드시 변수(`var`)나 상수(`let`)로 선언해야 합니다.**

이름은 일반적으로 바뀔 수 있으므로, 우리는 변수를 사용해야 합니다.

```swift
var name = "FangJunyu"
```

여기서는 `name`이라는 이름의 변수를 선언했으며, 변수 타입은 `String`, 값은 `"FangJunyu"`입니다.

### 변수와 상수의 차이

변수는 `var`로 선언합니다:

```swift
var
```

상수는 `let`으로 선언합니다:

```swift
let
```

예:

```swift
var name = "FangJunyu"
let id = 123456
```

변수로 선언하면 이 값은 수정할 수 있고, 상수로 선언하면 더 이상 수정할 수 없습니다.

따라서 변수와 상수의 가장 큰 차이는: 값 변경이 허용되는지 여부입니다.

### 상수 수정하기

코드가 상수의 값을 수정하려고 하면:

```swift
let name = "Sam"
name = "Bob"
```

Swift는 이것이 상수이므로 값을 다시 할당할 수 없다고 알려 줍니다.

```
Cannot assign to property: 'name' is a 'let' constant
```

이 메커니즘은 중요한 데이터가 실수로 변경되는 것을 막는 데 도움이 됩니다.

### SwiftUI에서 표시하기

ContentView.swift 파일을 열고, View 내부에 변수를 선언해 봅시다:

```swift
struct ContentView: View {
    var name = "FangJunyu"

    var body: some View {
        Text(name)
    }
}
```

여기서는 `name`이라는 변수를 선언하고, `Text`를 통해 화면에 표시합니다.

변수의 내용을 바꾸면:

```swift
struct ContentView: View {
    var name = "Sam"
    var body: some View {
        Text(name)
    }
}
```

`Text`에 표시되는 내용도 함께 바뀝니다.

이런 방식으로 우리는 매번 `Text` 안의 문자열을 직접 수정하지 않고도, 변수를 통해 화면 표시 내용을 제어할 수 있습니다.

또한 다른 곳에서 이 정보를 가져와야 한다면, 이 변수를 전달해 줄 수도 있습니다. 이는 마치 "서랍" 안의 내용을 다른 곳에 꺼내서 사용하는 것과 같습니다.

### SwiftUI에서 변수와 상수가 위치하는 곳

SwiftUI에서는 변수와 상수를 보통 `body` 바깥에 작성합니다:

```swift
struct ContentView: View {
    var name = "FangJunyu"

    var body: some View {
        Text(name)
    }
}
```

물론 변수와 상수를 `body` 내부에 작성할 수도 있습니다:

```swift
struct ContentView: View {
    var body: some View {
        var name = "FangJunyu"
        Text(name)
    }
}
```

하지만 두 방식에는 중요한 차이가 있습니다:

- 변수와 상수를 `body` 내부에 쓰면, 뷰가 다시 계산될 때마다 그 변수가 다시 만들어집니다
- `body` 외부에 쓰면, 변수는 뷰 구조체의 속성으로 존재하게 되어 코드 구조가 더 분명해집니다

그래서 실제 개발에서는 변수와 상수를 보통 `body` 외부에 작성합니다.

## 데이터 타입

변수는 서로 다른 여러 종류의 데이터를 저장할 수 있고, 이 종류를 데이터 타입이라고 합니다.

예를 들어 이전에 `Text`로 텍스트를 표시했습니다:

```swift
Text("Hello, World")
```

여기서 `"Hello, World"`는 문자열(`String`) 타입입니다.

문자열은 텍스트 내용을 나타내며, Swift에서는 반드시 큰따옴표 `""`로 감싸야 합니다.

예:

```swift
var hello = "Hello, World"
```

문자열 외에도 Swift에는 다양한 데이터 타입이 있습니다.

입문 단계에서 가장 자주 쓰는 네 가지 타입은 다음과 같습니다:

- String
- Int
- Double
- Bool

이 네 가지 타입만으로도 대부분의 기본 데이터 처리를 할 수 있습니다.

**String**

`String`은 텍스트 내용을 나타냅니다. 예:

```swift
var name = "FangJunyu"
var city = "Rizhao"
```

문자열은 보통 사용자 이름, 제목, 텍스트 내용 등을 표현하는 데 사용됩니다.

**Int**

`Int`는 정수, 즉 소수점이 없는 수를 나타냅니다.

예:

```swift
var age = 26
var count = 100
```

정수는 보통 나이, 수량, 카운트 등을 표현하는 데 사용됩니다.

**Double**

`Double`은 소수점을 포함하는 숫자를 나타냅니다.

예:

```swift
var weight = 74.5
var height = 185.0
```

Swift에서 소수는 기본적으로 `Double` 타입을 사용하는 경우가 많습니다.

이와 비슷한 타입으로 `Float`도 있지만, 표현 가능한 수의 범위가 더 작아서 실제 개발에서는 `Double`이 더 자주 사용됩니다.

**Bool**

`Bool` 타입은 `true`와 `false`, 두 가지 상태를 나타냅니다.

예:

```swift
var isShowAlert = true
var isLogin = false
```

`Bool` 타입은 조건 판단에 자주 사용됩니다. 예를 들어, 경고창을 표시할지 여부 같은 경우입니다.

결과가 두 가지뿐인 상황에서는 `Bool` 타입이 매우 적합합니다.

### SwiftUI에서 표시하기

`String` 타입을 표시할 때는 `Text`를 직접 사용할 수 있습니다:

```swift
Text(name)
```

하지만 `Int`, `Double` 같은 타입은 문자열이 아니므로, 그대로는 텍스트 내용의 일부로 사용할 수 없습니다.

이런 데이터를 `Text` 안에 표시하려면 문자열 보간(String Interpolation)을 사용할 수 있습니다:

```swift
\()
```

문자열 보간은 문자열 안에서 사용해야 하며, `\()` 안에 변수나 상수를 넣습니다.

예:

```swift
var num = 1

Text("\(num)")
Text("DoubleNum: \(num * 2)")
```

위 코드에서 `""`는 문자열을 의미하고, `\()`는 변수나 상수를 문자열 안에 삽입하는 데 사용됩니다.

예를 들어:

```swift
"DoubleNum: \(num * 2)"
```

화면에 표시되는 내용은:

```swift
DoubleNum: 2
```

이런 방식으로 문자열 안에 변수나 상수를 넣고, 그 결과를 `Text`로 화면에 표시할 수 있습니다.

팁: `\()`는 문자열 `""` 내부에서만 사용할 수 있습니다.

## 연산자

변수나 상수를 선언할 때 우리는 다음과 같은 형태를 볼 수 있습니다:

```swift
var num = 1
```

여기서 `=`는 대입 연산자입니다.

이 연산자의 역할은 오른쪽 값을 왼쪽 변수에 대입하는 것입니다.

이 예에서는 `1`이 변수 `num`에 대입됩니다.

대입 연산자 외에도, 자주 쓰이는 숫자 연산자가 있습니다:

- `+`
- `-`
- `*`
- `/`

숫자를 계산할 때, 예를 들어 `Int`나 `Double` 타입 데이터에서는 이런 연산자를 사용하게 됩니다.

예:

```swift
var age = 21 + 3 // 24
var value = 30 - 2 // 28
var num = 1 * 3 // 3
var weight = 88.5 / 2 // 44.25
```

연산이 끝나면 결과는 다시 변수에 저장됩니다.

## 정리

변수, 상수, 그리고 연산자는 프로그래밍에서 가장 기본적인 개념입니다.

변수와 상수를 통해 프로그램 안에 다양한 데이터를 저장할 수 있고, 데이터 타입을 통해 데이터의 종류를 명확히 구분할 수 있으며, 연산자를 통해 데이터를 계산하고 처리할 수 있습니다.

이 지식들은 프로그래밍 세계의 기초 도구와 같습니다. 이 내용을 익히면 이후 Swift와 SwiftUI를 배우는 데 튼튼한 기반이 됩니다。
