# Есептегіш

Алдыңғы екі сабақта біз айнымалыларды, тұрақтыларды, батырманы және әдістерді үйрендік.

Бұл сабақта осы білімдерді біріктіріп, қарапайым есептегіш қолданбасын жасаймыз.

Осы мысалда біз айнымалыны қалай сақтау және өзгерту керегін, сондай-ақ айнымалыны басқару үшін `@State` қолдануды үйренеміз.

## Есептегішті іске асыру

Есептегіш әдетте үнемі артып отыратын шаманы тіркеу үшін қолданылады. Мысалы, секіру саны, жүгірген айналым саны сияқты жағдайларда.

Енді қарапайым бір есептегіш жасайық: экранда бір сан мен батырма көрсетілсін, ал батырманы басқанда сан өссін.

![Num](../../RESOURCE/009_num.png)

### Санды көрсету

Алдымен санды көрсету үшін `Text` қолданамыз.

```swift
Text("0")
```

Бірақ сан өзгеруі керек болса, `"0"`-ді тікелей жазып қоюға болмайды, оның орнына санды айнымалыда сақтауымыз керек.

```swift
var num = 0
```

Содан кейін `Text` ішінде айнымалыны көрсетеміз:

```swift
Text("\(num)")
```

Мұнда жол интерполяциясы `\()` қолданылып, сан жолға айналдырылып көрсетіледі.

### Батырма арқылы санды өсіру

Келесі қадамда батырма қосамыз.

```swift
Button("+") {

}
.buttonStyle(.borderedProminent)
```

Батырма басылғанда, санның өскенін қалаймыз.

Сондықтан батырма ішінде айнымалыны өзгертуге болады:

```swift
Button("+") {
    num = num + 1
}
.buttonStyle(.borderedProminent)
```

Бұл кодтың мағынасы: батырма басылғанда, `num + 1` есептеліп, нәтиже қайтадан `num`-ге меншіктеледі.

### Ықшам жазылу

Swift одан да ықшам жазылуды ұсынады:

```swift
num += 1
```

Бұл мынаған тең:

```swift
num = num + 1
```

Мұндай жазылу "құрама меншіктеу операторы" деп аталады.

Swift-тегі жиі қолданылатын құрама меншіктеу операторлары:

```
+=   қосып меншіктеу
-=   азайтып меншіктеу
*=   көбейтіп меншіктеу
/=   бөліп меншіктеу
%=   қалдықпен меншіктеу
```

Мысалы:

```swift
num -= 2   // num = num - 2
num *= 3   // num = num * 3
```

### Толық код

Енді қарапайым бір есептегіштің кодын жаза аламыз:

```swift
struct ContentView: View {
    var num = 0
    var body: some View {
        VStack {
			Text("\(num)")
        	Button("+") {
        	    num += 1
        	}
        	.buttonStyle(.borderedProminent)
		}
    }
}
```

Бірақ бұл код қазір дұрыс жұмыс істемейді.

### Қате пайда болуы

Xcode мынадай қате көрсетеді:

```
Cannot assign to property: 'self' is immutable
```

Бұл view өзгермейтін болғандықтан, оның қасиетін өзгерту мүмкін емес дегенді білдіреді.

Неге олай болады?

## SwiftUI қалай жұмыс істейді

SwiftUI-де: **интерфейс деректермен басқарылады**, яғни дерек өзгерсе, интерфейс автоматты түрде жаңарады.

Бірақ маңызды бір жайт бар: тек күй (`@State`, `@Binding` сияқты) өзгерген кезде ғана SwiftUI бұл өзгерісті бақылап, view-ді жаңартады.

Егер кәдімгі айнымалы қолданылса, SwiftUI оның мәнін тек view алғаш құрылған кезде ғана оқиды, содан кейін айнымалы өзгерсе де, интерфейс жаңармайды.

Мысалы:

```swift
var num = 0   // 0
num = 1   // 0
```

Жоғарыдағы мысалда `num` — қарапайым айнымалы. Оның мәні өзгерсе де, SwiftUI view-ді жаңартпайды, сондықтан интерфейсте бастапқы `0` саны көрсетіле береді.

![Num](../../RESOURCE/009_state.png)

Сондықтан дерек күй ретінде белгіленген кезде ғана (`@State`, `@Binding` сияқты property wrapper қолданылғанда), SwiftUI оның өзгерісін автоматты түрде бақылап, view-ді жаңартады.

## @State property wrapper

SwiftUI-де айнымалыны өзгертіп, view-ді бірге жаңарту керек болса, міндетті түрде `@State` қолдану қажет.

```swift
@State var num = 0
```

`@State` SwiftUI-ге осы айнымалыны бақылауды тапсырады.

Айнымалы өзгергенде, SwiftUI view-ді қайта есептеп, қайта жаңартады.

Толық код:

```swift
struct ContentView: View {
    @State var num = 0
    var body: some View {
        VStack {
			Text("\(num)")
        	Button("+") {
        	    num += 1
        	}
        	.buttonStyle(.borderedProminent)
		}
    }
}
```

Енді батырманы басқанда:

```swift
num += 1
```

SwiftUI `num` өзгергенін байқап, view-ді автоматты түрде жаңартады.

![Num](../../RESOURCE/009_state1.png)

## @State қолдану ережелері

Нақты әзірлеуде `@State` әдетте екі ережеге бағынады.

### 1. @State әдетте private-пен жазылады

`@State` айнымалысы көбіне тек ағымдағы типтің ішінде ғана қолданылады (мысалы, ағымдағы View құрылымы ішінде), сондықтан әдетте былай жазылады:

```swift
@State private var num = 0
```

Бұл басқа view-лердің осы айнымалыға тікелей қол жеткізуіне немесе оны өзгертуіне жол бермейді.

Біз әлі `private`-ты толық оқыған жоқпыз, әзірге оны былай түсінуге болады:

**private-пен белгіленген мазмұнды тек ағымдағы view ішінде ғана қолдануға болады, сыртқы view-лер оны пайдалана алмайды.**

### 2. @State тек ағымдағы view күйі үшін қолданылады

`@State` ағымдағы View өзі басқаратын деректерге жарайды.

Мысалы, енгізу өрісінің мазмұны немесе қосқыштың күйі.

```swift
@State private var isOn = false
@State private var text = ""
```

Егер деректі бірнеше view арасында бөлісу қажет болса, онда `@Binding` немесе `@Observable` сияқты басқа күй түрлері қолданылады.

Бұл тақырыптарды кейінгі сабақтарда үйренеміз.

### Азайту батырмасы

Есептегіш санды тек өсіріп қана қоймай, азайта да алуы керек.

Ол үшін `-` батырмасын қосамыз.

```swift
Button("-") {
    num -= 1
}
.buttonStyle(.borderedProminent)
```

Содан кейін батырмаларды көлденең орналастыру үшін `HStack` қолданамыз:

```swift
HStack {
    Button("+") {
        num += 1
    }
    .buttonStyle(.borderedProminent)
    Button("-") {
        num -= 1
    }
    .buttonStyle(.borderedProminent)
}
```

Нәтиже:

![Num](../../RESOURCE/009_num1.png)

`+` батырмасы басылса, сан 1-ге артады; `-` батырмасы басылса, сан 1-ге кемиді.

### Қайта орнату батырмасы

Сонымен қатар қайта орнату батырмасын қосып, санды қайтадан 0 қыла аламыз.

```swift
Button("0") {
    num = 0
}
.buttonStyle(.borderedProminent)
```

Қайта орнату батырмасы басылғанда, `num` мәніне 0 беріледі.

Нәтиже:

![Num](../../RESOURCE/009_num2.png)

`0` батырмасы басылғанда, сан 0-ге өзгереді.

### Өзгертілетін батырма

Қазір біздің батырмалар тек мәтінді ғана көрсете алады:

```swift
Button("+") {

}
```

Мұндай жазылуда батырма тек мәтіндік мазмұнды ғана көрсете алады, мысалы `+` немесе `-`.

Нақты әзірлеуде батырманың сыртқы көрінісі байырақ болғанын қалаймыз, мысалы белгіше немесе басқа view қолданғымыз келеді.

SwiftUI батырманың көрсетілетін мазмұнын өзгертуге мүмкіндік береді, ол үшін батырманың басқа бір жазылу формасын қолданамыз:

```swift
Button(action: {

}, label: {

})
```

Бұл жазылуда `action` — батырма басылғанда орындалатын код, ал `label` — батырманың экранда көрсетілетін view-і.

Мысалы, батырмаға SF Symbols белгішесін көрсетуге болады:

```swift
Button(action: {
    num += 1
}, label: {
    Image(systemName: "plus.app")
})
```

Осылайша батырма мәтін емес, белгіше көрсетеді.

### Есептегіш интерфейсін жақсарту

Есептегіш интерфейсін анығырақ ету үшін, санның көрсетілу стилін де өзгерте аламыз. Мысалы, санды үлкейту:

```swift
Text("\(num)")
    .font(.largeTitle)
    .fontWeight(.bold)
```

Келесі қадамда батырмалардың мазмұнын да белгішеге ауыстырып, белгіше өлшемін сәл үлкейтеміз:

```swift
Button(action: {
    num += 1
}, label: {
    Image(systemName: "plus.app")
        .imageScale(.large)
})
```

Дәл сол сияқты азайту және қайта орнату батырмаларын да белгіше түріне ауыстыруға болады:

```swift
Button(action: {
    num -= 1
}, label: {
    Image(systemName: "minus.square")
        .imageScale(.large)
})

Button(action: {
    num = 0
}, label: {
    Image(systemName: "arrow.trianglehead.2.clockwise.rotate.90")
        .imageScale(.large)
})
```

Енді батырмалар SF Symbols белгішелерін көрсетеді, ал интерфейс анағұрлым түсінікті болады.

![Num](../../RESOURCE/009_num3.png)

## Қорытынды

Бұл сабақта қарапайым есептегішті іске асыру арқылы айнымалыларды, `Text` пен `Button` қолдануды кешенді түрде қайталадық. Біз санды айнымалыда сақтап, батырмалар арқылы оны өсіруді және азайтуды үйрендік, сондай-ақ `num += 1` және `num -= 1` сияқты құрама меншіктеу операторларын қолдандық.

Іске асыру барысында SwiftUI-дің жұмыс істеу тәсілін де түсіндік: интерфейс деректермен басқарылады. Дерек өзгергенде ғана интерфейс жаңарады. Сондықтан өзгеретін күйді сақтау үшін `@State` қолдану керек, ал `@State` айнымалысы өзгерсе, SwiftUI view-ді автоматты түрде жаңартады.

Соңында интерфейсті аздап жақсарттық: `Text` қаріп стилін өзгерттік және `Button(action:label:)` арқылы батырма мазмұнын өзгертіп, белгішелерді көрсеттік. Соның арқасында есептегіш интерфейсі анығырақ әрі әдемірек болды.

Енді біз SwiftUI view-лерінің негізгі қолданылу тәсілімен таныстық. SwiftUI-де интерфейстің басым бөлігі айнымалылар арқылы басқарылады. Айнымалы өзгергенде, SwiftUI view-ді автоматты түрде жаңартады. Сондықтан SwiftUI қолданбасын әзірлегенде, алдымен қандай деректі сақтау керек екенін жобалап алып, кейін сол деректер арқылы интерфейс пен интерактив логиканы құру әдетке айналады.

## Толық код

```swift
struct ContentView: View {
    @State var num = 0
    var body: some View {
        VStack(spacing: 20) {
            Text("\(num)")
                .font(.largeTitle)
                .fontWeight(.bold)
            HStack {
                Button(action: {
                    num += 1
                }, label: {
                    Image(systemName: "plus.app")
                        .imageScale(.large)
                })
                
                Button(action: {
                    num -= 1
                }, label: {
                    Image(systemName: "minus.square")
                        .imageScale(.large)
                })
                
                Button(action: {
                    num = 0
                }, label: {
                    Image(systemName: "arrow.trianglehead.2.clockwise.rotate.90")
                        .imageScale(.large)
                })
            }
        }
    }
}
```
