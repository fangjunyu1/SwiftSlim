# Айнымалылар мен тұрақтылар

Бұл сабақта біз Swift-тегі айнымалылар мен тұрақтыларды, сондай-ақ кейбір жиі қолданылатын дерек түрлері мен базалық операторларды үйренеміз.

Бұл мазмұн бағдарламалаудағы ең негізгі білімдердің бірі және SwiftUI-ді әрі қарай үйренудің маңызды іргетасы болып саналады.

## Күнделікті өмірдегі айнымалылар мен тұрақтылар

Айнымалы мен тұрақтыны күнделікті өмірдегі заттар арқылы түсінуге болады.

Өмірдегі айнымалылар:

- Теледидардағы әр бағдарламаның мазмұны әртүрлі болады
- Күн сайын ауа райы өзгеріп тұрады
- Сағат әр секунд сайын өзгереді

Олардың ортақ ерекшелігі: бәрі өзгеріп отырады.

Егер теледидарда әрқашан бір ғана сурет тұрса, ауа райы үнемі ашық болса, сағат тілі ешқашан қозғалмаса, онда мұндай нәрселер тұрақты болар еді.

Бірі өзгеруі мүмкін, екіншісі өзгермейді.

## Айнымалы мен тұрақтыны түсіну

App әзірлеуде пайдаланушыдан кейбір ақпаратты енгізу немесе сақтау жиі қажет болады.

Мысалы:

- аккаунт атауы
- туған күн
- байланыс ақпараты
- мекенжай

Бұл ақпарат сақталып, кейін экранда көрсетіледі.

Мысалы, пайдаланушы App ішінде мынадай атау енгізді:

```
FangJunyu
```

Біз бұл атауды сақтап қоюымыз керек, сонда оны App ішінде көрсете аламыз.

Сақтау процесін қарапайым түрде былай елестетуге болады: бір жәшікке зат салу.

Атауды сақтағанда, оны бір жәшікке салып қойғандай боламыз.

Сақталатын нәрсе көп болуы мүмкін, яғни жәшіктер де көп болады. Әр жәшікте не жатқанын білу үшін, оларға ат беруіміз керек.

Мысалы:

```
name
```

Бұл мысалда `name` — атау, ал `FangJunyu` — сақталатын ақпарат.

![Var](../../RESOURCE/007_var.png)

**Swift-те деректі сақтау керек болса, міндетті түрде айнымалы (`var`) немесе тұрақты (`let`) жариялау қажет.**

Атау әдетте өзгеруі мүмкін болғандықтан, мұнда айнымалы қолдануымыз керек.

```swift
var name = "FangJunyu"
```

Мұнда `name` атты айнымалы жарияланды, оның типі — `String`, ал мәні — `"FangJunyu"`.

### Айнымалы мен тұрақтының айырмашылығы

Айнымалы `var` арқылы жарияланады:

```swift
var
```

Тұрақты `let` арқылы жарияланады:

```swift
let
```

Мысалы:

```swift
var name = "FangJunyu"
let id = 123456
```

Егер айнымалы деп жарияланса, оның мәнін кейін өзгертуге болады; ал тұрақты болса, оны қайта өзгертуге болмайды.

Сондықтан айнымалы мен тұрақтының негізгі айырмашылығы мынада: мәннің өзгеруіне рұқсат етіле ме, жоқ па.

### Тұрақтыны өзгерту

Егер код тұрақтының мәнін өзгертуге тырысса:

```swift
let name = "Sam"
name = "Bob"
```

Swift бұл меншіктеуді орындай алмайтынын көрсетеді, себебі ол тұрақты.

```
Cannot assign to property: 'name' is a 'let' constant
```

Мұндай механизм әзірлеушіге кейбір маңызды деректердің кездейсоқ өзгеріп кетуінен сақтануға көмектеседі.

### SwiftUI ішінде көрсету

`ContentView.swift` файлын ашып, View ішінде айнымалы жариялайық:

```swift
struct ContentView: View {
    var name = "FangJunyu"

    var body: some View {
        Text(name)
    }
}
```

Мұнда `name` атты айнымалы жарияланып, `Text` арқылы экранда көрсетілді.

Егер айнымалының мазмұнын өзгертсек:

```swift
struct ContentView: View {
    var name = "Sam"
    var body: some View {
        Text(name)
    }
}
```

`Text` көрсететін мәтін де соған сәйкес өзгереді.

Осылайша, біз интерфейсте не көрсетілетінін айнымалы арқылы басқара аламыз, ал `Text` ішіндегі мәтінді әр жолы тікелей өзгерту қажет болмайды.

Егер бұл ақпарат басқа жерде де қажет болса, айнымалыны сол жаққа бере аламыз. Яғни "жәшіктегі" мазмұнды басқа жерге қолдануға жібергендей боламыз.

### SwiftUI-дегі айнымалы мен тұрақтының орны

SwiftUI-де айнымалылар мен тұрақтылар әдетте `body`-дің сыртында жазылады:

```swift
struct ContentView: View {
    var name = "FangJunyu"

    var body: some View {
        Text(name)
    }
}
```

Әрине, айнымалы мен тұрақтыны `body`-дің ішінде де жазуға болады:

```swift
struct ContentView: View {
    var body: some View {
        var name = "FangJunyu"
        Text(name)
    }
}
```

Бірақ мұнда маңызды айырмашылық бар:

- Егер айнымалы немесе тұрақты `body` ішінде жазылса, view әр қайта есептелген сайын ол қайтадан жасалады
- Егер `body` сыртында жазылса, айнымалы view құрылымының қасиеті ретінде сақталады, сондықтан код құрылымы анығырақ болады

Сондықтан нақты әзірлеуде айнымалы мен тұрақтыны көбіне `body`-дің сыртында жазады.

## Дерек түрлері

Айнымалылар көптеген әртүрлі типтегі деректерді сақтай алады, бұл типтер дерек түрлері деп аталады.

Мысалы, бұған дейін мәтінді `Text` арқылы көрсеткенбіз:

```swift
Text("Hello, World")
```

Мұндағы `"Hello, World"` — жол (`String`) типіне жатады.

Жол мәтіндік мазмұнды білдіреді және Swift-те ол міндетті түрде `""` қос тырнақшамен жазылады.

Мысалы:

```swift
var hello = "Hello, World"
```

Жолдан бөлек, Swift-те тағы көптеген дерек түрлері бар.

Бастапқы кезеңде ең жиі кездесетін төрт түрі:

- String
- Int
- Double
- Bool

Бұл төрт түрі негізгі дерек өңдеудің көп бөлігін орындауға жеткілікті.

**String**

`String` мәтіндік мазмұнды білдіреді. Мысалы:

```swift
var name = "FangJunyu"
var city = "Rizhao"
```

Жолдар әдетте пайдаланушы аты, тақырып, мәтін мазмұны сияқты деректерді көрсету үшін қолданылады.

**Int**

`Int` — бүтін сан, яғни бөлшек бөлігі жоқ сан.

Мысалы:

```swift
var age = 26
var count = 100
```

Бүтін сан әдетте жас, саны, есептегіш мәні сияқты деректерді білдіреді.

**Double**

`Double` — бөлшек бөлігі бар сан.

Мысалы:

```swift
var weight = 74.5
var height = 185.0
```

Swift-те бөлшек сандар әдетте әдепкі бойынша `Double` типін қолданады.

Оған ұқсас тағы бір түр — `Float`, бірақ оның көрсете алатын мәндер ауқымы кішірек, сондықтан нақты әзірлеуде `Double` жиі қолданылады.

**Bool**

`Bool` түрі екі күйді білдіру үшін қолданылады: `true` және `false`.

Мысалы:

```swift
var isShowAlert = true
var isLogin = false
```

`Bool` көбіне шарт тексеру үшін қолданылады. Мысалы, ескерту терезесі көрсетілсін бе, жоқ па.

Егер нәтиже тек екі түрлі болса, оны `Bool` арқылы көрсету өте ыңғайлы.

### SwiftUI ішінде көрсету

Біз `String` типін көрсеткенде, оны тікелей `Text` ішінде жаза аламыз:

```swift
Text(name)
```

Ал `Int`, `Double` сияқты түрлер жол емес, сондықтан оларды мәтіннің бір бөлігі ретінде тікелей қолдануға болмайды.

Егер бұл деректерді `Text` ішінде көрсету керек болса, жол интерполяциясын (`String Interpolation`) қолдануға болады:

```swift
\()
```

Жол интерполяциясы мәтіннің ішінде жазылады және айнымалы не тұрақты `\()` арқылы оралады.

Мысалы:

```swift
var num = 1

Text("\(num)")
Text("DoubleNum: \(num * 2)")
```

Жоғарыдағы кодта `""` — жол, ал `\()` айнымалыны не тұрақтыны сол жолдың ішіне енгізу үшін қолданылады.

Мысалы:

```swift
"DoubleNum: \(num * 2)"
```

Көрсетілетін мазмұн:

```swift
DoubleNum: 2
```

Осылайша, айнымалы мен тұрақтыны жолдың ішіне кірістіріп, оларды `Text` арқылы интерфейсте көрсете аламыз.

Ескерту: `\()` тек `""` жолының ішінде ғана қолданылады.

## Операторлар

Айнымалы не тұрақты жариялаған кезде, біз мынадай жазуды көреміз:

```swift
var num = 1
```

Мұндағы `=` — меншіктеу операторы.

Оның қызметі: оң жақтағы мәнді сол жақтағы айнымалыға беру.

Бұл мысалда `1` мәні `num` айнымалысына беріліп тұр.

Меншіктеу операторынан бөлек, сандармен жиі қолданылатын басқа операторлар да бар:

- `+`
- `-`
- `*`
- `/`

Сандарға есептеу жүргізгенде, мысалы `Int` не `Double` түріндегі деректермен жұмыс істегенде, осы операторлар қолданылады.

Мысалы:

```swift
var age = 21 + 3 // 24
var value = 30 - 2 // 28
var num = 1 * 3 // 3
var weight = 88.5 / 2 // 44.25
```

Есептеу аяқталған соң, нәтиже айнымалыға сақталады.

## Қорытынды

Айнымалылар, тұрақтылар және операторлар — бағдарламалаудағы ең негізгі ұғымдар.

Айнымалы мен тұрақты арқылы біз бағдарлама ішінде әртүрлі деректерді сақтай аламыз; дерек түрлері арқылы бұл деректердің қандай санатқа жататынын нақтылаймыз; ал операторлар арқылы оларды есептеп, өңдейміз.

Бұл білімдер бағдарламалау әлеміндегі базалық құралдар сияқты. Оларды жақсы меңгеру Swift пен SwiftUI-ді әрі қарай үйренуге берік негіз қалайды.
