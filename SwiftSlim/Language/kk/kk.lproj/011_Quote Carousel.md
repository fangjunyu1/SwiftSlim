# Дәйексөз каруселі

Бұл сабақта біз "дәйексөз каруселі" функциясын жасаймыз және Array (массив), if-else сияқты Swift-тің базалық білімдерін тереңірек үйренеміз.

Біз бірнеше дәйексөзді қалай сақтау керегін және батырма интеракциясы арқылы оларды циклдік түрде қалай ауыстыруға болатынын көрсетеміз.

![alt text](../../RESOURCE/011_word.png)

## Дәйексөзді көрсету

Алдымен SwiftUI ішінде бір дәйексөзді көрсетуіміз керек.

Ең қарапайым тәсіл — `Text` view-ін пайдалану:

```swift
Text("Slow progress is still progress.")
```

Бұл код тек бір ғана тұрақты дәйексөзді көрсетеді. Егер бірнеше дәйексөзді көрсетіп, оларды ауыстыру функциясын жасағымыз келсе, дәйексөздерді сақтап қоюымыз керек.

Бірақ қарапайым жолдық айнымалы тек бір ғана дәйексөзді сақтай алады:

```swift
let sayings = "Slow progress is still progress."
```

Егер бірнеше дәйексөз сақтау керек болса, әрқайсысы үшін жеке айнымалы жариялау қажет:

```swift
let saying1 = "Slow progress is still progress."
let saying2 = "Difficult roads lead to beautiful destinations."
let saying3 = "Awake minds are the strongest."
let saying4 = "Long-term thinking wins over short-term tricks."
let saying5 = "Master one thing, and do it well."
```

Бірақ нақты әзірлеуде мұндай тәсіл тек қолайсыз ғана емес, әр айнымалы тәуелсіз болғандықтан, икемді карусель ауыстыруын жасауға мүмкіндік бермейді.

Бірнеше дәйексөзді ыңғайлы басқару үшін оларды бірге сақтайтын дерек құрылымы керек, ол — массив (Array).

Массив қолданылғаннан кейін, жоғарыдағы кодты былай сақтауға болады:

```swift
let sayings = [
    "Slow progress is still progress.",
    "Difficult roads lead to beautiful destinations.",
    "Awake minds are the strongest.",
    "Long-term thinking wins over short-term tricks.",
    "Master one thing, and do it well."
    ]
```

**Ескерту: бағдарламалау стилінде бірнеше элемент сақтайтын массив айнымалысының атауы әдетте көпше түрде жазылады, мысалы `sayings`, бұл оның жинақ екенін көрсетеді.**

## Массив

Swift-те массив — реттелген элементтер жиыны, ол тік жақша `[]` арқылы көрсетіледі.

```swift
[]
```

Массив ішінде бір типке жататын бірнеше элемент болуы мүмкін, ал элементтер ағылшынша үтір `,` арқылы бөлінеді.

Мысалы:

```swift
[101, 102, 103, 104, 105]
```

Массивті қарапайым түрде бір пойызға ұқсатуға болады:

![Array](../../RESOURCE/011_array1.png)

Бүкіл пойыз осы массив объектісін білдіреді, ал әр вагон ретімен орналасады.

### Индекс және элементке қол жеткізу

Массив реттелген болғандықтан, жүйе реті бойынша нақты элементті таба алады. Бұл табу механизмі индекс (Index) деп аталады.

Swift-те (және бағдарламалау тілдерінің басым бөлігінде) массив индексі 1-ден емес, 0-ден басталады. Бұл дегеніміз, бірінші элементтің индексі — 0, екіншісі — 1, ары қарай осылай жалғасады.

![Array](../../RESOURCE/011_array2.png)

Массивтің белгілі бір элементіне қол жеткізу үшін, массив атауынан кейін тік жақша жазып, оның ішіне керекті элементтің индексін көрсету жеткілікті.

Мысалы:

```swift
sayings[0]   // 101
sayings[1]   // 102
```

Егер массивтің жарамды шегінен тыс индекске жүгінсек, “индекс ауқымнан тыс” (`Index Out of Range`) мәселесі туындайды. Сондықтан массивке қол жеткізгенде, индекс жарамды аралықта екеніне көз жеткізу керек.

**Индекс ауқымынан шығу**

Мысалы, массивте небәрі 5 элемент болса, рұқсат етілген индекс аралығы — 0-ден 4-ке дейін. Егер біз `sayings[5]`-ке жүгінсек, бағдарлама сәйкес “вагонды” таба алмайды, соның салдарынан “индекс ауқымынан тыс” қатесі шығып, қолданба құлайды.

![Array](../../RESOURCE/011_array3.png)

### Массивпен жұмыс істеу

Массив тек статикалық түрде анықталып қана қоймайды, сонымен қатар элемент қосу, жою, өзгерту сияқты операцияларды да қолдайды, әрі массив ұзындығын алуға болады.

Ескерту: егер массивті өзгерту қажет болса, оны `let` тұрақтысымен емес, `var` айнымалысымен жариялау керек.

**1. Элемент қосу**

Массив соңына элемент қосу үшін `append` әдісі қолданылады:

```swift
var num = [0, 1, 2]
num.append(3)   // [0, 1, 2, 3]
```

**2. Элемент жою**

`remove(at:)` әдісі арқылы массивтегі нақты элементті жоюға болады:

```swift
var num = [0, 1, 2]
num.remove(at: 0)   // [1, 2]
```

**3. Элементті өзгерту**

Массив элементін тікелей индекс арқылы өзгертуге болады:

```swift
var num = [0, 1, 2]
num[1] = 4   // [0, 4, 2]
```

**4. Массив ұзындығын есептеу**

Элементтер санын `count` қасиеті арқылы аламыз:

```swift
var num = [0, 1, 2]
num.count   // 3
```

### Массив арқылы дәйексөз көрсету

Бірнеше дәйексөз көрсету үшін, оларды массивке сақтап, кейін индекс арқылы оқып көрсетуге болады.

Алдымен `ContentView` ішінде дәйексөздерді сақтайтын `sayings` массивін жасаймыз, содан кейін `Text` view-і арқылы индекс бойынша қажетті дәйексөзді көрсетеміз:

```swift
struct ContentView: View {
    let sayings = [
        "Slow progress is still progress.",
        "Difficult roads lead to beautiful destinations.",
        "Awake minds are the strongest.",
        "Long-term thinking wins over short-term tricks.",
        "Master one thing, and do it well."
    ]
    
    var body: some View {
        VStack {
            Text(sayings[0])
        }
        .padding()
    }
}
```

Мұнда `sayings[0]` массивтегі бірінші дәйексөзді білдіреді.

Егер басқа дәйексөзді көрсеткіміз келсе, тек жақша ішіндегі индексті өзгерту жеткілікті:

```swift
Text(sayings[0]) // "Slow progress is still progress."
Text(sayings[1]) // "Difficult roads lead to beautiful destinations."
```

### Дәйексөз индексін анықтау

Дәйексөздерді динамикалық түрде ауыстыру үшін, индексті `Text` ішіне “қатып қалған” түрде жазып қоюға болмайды.

Ағымдағы көрсетілетін индекс мәнін бөлек сақтайтын айнымалы жасауымыз керек.

SwiftUI-де өзгеретін индексті `@State` арқылы жариялай аламыз:

```swift
@State private var index = 0
```

SwiftUI `@State` арқылы оралған айнымалыны бақылайды. `index` өзгерген кезде, SwiftUI view-ді қайта салып, сәйкес дәйексөзді көрсетеді.

Содан кейін массивтегі дәйексөзді `sayings[index]` арқылы динамикалық аламыз:

```swift
struct ContentView: View {
    @State private var index = 0
    let sayings = [
        "Slow progress is still progress.",
        "Difficult roads lead to beautiful destinations.",
        "Awake minds are the strongest.",
        "Long-term thinking wins over short-term tricks.",
        "Master one thing, and do it well."
    ]
    
    var body: some View {
        VStack {
            Text(sayings[index])
        }
        .padding()
    }
}
```

`index` мәні өзгерген сайын, `Text` басқа дәйексөзді көрсетеді.

### Индексті батырма арқылы басқару

Дәйексөз ауысуын басқару үшін, `Button` арқылы `index` мәнін өзгерте аламыз. Батырманы әр басқан сайын, `index` автоматты түрде 1-ге артады:

```swift
Button("Next") {
    index += 1
}
```

Батырманы басқанда, `index` 0-ден 1-ге өзгереді, view қайта салынып, `Text(sayings[index])` келесі дәйексөзді оқиды.

Бірақ мұнда бір ықтимал мәселе бар: егер батырманы үздіксіз баса берсек, `index` массив ауқымынан шығып кететін мәнге дейін өседі. Бұл массив индексінің шектен шығу қатесіне әкеледі. Мысалы, `index` 5-ке жетсе (ал массив индексі 0-ден 4-ке дейін ғана), бағдарлама құлайды.

Массив индексінің ауқымнан шығып кетуіне жол бермеу үшін, `index` массив шегінен аспайтынын тексеретін шарттық басқару енгізу керек. Бұл үшін if-else сөйлемін қолдануға болады.

## Шарттық басқару: if-else сөйлемі

`if-else` — Swift-тегі ең жиі қолданылатын шарттық тармақталу құрылымы. Ол шарттың орындалуын тексеріп, нәтижесіне қарай әртүрлі кодты орындайды.

Негізгі құрылымы:

```swift
if condition {
    // condition true болса орындалатын код
} else {
    // condition false болса орындалатын код
}
```

`if` ішіндегі `condition` — `Bool` типіндегі логикалық мән, яғни `true` немесе `false`. Егер шарт `true` болса, `if` бөлігі орындалады, әйтпесе `else` бөлігі орындалады.

Мысалы:

```swift
let age = 25
if age > 18 {
    print("Big Boy")
} else {
    print("Little Body")
}
```

Бұл мысалда `age` мәні 25, сондықтан `if` сөйлемі `age > 18` шартын тексереді, шарт орындалғандықтан `"Big Boy"` шығады.

Егер `else` бөлігі қажет болмаса, оны түсіріп тастауға болады:

```swift
if condition {
    // condition true болса орындалатын код
}
```

### Шарт арқылы индекс ауқымын басқару

Массив индексінің ауқымнан шығып кетпеуі үшін, `index` массив шегінен аспайтынын тексеретін `if` сөйлемін қолдануға болады:

```swift
Button("Next") {
    if index < sayings.count - 1 {
        index += 1
    }
}
```

Логикалық талдау: `sayings.count` мәні — 5 (яғни 5 дәйексөз бар), сондықтан `sayings.count - 1` мәні — 4, бұл массивтің соңғы жарамды индексі.

`index` 4-тен кіші болғанда, батырманы басып 1-ге арттыру қауіпсіз; ал `index` 4-ке жеткеннен кейін шарт орындалмайды, сондықтан батырма басылғанымен ештеңе болмайды.

Енді код дәйексөз ауыстыру функциясын іске асырды:

```swift
struct ContentView: View {
    @State private var index = 0
    let sayings = [
        "Slow progress is still progress.",
        "Difficult roads lead to beautiful destinations.",
        "Awake minds are the strongest.",
        "Long-term thinking wins over short-term tricks.",
        "Master one thing, and do it well."
    ]
    var body: some View {
        VStack {
            Text(sayings[index])
            Button("Next") {
                if index < sayings.count - 1 {
                    index += 1
                }
            }
        }
        .padding()
    }
}
```

### Дәйексөзді циклге айналдыру

Егер соңғы дәйексөз көрсетілгеннен кейін батырманы басқанда қайтадан бірінші дәйексөз көрсетілсін десек, яғни циклдік ауысу керек болса, оны `else` бөлігі арқылы жасауға болады:

```swift
Button("Next") {
    if index < sayings.count - 1 {
        index += 1
    } else {
        index = 0
    }
}
```

Батырма басылғанда, егер `index` массивтің соңғы элементіне жетсе, `index` қайтадан 0-ге түсіп, дәйексөздер циклдік түрде көрсетіледі.

## Дәйексөз view-ін жақсарту

Қазір біздің дәйексөз каруселі логикасы дайын, бірақ интерфейсті одан әрі әдемілеуге болады.

Толық код:

```swift
struct ContentView: View {
    @State private var index = 0
    let sayings = [
        "Slow progress is still progress.",
        "Difficult roads lead to beautiful destinations.",
        "Awake minds are the strongest.",
        "Long-term thinking wins over short-term tricks.",
        "Master one thing, and do it well."
    ]
    var body: some View {
        VStack(spacing: 10) {
            Text(sayings[index])
                .padding(20)
                .background(.white)
                .cornerRadius(10)
                .opacity(0.8)
            Button("Next") {
                if index < sayings.count - 1 {
                    index += 1
                } else {
                    index = 0
                }
            }
            .buttonStyle(.borderedProminent)
        }
        .padding()
        .background(
            Image("001")
        )
    }
}
```

Бұл мысалда `Text` view-іне ақ жартылай мөлдір фон мен дөңгелек бұрыш берілген, `Button` батырмасы `.borderedProminent` стилін пайдаланады, ал `VStack`-ке фон суреті орнатылған.

Қосымша білім: `background()` modifier-ін суреттік фонмен қолданғанда, оның әдепкі әрекеті — ағымдағы view орналасқан layout аймағын мүмкіндігінше толық жабу. Көп жағдайда ол табиғи түрде қауіпсіз аймаққа (`Safe Area`) дейін созылады.

Осылайша, дәйексөз каруселі view-ін жасадық.

## Қорытынды

Бұл сабақ арқылы біз бірнеше дәйексөзді сақтау үшін массивті қолдануды және `if`, `if-else` сөйлемдері арқылы дәйексөз каруселін іске асыруды үйрендік.

Сонымен қатар массивпен жұмыс істеудің негізгі операцияларын, мысалы элемент қосу, жою, өзгерту, сондай-ақ индекс ауқымынан шығып кетуді болдырмау тәсілін түсіндік.

Бұл сабақ тек дәйексөз каруселінің қалай жасалатынын ғана емес, сонымен бірге массив пен шарт сөйлемдерінің негізгі қолданысын біріктіріп, деректерді өңдеу мен бағдарламаның орындалу ағынын басқару қабілетін меңгеруге көмектесті.

## Қосымша білім - Көп тармақты шарт: if-else if-else сөйлемі

Нақты әзірлеуде жиі бірнеше шартты өңдеу керек болады. Мысалы, бір ойында ұпай 1-ге тең болса A оқиғасы, 2-ге тең болса B оқиғасы, 3-ке тең болса C оқиғасы іске қосылады, әрі қарай солай жалғасады.

Екіден көп шарт тармағы бар жағдайларда, `if-else if-else` сөйлемі қолданылады.

Негізгі синтаксис:

```swift
if conditionA {
    // conditionA true болса орындалатын код
} else if conditionB {
    // conditionB true болса орындалатын код
} else if conditionC {
    // conditionC true болса орындалатын код
} else {
    // Барлық шарт орындалмаса орындалатын код
}
```

Мұндай жағдайда бағдарлама шарттарды ретімен тексеріп, бірінші орындалған шарттың кодын орындайды. Егер ешқайсысы орындалмаса, `else` ішіндегі код орындалады.

Дәйексөз каруселінде де `if-else if-else` арқылы көп тармақты тексеру жасауға болады:

```swift
Button("Next") {
    if index == 0 {
        index += 1
    } else if index == 1 {
        index += 1
    } else if index == 2 {
        index += 1
    } else if index == 3 {
        index += 1
    } else if index == 4 {
        index = 0
    } else {
        index = 0
    }
}
```

`index` 0, 1, 2, 3 болғанда, батырманы әр басқан сайын `index += 1` орындалады; ал `index` 4 болғанда (соңғы элемент), `index` қайтадан 0-ге түсіп, цикл жасалады.

`else` бөлігі қорғаныс тармағы ретінде жазылған, яғни `index` қате мәнге өзгеріп кетсе де, ол 0-ге қайта оралады.

Маңыздысы, мұнда қолданылған `==` — "тең бе?" дегенді тексеретін оператор. Егер `if` сөйлемі `index` пен белгілі бір санның тең екенін анықтаса, нәтиже `true` болып, кейінгі код блогы орындалады. Егер тең болмаса, келесі `if` тармағына өтеді.

Осындай көп шартты тексеру әртүрлі жағдайларда әртүрлі код орындауға мүмкіндік береді.
