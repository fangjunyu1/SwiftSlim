# Операторлар

Бұл сабақта біз Swift операторларын (`Operators`) үйренеміз.

Есептеу, салыстыру немесе логикалық тексеру жасағанда, міндетті түрде операторларды қолданамыз. Мұндай әрекеттер бағдарламаның барлық кезеңінде кездеседі және бағдарламалаудағы ең негізгі қабілеттердің бірі болып саналады.

```swift
let a = 100 * 8
```

Бағдарламалауды үйрену барысында көптеген адамдар мынаны байқайды: белгілі бір деңгейде математикалық ойлауы бар адам бағдарламалық логиканы жылдамырақ түсінеді. Себебі бағдарламалау табиғаты жағынан “есептеу” мен “ережеге сай тексеруге” сүйенеді, ал осы екеуін дәл операторлар орындайды.

Нақты әзірлеуде операторлар тек қарапайым сандық есептеуге ғана емес, түрлі нақты сценарийлерде кеңінен қолданылады.

Мысалы, блок ойындарында пайдаланушы бір блокты белгілі бір орынға сүйрегенде, мынадай нәрселер есептелуі керек:

- блоктың координат орны
- блок толтыра алатын аймақ
- мақсатты орынның қолжетімді екенін тексеру

Осындай “есептеу + тексеру” процестерінің барлығы негізінде операторларға сүйенеді.

![Block](../../Resource/013_block.png)

Операторларды үйрену — жай ғана сандарды есептеуді үйрену емес, ол кейінгі интерфейс layout-ы, дерек өңдеу және интерактив логика үшін негіз дайындау.

## SwiftUI ішіндегі есептеудің орындалу тәсілі

SwiftUI-де есептеу логикасы әдетте `Button` арқылы функцияны іске қосып, сол функция ішінде орындалады.

Мысалы:

```swift
struct ContentView: View {
    func calculate() {
        let a = 10
        let b = 10
        let c = a + b
        print("c: \(c)")
    }
    var body: some View {
        VStack(spacing: 10) {
            Button("Button") {
                calculate()
            }
        }
    }
}
```

Батырманы басқанда, Console ішінде мынаны көреміз:

```
c: 20
```

Бұл сабақта біз есептеу логикасын біркелкі түрде `func` ішінде жазамыз, оны батырма арқылы іске қосамыз және нәтижені `print` арқылы шығарамыз.

Кейінгі мысалдарда кодты ықшамдау үшін, есептеу нәтижесін код ішінде тікелей белгілейміз, сондықтан `print`-ті қайта-қайта қолданбаймыз.

## Console шығаруды басқару

Әр басқан сайын жаңа шығулар пайда болатындықтан, Console аймағының төменгі оң жағындағы тазалау батырмасын басып, ескі логтарды өшіруге болады. Бұл ағымдағы нәтижені көруді жеңілдетеді.

![Console](../../Resource/013_console.png)

## Button-ның тағы бір жазылу түрі

Егер батырма тек бір функцияны ғана орындайтын болса, жазуды ықшамдауға болады:

```swift
Button(action: ) {
    // View
}
```

Бұл жазылуда `action` батырма басылғанда орындалатын функцияны (параметрсіз функцияны) көрсетеді, ал ирек жақша `{}` батырманың view-ін анықтайды.

Жоғарыдағы `calculate` функциясымен бірге былай жазуға болады:

```swift
Button(action: calculate) {
    Text("Button")
}
```

Батырманы басқанда, `calculate` функциясы тікелей орындалады.

Сондықтан батырма логикасы қарапайым болып, тек бір функцияны шақырса, `action` жазылуы анығырақ және код стиліне көбірек сай келеді.

## Арифметикалық операторлар

Арифметикалық операторлар (`Arithmetic Operators`) сандық есептеу үшін қолданылады:

- `+` қосу
- `-` азайту
- `*` көбейту
- `/` бөлу
- `%` қалдық алу

Мысал:

```swift
let a = 10 + 5  // 15
let b = 10 - 3  // 7
let c = 10 * 10 // 100
let d = 10 / 5  // 2
let e = 10 % 3  // 1
```

Маңыздысы, Swift — қатаң типтелген тіл, сондықтан әртүрлі типтерді автоматты түрде араластырмайды:

```swift
let a = 10  // Int
let b = 5.0 // Double

let c = a + b   // Error
let d = Double(a) + b   // 15.0
```

Егер екі мәннің типі әртүрлі болса, алдымен оларды қолмен бір типке келтіру керек.

## Салыстыру операторлары

Салыстыру операторлары (`Comparison Operators`) екі мәннің арақатынасын тексеру үшін қолданылады, ал нәтиже `Bool` болады:

- `==` тең
- `!=` тең емес
- `>` үлкен
- `<` кіші
- `>=` үлкен немесе тең
- `<=` кіші немесе тең

Мысал:

```swift
let a = 5 == 3  // false
let b = 2 != 4  // true
let c = 4 > 2   // true
let d = 5 < 3   // false
let e = 4 >= 4  // true
let f = 10 <= 7 // false
```

Салыстыру операторлары шарттық сөйлемдерде жиі қолданылады. Мысалы, “дәйексөз каруселінде” индекс массив шегінен аспасын десек:

```swift
if index < sayings.count - 1 {
    index += 1
}
```

Егер `index` мәні `sayings.count - 1`-ден кіші болса, салыстыру нәтижесі `true` болады және `if` ішіндегі код орындалады; ал `false` болса, кейінгі код орындалмайды.

## Логикалық операторлар

Логикалық операторлар (`Logical Operators`) бірнеше шартты біріктіру үшін қолданылады:

- `&&` және (`AND`)
- `||` немесе (`OR`)
- `!` емес (`NOT`)

### && және

Бірнеше шарт бір уақытта орындалуы керек болса, `&&` қолданылады.

Мысалы:

```swift
let age = 16
let legalAge = age >= 18 && age <= 100
```

Бұл кодта екі шарт бар: `age >= 18` және `age <= 100`.

Тек осы екі шарт та `true` болғанда ғана `legalAge` мәні `true` болады; егер біреуі орындалмаса, нәтиже `false`.

### || немесе

Бірнеше шарттың тек біреуі орындалса жеткілікті болса, `||` қолданылады.

Мысалы:

```swift
let money = 1000
let creditCard = 300
let payAllowed = money >= 50 || creditCard >= 50
```

Бұл кодтың мағынасы: `money >= 50` немесе `creditCard >= 50` шарттарының кез келген бірі орындалса, `payAllowed` `true` болады.

Тек екі шарттың екеуі де орындалмаса ғана нәтиже `false` болады.

Мысалы, төлем кезінде ақшамен де, кредиттік картамен де төлеуге болады. Егер екеуі де жарамаса, төлем орындалмайды.

### ! емес

`!` — логикалық терістеу операторы (`NOT`), ол `Bool` мәнін кері айналдырады.

Қарапайым түсіндірсек, `!` `true`-ды `false`-қа, ал `false`-ты `true`-қа ауыстырады.

Мысалы:

```swift
let allowed = true
let result = !allowed   // false
```

Мұндағы `!allowed` — `allowed` мәнін терістеу.

Маңыздысы, `!` пен `!=` екі түрлі оператор: `!` `Bool` мәнін терістейді, ал `!=` салыстыру операторы, ол `Bool` нәтижесін қайтарады.

## Меншіктеу операторлары

Меншіктеу операторлары (`Assignment Operators`) мән беру не айнымалыны жаңарту үшін қолданылады:

- `=` меншіктеу

Айнымалы жариялауда да, есептеуде де меншіктеу әрекеті жиі кездеседі:

```swift
let a = 5
```

Бұл код сан мәні `5`-ті `a` айнымалысына беруді білдіреді.

### Құрама меншіктеу операторлары

Нақты әзірлеуде қарапайым меншіктеуден бөлек, “құрама меншіктеу операторлары” жиі қолданылады. Олар бар мәнді тікелей есептеп, сол айнымалыға қайта жазады:

- `+=`
- `-=`
- `*=`
- `/=`

Бұған дейін “есептегіш” сабағында біз құрама меншіктеу операторларын қысқаша қарастырғанбыз.

Мысалы, батырма басылған сайын айнымалы өссін десек:

```swift
var num = 10
num += 5   // num = 15
```

Бұл код `num` пен `5`-ті қосып, шыққан нәтижені қайтадан `num`-ге меншіктейді.

Сондықтан ол мына кодқа тең:

```swift
var num = 10
num = num + 5   // num = 15
```

## Үштік оператор

Swift-тегі үштік оператор (`Ternary Operator`) шартты тексеріп, екі нәтижеден біреуін қайтарады:

```swift
шарт ? мән1 : мән2
```

Орындалу логикасы мынадай: егер шарт `true` болса, `мән1` қайтарылады; егер шарт `false` болса, `мән2` қайтарылады.

Мысалы:

```swift
let showColor = true
let backgroundColor = showColor ? Color.blue : Color.clear
```

Бұл код `showColor` мәнін тексереді: егер `true` болса, `Color.blue` қайтарады және фон көк болады; егер `false` болса, `Color.clear` қайтарылады, яғни фон түсі көрінбейді.

Ескерту: `Color.clear` — мөлдір түс (ештеңе көрінбейді).

### if-else-тен айырмашылығы

Үштік оператор логика жағынан `if-else` сөйлеміне тең.

Мысалы, жоғарыдағы кодты былай жазуға болады:

```swift
var backgroundColor: Color

if showColor {
    backgroundColor = Color.blue
} else {
    backgroundColor = Color.clear
}
```

Екі тәсілдің де орындалу логикасы бірдей: шарт `true` болса, бірінші нәтиже орындалады; шарт `false` болса, екінші нәтиже орындалады.

Қолдану жағынан үштік оператор ықшамырақ және “екінің бірін таңдау” сияқты қарапайым жағдайларды жазуға өте ыңғайлы, әсіресе бір мәнді тікелей қайтару қажет болғанда.

SwiftUI view құруда үштік оператор өте жиі кездеседі. Мысалы:

```swift
Text("Hello")
    .background(showColor ? Color.blue : Color.clear)
```

Мұндай жазылу бір жолдың ішінде шартқа қарай view әсерін динамикалық өзгертуге мүмкіндік береді.

## Аралық операторлары

Аралық операторлары (`Range Operators`) белгілі бір интервалды білдіру үшін қолданылады:

- `...` жабық аралық (екі шетін де қосады)
- `..<` жартылай ашық аралық (оң жақ шетін қоспайды)

Мысалы:

```
1...3   // 1, 2, 3
5..<8   // 5, 6, 7
```

Қарапайым түсіндірсек, `1...3` — 1-ден 3-ке дейін, 3-ті қоса алады; ал `5..<8` — 5-тен 8-ге дейін, бірақ 8-ді қоспайды.

Кеңейтілген формалар (тек танысу үшін):

```
5...    // >= 5
...5    // <= 5
```

Бұл жазылу “бір жақты аралықты” білдіреді, көбіне шарттық тексеру не pattern matching ішінде қолданылады.

Аралық операторлары кейін `for` циклін үйренгенде тағы қолданылады, әзірге оны бір үздіксіз сан аралығы ретінде түсінсеңіз жеткілікті.

## Операторлармен жаттығу

Енді бірнеше қарапайым мысал арқылы view ішінде операторлардың қалай жұмыс істейтінін тәжірибе жүзінде көрейік.

### 1. Көліктердің жалпы санын есептеу

Мысалы, бір қиылыстан күн сайын 500 көлік өтеді делік, енді 30 күндегі жалпы көлік санын есептейік.

Ол үшін арифметикалық оператор қолданамыз:

```swift
let dayVehicles = 500
let totalVehicles = 500 * 30    // 15000
```

Бұл есептеудің мағынасы: күніне 500 көлік, 30 күндегі жалпы саны `500 × 30`, сонда соңғы нәтиже — 15000 көлік.

### 2. Ішкі сатып алу мүшелігін тексеру

Бір адамның ішкі сатып алу мүшесі екенін тексере аламыз: егер онда өмірлік мүшелік немесе жазылым мүшелігінің біреуі болса, біз оны мүше деп санаймыз.

```swift
let lifeTime = false
let subscription = true
let isMember = lifeTime || subscription   // true
```

Бұл кодта `||` (`немесе`) операторы қолданылған: `lifeTime = false` — өмірлік мүшелік жоқ, `subscription = true` — жазылым мүшелігі бар.

Тек бір шарт орындалса жеткілікті болғандықтан, `isMember` мәні `true` болады. Демек, бұл адам ішкі сатып алу мүшесі.

## Қорытынды

Swift-те операторлар — дерекпен есептеу мен логикалық тексерудің негізі. `Text` мазмұнды көрсетсе, операторлар деректің өзін өңдеуге жауап береді.

Ең қарапайым сандық есептен бастап, күрделі логикалық тексеруге дейін, сондай-ақ view ішіндегі күй басқару мен layout есептеуіне дейін — барлығы операторларға сүйенеді.

Әртүрлі сценарийлердегі жаттығулар арқылы осы сабақта қарастырылған оператор түрлерін біртіндеп танып, меңгеруге болады. Бұл кейінгі оқудың маңызды негізі.

### Қосымша білім - Биттік операторлар

Сонымен қатар Swift төменгі деңгейдегі тағы бір операторлар тобын береді — биттік операторлар (`Bitwise Operators`):

- `&` (биттік және)
- `|` (биттік немесе)
- `^` (биттік XOR)
- `~` (биттік терістеу)
- `>>` (оңға жылжыту)
- `<<` (солға жылжыту)

Мысалы:

```swift
let a = 6  // 2лік жүйе 110
let b = 3  // 2лік жүйе 011

print(a & b) // 2 (2лік жүйе 010)
print(a | b) // 7 (2лік жүйе 111)
print(a ^ b) // 5 (2лік жүйе 101)
print(~a)    // -7 (биттік терістеу)
```

Swift-те бүтін сандар екілік толықтауыш (`two's complement`) түрінде сақталады, сондықтан терістегеннен кейін сәйкес теріс сан нәтижесі алынады.

Биттік операторлар екілік мәндермен тікелей жұмыс істейді және көбіне төменгі деңгейлі дерек өңдеу мен өнімділікті оңтайландыру сценарийлерінде қолданылады.

Жаңадан бастаушылар үшін бұл операторлардың қолданылу аясы сирек, әрі кәдімгі iOS немесе SwiftUI әзірлеуде жиі кездеспейді, сондықтан мұнда тереңірек талқыланбайды.

Егер кейін төменгі деңгейлі логика немесе өнімділік оңтайландыруы қажет болса, осы тақырыпты қайтадан тереңдеп үйренуге болады.
