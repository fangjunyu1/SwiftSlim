# Батырманы жұмыс істетейік

Бұл сабақта біз негізінен `Button` батырмасын, `print` арқылы жөндеу ақпаратын шығаруды және Swift функцияларын үйренеміз.

Бұл мазмұн SwiftUI-дегі интерактивті бағдарламалаудың негізі болып саналады.

## Батырма

Батырма — ең жиі қолданылатын UI контролдарының бірі. Жүктеу, ашу, шығу сияқты әрекеттер керек болғанда, біз әдетте батырманы басамыз.

Мысалы, App Store ішінде App жүктеу үшін "Алу" батырмасын басамыз.

![Button](../../RESOURCE/008_button.png)

SwiftUI-де батырма `Button` арқылы көрсетіледі.

Негізгі қолданылуы:

```swift
Button("") {
    
}
```

Құрылымды екі бөлікке бөлуге болады:

```swift
Button("Батырма мәтіні") {
    Батырманы басқанда орындалатын код
}
```

Мысалы:

```swift
struct ContentView: View {
    var body: some View {
        Button("Start") {

        }
    }
}
```

Іске қосқаннан кейін `Start` батырмасы көрсетіледі.

![Button](../../RESOURCE/008_button1.png)

Пайдаланушы батырманы басқанда, `{}` ішіндегі код орындалады.

Маңыздысы: `Button` — SwiftUI интерфейс контролы (`View`), сондықтан ол міндетті түрде `body` ішінде жазылуы керек.

### buttonStyle modifier-і

SwiftUI-де `Button` үшін жүйеге ендірілген батырма стильдері бар, оларды `buttonStyle` modifier-і арқылы қолдануға болады.

Мысалы:

```swift
Button("Start") {
    
}
.buttonStyle(.bordered)
```

`buttonStyle` жүйе беретін батырма стилін қолданады.

Жиі қолданылатын нұсқалар:

- `.automatic`
- `.plain`
- `.bordered`
- `.borderedProminent`
- `.borderless`

![Button](../../RESOURCE/008_button7.png)

Әртүрлі стильдердің айырмашылығы негізінен батырмада шекара, фон және визуал екпін бар-жоғына байланысты.

## print шығару

Бағдарламалауда кодтың белгілі бір бөлігіне жеткен-жетпегенін тексеру жиі қажет болады.

Swift жөндеу ақпаратын шығару үшін `print` функциясын ұсынады.

Негізгі жазылуы:

```swift
print("Hello")
```

Бұл жол мәтінді Console-ға (консольге) шығарады.

### Батырманың жұмысын тексеру

Батырманың басылған-басылмағанын тексеру үшін `print`-ті батырма ішіне жазуға болады.

```swift
Button("Start") {
    print("Test 123")
}
```

Батырманы басқанда, Console ішінде мынаны көреміз:

```
Test 123
```

Бұл батырманың сәтті іске қосылғанын білдіреді.

![Button](../../RESOURCE/008_button3.png)

Ескерту: `print` — Swift коды, ол интерфейс контролы емес. Оны `body` ішінде тікелей жаза алмаймыз, әйтпесе қате шығады.

Мысалы, мына жазылу қате:

```swift
struct ContentView: View {
    var body: some View {
        print("Test 123")  // Қате: body міндетті түрде View қайтаруы керек
    }
}
```

SwiftUI-де `body` интерфейс құрылымын сипаттау үшін пайдаланылады, сондықтан оның ішінде тек интерфейс контролдары (`View`) болуы керек, мысалы `Text`, `Button`, `Image` және т.б.

Қарапайым код (есептеу логикасы, `print` және т.б.) батырмада, функцияда немесе басқа оқиға өңдеушілерінің ішінде жазылуы тиіс.

### Console консолі

`print` арқылы шыққан мазмұн Xcode-тағы Console ішінде көрінеді.

Егер Console көрінбесе, төменгі оң жақтағы `Show the Debug Area` батырмасын басып, жөндеу аймағын ашыңыз, содан кейін `Show the Console` батырмасын басып консольді көрсетіңіз.

![](../../RESOURCE/008_button4.png)

Console-дың төменгі сол жағында екі белгі бар: `Executable` және `Previews`.

![](../../RESOURCE/008_button5.png)

Бұл екі белгі әртүрлі орындалу ортасына сәйкес келеді: Canvas ішінде алдын ала қарау жасап тұрсаңыз, `print` нәтижесі `Previews` астында көрінеді; ал қолданбаны симуляторда не нақты құрылғыда іске қоссаңыз, нәтиже `Executable` астында шығады.

Сондықтан `print` шықпай тұрса, алдымен дұрыс белгіні таңдағаныңызды тексеріңіз.

## Мысал - ASCII символдық сурет

Кейбір код жобаларында ASCII символдық суреттер жиі кездеседі.

Символдық сурет — кәдімгі таңбалар арқылы құрастырылған бейне. Мысалы:

![ASCII](../../RESOURCE/008_ascii.png)

Біз батырма + `print` арқылы бір ASCII суретті шығарып көре аламыз.

Мысал код:

```swift
struct ContentView: View {
    var body: some View {
        Button("Show ASCII") {
            print(" /\\_/\\")
            print("( o.o )")
            print(" > ^ <")
        }
        .buttonStyle(.borderedProminent)
    }
}
```

Батырманы басқанда, Console ішінде мыналар шығады:

```
 /\_/\
( o.o )
 > ^ <
```

Ескерту: жол ішіндегі `\` белгісін `\\` деп жазу керек, өйткені кері қиғаш сызық жол ішінде escape character болып саналады.

## Функция

Батырма ішінде кодты тікелей жаза аламыз.

Мысалы:

```swift
Button("Start") {
    print("Hello")
}
```

Егер код аз болса, мұндай жазылу жеткілікті. Бірақ нақты әзірлеуде бір батырма іске қосатын логика әлдеқайда көп болуы мүмкін.

Мысалы:

```swift
Button("Start") {
    print("Hello")
    print("World")
    print("This")
    print("is")
    // Әлі де көп код болуы мүмкін
}
```

Код көбейген сайын, оны батырма ішіне тікелей жазу интерфейс кодын шатастырып жібереді және оқу мен қолдауды қиындатады.

Сондықтан мұндай кодты әдетте бір функцияға жинақтап, кейін батырмадан сол функцияны шақырамыз.

### Функция деген не

Функция — қайта-қайта пайдалануға болатын код бөлігі.

Белгілі бір код орындалуы керек кезде, біз тек сол функцияны шақырамыз, сонда оның ішіндегі код іске қосылады.

Бұл код құрылымын анығырақ етеді және қайта қолдануды жеңілдетеді.

### Негізгі жазылуы

Swift-те функция `func` кілт сөзі арқылы анықталады:

```swift
func randomInt() {
    // code
}
```

Бұл код бір функцияны анықтайды.

`randomInt` — функция атауы, ол осы код бөлігін белгілеу үшін керек.

`() ` — параметр орны, мұнда сырттан берілетін деректер қабылдануы мүмкін. Егер дерек керек болмаса, бос қалдыруға болады.

`{}` ішіндегі аймақ — функцияның коды, орындалуы керек барлық код осында жазылады.

Мысалы:

```swift
func getName() {
    print("FangJunyu")
}
```

Бұл функцияның қызметі — бір мәтінді шығару.

### Функцияны шақыру

Функцияны анықтау — кодты жасап қою ғана. Оны шын мәнінде орындау үшін, функцияны шақыру қажет.

Шақыру тәсілі — функция атауынан кейін `()` жазу:

```swift
getName()
```

Бағдарлама осы жолға жеткен кезде, `getName` ішіндегі код орындалады.

Толық мысал:

```swift
func getName() {
    print("FangJunyu")
}

getName()
```

Іске қосқаннан кейін, Console ішінде мынаны көреміз:

```
FangJunyu
```

### SwiftUI ішінде функция қолдану

SwiftUI view ішінде функция әдетте `body` сыртында жазылады.

Мысалы:

```swift
struct ContentView: View {

    func getName() {
        print("FangJunyu")
    }

    var body: some View {
        Button("getName") {
            getName()
        }
    }
}
```

Батырманы басқанда, `getName()` функциясы шақырылады.

Осылайша интерфейс коды таза болып қалады, ал нақты логика функцияның ішінде өңделеді.

### Функция параметрлері

Кейде функция әртүрлі дерекке қарай әртүрлі әрекет орындауы керек болады.

Мысалы, функция тек `"FangJunyu"` емес, әртүрлі аттарды шығара алсын десек.

Мұндайда параметр қолдануға болады. Параметр дегеніміз — функция шақырылған кезде берілетін дерек.

Мысалы:

```swift
func getName(name: String) {
    print(name)
}
```

Бұл функцияға жаңа параметр қосылды: `name` — параметр атауы, `String` — оның типі.

Бұл функция `String` типіндегі деректі қабылдауы керек дегенді білдіреді.

Енді функцияға дерек керек болғандықтан, шақырғанда міндетті түрде сол деректі беру қажет.

```swift
getName(name: "Sam")
```

Шақыру кезінде `"Sam"` берілсе, функция ішінде дәл осы мән шығарылады.

Нәтиже:

```
Sam
```

## Мысал - функция арқылы ASCII сурет шығару

Алдыңғы мысалда `print`-ті батырма ішіне тікелей жаздық.

Бірақ символдық сурет күрделірек болса, код тым ұзарып кетеді. Ондайда логиканы функцияға шығарып, интерфейс кодын таза етуге болады.

Мысал код:

```swift
struct ContentView: View {

    func printCat() {
        print(" /\\_/\\")
        print("( o.o )")
        print(" > ^ <")
    }

    var body: some View {
        Button("Show ASCII") {
            printCat()
        }
        .buttonStyle(.borderedProminent)
    }
}
```

Іске қосып, батырманы басқанда Console ішінде мыналар шығады:

```
 /\_/\
( o.o )
 > ^ <
```

Мұнда пайдаланушы батырманы басқаннан кейін екі нәрсе болады:

1. Батырма `printCat()` функциясын шақырады
2. Функция ішіндегі `print` кодтары орындалады

Осылайша интерфейс коды тек оқиғаны іске қосуға жауап береді, ал нақты логика функция ішінде өңделеді.

Нақты әзірлеуде батырма әрекеттерінің басым бөлігі функция арқылы орындалады.

## Қорытынды

Бұл сабақта үш маңызды мазмұнды үйрендік:

**1. Button батырмасы**

`Button` — SwiftUI-дегі әрекет іске қосатын control.

```swift
Button("Start") {

}
```

Пайдаланушы батырманы басқанда, ирек жақша ішіндегі код орындалады.

**2. print шығару**

`print` Console ішіне жөндеу ақпаратын шығару үшін қолданылады.

```swift
print("Hello")
```

Әзірлеуші Console арқылы бағдарламаның орындалу күйін бақылай алады.

**3. Функция**

Функция — қайта пайдалануға болатын код бөлігі.

```swift
func sayHello() {
    print("Hello")
}
```

Функцияны шақыру:

```swift
sayHello()
```

Егер функцияға дерек керек болса, параметр қолдануға болады:

```swift
func sayHello(name: String) {
    print(name)
}
```

Келесі сабақтарда біз тағы да басқа SwiftUI контролдарын және деректердің интерфейспен қалай әрекеттесетінін үйренеміз.

## Жаттығу

Төмендегі жаттығуларды орындап көріңіз:

1. Мәтіні `"Hello"` болатын батырма жасаңыз
2. Батырманы басқанда Console ішіне `Hello Swift` шығарыңыз
3. Шығару кодын функцияға жазып, кейін батырмадан сол функцияны шақырыңыз
