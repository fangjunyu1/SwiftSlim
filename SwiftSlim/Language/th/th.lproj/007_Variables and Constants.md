# ตัวแปรและค่าคงที่

บทนี้จะเน้นเรียนรู้เรื่องตัวแปรและค่าคงที่ใน Swift พร้อมทั้งทำความรู้จักกับชนิดข้อมูลที่ใช้บ่อยบางประเภท และตัวดำเนินการพื้นฐาน

เนื้อหาเหล่านี้เป็นความรู้พื้นฐานที่สุดของการเขียนโปรแกรม และยังเป็นรากฐานสำคัญสำหรับการเรียน SwiftUI ในลำดับถัดไป

## ตัวแปรและค่าคงที่ในชีวิตประจำวัน

เราสามารถใช้สิ่งต่าง ๆ ในชีวิตประจำวันมาทำความเข้าใจตัวแปรและค่าคงที่ได้

ตัวแปรในชีวิตประจำวัน:

- เนื้อหาของแต่ละรายการในโทรทัศน์ไม่เหมือนกัน
- สภาพอากาศในแต่ละวันไม่เหมือนกัน
- นาฬิกาเปลี่ยนแปลงทุกวินาที

ลักษณะร่วมของสิ่งเหล่านี้คือ: มันสามารถเปลี่ยนแปลงได้

ถ้าโทรทัศน์มีภาพเดิมตลอดเวลา สภาพอากาศมีแต่แดดออกเสมอ และเข็มนาฬิกาไม่เคยขยับ สิ่งเหล่านี้ก็จะกลายเป็นค่าคงที่

อย่างหนึ่งอาจเปลี่ยนแปลงได้ แต่อีกอย่างหนึ่งจะไม่เปลี่ยนแปลง

## ทำความเข้าใจตัวแปรและค่าคงที่

ในการพัฒนาแอป โดยทั่วไปเรามักต้องให้ผู้ใช้กรอกหรือบันทึกข้อมูลบางอย่าง

ตัวอย่างเช่น:

- ชื่อบัญชี
- วันเกิด
- ข้อมูลติดต่อ
- ที่อยู่

ข้อมูลเหล่านี้จะถูกบันทึกและนำมาแสดงผล

ตัวอย่างเช่น ผู้ใช้กรอกชื่อหนึ่งลงในแอป:

``` id="6t5g1v"
FangJunyu
````

เราจำเป็นต้องเก็บชื่อนี้ไว้ เพื่อให้สามารถนำมาแสดงในแอปได้

เราสามารถเข้าใจขั้นตอนการเก็บข้อมูลอย่างง่ายได้ว่า: เป็นการเก็บของลงในลิ้นชักหนึ่งใบ

เมื่อเราบันทึกชื่อ ก็เหมือนกับการนำชื่อนั้นใส่ลงไปในลิ้นชัก

สิ่งที่ต้องเก็บอาจมีหลายอย่าง และลิ้นชักก็อาจมีหลายใบ เพื่อให้รู้ว่าแต่ละลิ้นชักเก็บอะไรไว้ เราจึงต้องตั้งชื่อลิ้นชัก

ตัวอย่างเช่น:

```id="djqwse"
name
```

ในตัวอย่างนี้ name คือชื่อ และ FangJunyu คือข้อมูลที่ถูกเก็บไว้

![Var](../../RESOURCE/007_var.png)

**ใน Swift ถ้าต้องการเก็บข้อมูล จะต้องประกาศด้วยตัวแปร (var) หรือค่าคงที่ (let) เสมอ**

เนื่องจากชื่อมักเป็นข้อมูลที่สามารถแก้ไขได้ ดังนั้นเราจึงควรใช้ตัวแปร

```swift id="i9vx0z"
var name = "FangJunyu"
```

ตรงนี้เป็นการประกาศตัวแปรชื่อ name โดยมีชนิดข้อมูลเป็น String และมีค่าเป็น "FangJunyu"

### ความแตกต่างระหว่างตัวแปรและค่าคงที่

ตัวแปรประกาศด้วย var:

```swift id="ac8ij4"
var
```

ค่าคงที่ประกาศด้วย let:

```swift id="h7bq3u"
let
```

ตัวอย่างเช่น:

```swift id="eisw4q"
var name = "FangJunyu"
let id = 123456
```

ถ้าประกาศเป็นตัวแปร แสดงว่าค่านี้สามารถเปลี่ยนแปลงได้ แต่ถ้าประกาศเป็นค่าคงที่ ก็จะไม่สามารถเปลี่ยนได้อีก

ดังนั้น ความแตกต่างหลักของตัวแปรและค่าคงที่ก็คือ: ค่าอนุญาตให้เปลี่ยนแปลงได้หรือไม่

### การแก้ไขค่าคงที่

ถ้าโค้ดพยายามแก้ไขค่าของค่าคงที่:

```swift id="cgbr1a"
let name = "Sam"
name = "Bob"
```

Swift จะแจ้งว่าไม่สามารถกำหนดค่าได้ เพราะมันเป็นค่าคงที่

```id="06h4xg"
Cannot assign to property: 'name' is a 'let' constant
```

กลไกนี้ช่วยให้นักพัฒนาหลีกเลี่ยงการที่ข้อมูลสำคัญบางอย่างถูกเปลี่ยนโดยไม่ตั้งใจ

### การแสดงผลใน SwiftUI

เปิดไฟล์ ContentView.swift แล้วประกาศตัวแปรภายใน View:

```swift id="fyz8p0"
struct ContentView: View {
    var name = "FangJunyu"

    var body: some View {
        Text(name)
    }
}
```

ตรงนี้เราได้ประกาศตัวแปรชื่อ name แล้วใช้ Text แสดงมันบนหน้าจอ

ถ้าเราเปลี่ยนค่าของตัวแปร:

```swift id="l14eha"
struct ContentView: View {
    var name = "Sam"
    var body: some View {
        Text(name)
    }
}
```

ข้อความที่ Text แสดงก็จะเปลี่ยนไปด้วย

ด้วยวิธีนี้ เราสามารถใช้ตัวแปรเพื่อควบคุมเนื้อหาที่แสดงบนหน้าจอได้ โดยไม่จำเป็นต้องแก้ข้อความใน Text โดยตรงทุกครั้ง

ถ้าที่อื่นต้องการใช้ข้อมูลนี้ เราก็สามารถส่งตัวแปรนี้ต่อไปได้ ซึ่งก็เหมือนกับการส่งต่อเนื้อหาที่อยู่ใน “ลิ้นชัก” ไปให้ส่วนอื่นใช้งาน

### ตำแหน่งของตัวแปรและค่าคงที่ใน SwiftUI

ใน SwiftUI ตัวแปรและค่าคงที่มักเขียนไว้นอก body:

```swift id="ashk5u"
struct ContentView: View {
    var name = "FangJunyu"

    var body: some View {
        Text(name)
    }
}
```

แน่นอนว่าเราสามารถเขียนตัวแปรและค่าคงที่ไว้ภายใน body ได้เช่นกัน:

```swift id="cq4b8n"
struct ContentView: View {
    var body: some View {
        var name = "FangJunyu"
        Text(name)
    }
}
```

แต่ทั้งสองแบบมีความแตกต่างสำคัญอย่างหนึ่ง:

* ถ้าตัวแปรและค่าคงที่เขียนอยู่ภายใน body ทุกครั้งที่มุมมองถูกคำนวณใหม่ ตัวแปรนั้นจะถูกสร้างใหม่
* ถ้าเขียนไว้นอก body ตัวแปรจะอยู่ในฐานะ property ของโครงสร้างมุมมอง ทำให้โครงสร้างโค้ดชัดเจนกว่า

ดังนั้น ในการพัฒนาจริง โดยทั่วไปเรามักเขียนตัวแปรและค่าคงที่ไว้นอก body

## ชนิดข้อมูล

ตัวแปรสามารถเก็บข้อมูลได้หลายประเภท ซึ่งประเภทเหล่านี้เรียกว่า ชนิดข้อมูล

ตัวอย่างเช่น ก่อนหน้านี้เราใช้ Text แสดงข้อความ:

```swift id="g3qj5t"
Text("Hello, World")
```

ตรงนี้ "Hello, World" เป็นชนิดข้อมูลแบบสตริง (String)

สตริงใช้สำหรับแทนข้อความ และใน Swift จำเป็นต้องครอบด้วยเครื่องหมายอัญประกาศคู่ ""

ตัวอย่างเช่น:

```swift id="im2jxe"
var hello = "Hello, World"
```

นอกจากสตริงแล้ว ใน Swift ยังมีชนิดข้อมูลอีกหลายแบบ

ในช่วงเริ่มต้น ชนิดข้อมูลที่พบบ่อยที่สุด 4 แบบคือ:

* String
* Int
* Double
* Bool

ชนิดข้อมูลทั้งสี่นี้สามารถรองรับการจัดการข้อมูลพื้นฐานส่วนใหญ่ได้แล้ว

**String**

String ใช้แทนเนื้อหาข้อความ ตัวอย่างเช่น:

```swift id="ucio78"
var name = "FangJunyu"
var city = "Rizhao"
```

สตริงมักใช้แทน: ชื่อผู้ใช้ ชื่อเรื่อง และข้อความเนื้อหา

**Int**

Int ใช้แทนจำนวนเต็ม หรือก็คือจำนวนที่ไม่มีจุดทศนิยม

ตัวอย่างเช่น:

```swift id="n0iyq3"
var age = 26
var count = 100
```

จำนวนเต็มมักใช้แทน: อายุ จำนวน หรือการนับ

**Double**

Double ใช้แทนตัวเลขที่มีจุดทศนิยม

ตัวอย่างเช่น:

```swift id="mjgnop"
var weight = 74.5
var height = 185.0
```

ใน Swift ตัวเลขทศนิยมโดยทั่วไปจะใช้ชนิด Double เป็นค่าเริ่มต้น

ยังมีอีกชนิดหนึ่งที่คล้ายกันคือ Float แต่ช่วงค่าที่มันแทนได้เล็กกว่า ดังนั้นในการพัฒนาจริง Double จึงถูกใช้บ่อยกว่า

**Bool**

ชนิดข้อมูล Bool ใช้แทนสองสถานะ: true และ false

ตัวอย่างเช่น:

```swift id="0l53vo"
var isShowAlert = true
var isLogin = false
```

ชนิด Bool มักใช้สำหรับการตัดสินใจตามเงื่อนไข ตัวอย่างเช่น จะแสดงกล่องแจ้งเตือนหรือไม่

สำหรับกรณีที่มีเพียงสองผลลัพธ์ ชนิด Bool จึงเหมาะมากในการใช้แทนค่า

### การแสดงผลใน SwiftUI

เมื่อเราจะแสดงชนิดข้อมูล String เราสามารถใช้ Text ได้โดยตรง:

```swift id="cuvc91"
Text(name)
```

แต่ Int, Double และชนิดอื่น ๆ ไม่ใช่สตริง ดังนั้นจึงไม่สามารถนำมาใช้เป็นส่วนหนึ่งของข้อความได้โดยตรง

ถ้าต้องการแสดงข้อมูลเหล่านี้ใน Text สามารถใช้การแทรกสตริง (String Interpolation):

```swift id="p6tko7"
\()
```

การแทรกสตริงต้องเขียนอยู่ภายในข้อความ และใช้ \() ครอบตัวแปรหรือค่าคงที่

ตัวอย่างเช่น:

```swift id="jlwmll"
var num = 1

Text("\(num)")
Text("DoubleNum: \(num * 2)")
```

ในโค้ดข้างต้น เครื่องหมาย "" หมายถึงสตริง ส่วน \() ใช้สำหรับแทรกตัวแปรหรือค่าคงที่เข้าไปในสตริง

ตัวอย่างเช่น:

```swift id="m54f7z"
"DoubleNum: \(num * 2)"
```

เนื้อหาที่แสดงจะเป็น:

```id="wtfwrr"
DoubleNum: 2
```

ด้วยวิธีนี้ เราสามารถแทรกตัวแปรหรือค่าคงที่ลงในสตริง แล้วใช้ Text แสดงมันบนหน้าจอได้

คำแนะนำ: \() ใช้ได้เฉพาะภายในสตริง "" เท่านั้น

## ตัวดำเนินการ

ตอนที่เราประกาศตัวแปรหรือค่าคงที่ เราจะเห็นรูปแบบการเขียนดังนี้:

```swift id="jtwdrs"
var num = 1
```

ตรงนี้ = เรียกว่าตัวดำเนินการกำหนดค่า

หน้าที่ของมันคือ: นำค่าทางด้านขวาไปกำหนดให้ตัวแปรทางด้านซ้าย

ในตัวอย่างนี้ 1 ถูกกำหนดให้กับตัวแปร num

นอกจากตัวดำเนินการกำหนดค่าแล้ว ยังมีตัวดำเนินการทางตัวเลขที่พบบ่อยอีกบางตัว:

* `+`
* `-`
* `*`
* `/`

เมื่อเราคำนวณตัวเลข เช่น ข้อมูลชนิด Int หรือ Double ก็จะใช้ตัวดำเนินการเหล่านี้

ตัวอย่างเช่น:

```swift id="ifoy4l"
var age = 21 + 3 // 24
var value = 30 - 2 // 28
var num = 1 * 3 // 3
var weight = 88.5 / 2 // 44.25
```

หลังจากคำนวณเสร็จ ผลลัพธ์จะถูกเก็บไว้ในตัวแปร

## สรุป

ตัวแปร ค่าคงที่ และตัวดำเนินการ คือแนวคิดพื้นฐานที่สุดของการเขียนโปรแกรม

ผ่านตัวแปรและค่าคงที่ เราสามารถเก็บข้อมูลหลากหลายประเภทไว้ในโปรแกรมได้ ผ่านชนิดข้อมูล เราสามารถระบุประเภทของข้อมูลได้อย่างชัดเจน และผ่านตัวดำเนินการ เราสามารถนำข้อมูลไปคำนวณและประมวลผลได้

ความรู้เหล่านี้เปรียบเสมือนเครื่องมือพื้นฐานของโลกการเขียนโปรแกรม เมื่อเข้าใจสิ่งเหล่านี้แล้ว ก็จะสามารถวางรากฐานที่มั่นคงสำหรับการเรียน Swift และ SwiftUI ในลำดับถัดไปได้
