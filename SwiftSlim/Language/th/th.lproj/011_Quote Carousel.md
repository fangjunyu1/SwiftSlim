# การเลื่อนแสดงคำคม

ในบทเรียนนี้ เราจะสร้างฟังก์ชัน “การเลื่อนแสดงคำคม” และเรียนรู้พื้นฐานของ Swift ให้ลึกขึ้น เช่น อาร์เรย์ (Array) และคำสั่งเงื่อนไข (if-else)

เราจะแสดงวิธีเก็บคำคมหลายข้อความ และใช้ปุ่มเพื่อให้คำคมเปลี่ยนวนลูปไปเรื่อย ๆ

![alt text](../../RESOURCE/011_word.png)

## แสดงคำคม

ก่อนอื่น เราต้องแสดงคำคมหนึ่งข้อความใน SwiftUI

วิธีที่ง่ายที่สุดคือใช้มุมมอง `Text`:

```swift
Text("Slow progress is still progress.")
```

โค้ดนี้จะแสดงได้เพียงคำคมคงที่หนึ่งข้อความเท่านั้น หากต้องการแสดงหลายคำคมและสลับไปมาได้ เราจำเป็นต้องเก็บคำคมเหล่านั้นไว้ก่อน

แต่ตัวแปรสตริงทั่วไปสามารถเก็บได้เพียงคำคมเดียว:

```swift
let sayings = "Slow progress is still progress."
```

ถ้าต้องการเก็บหลายคำคม ก็ต้องประกาศตัวแปรแยกสำหรับแต่ละคำคม:

```swift
let saying1 = "Slow progress is still progress."
let saying2 = "Difficult roads lead to beautiful destinations."
let saying3 = "Awake minds are the strongest."
let saying4 = "Long-term thinking wins over short-term tricks."
let saying5 = "Master one thing, and do it well."
```

แต่ในการพัฒนาจริง วิธีนี้ไม่เพียงยุ่งยากเท่านั้น แต่เพราะตัวแปรแต่ละตัวแยกจากกัน เราจึงไม่สามารถสร้างการเลื่อนแสดงแบบยืดหยุ่นได้

เพื่อให้จัดการคำคมหลายข้อความได้สะดวก เราจำเป็นต้องใช้โครงสร้างข้อมูลชนิดหนึ่งเพื่อเก็บพวกมันไว้ด้วยกัน ซึ่งก็คืออาร์เรย์ (Array)

หลังจากใช้อาร์เรย์ โค้ดด้านบนสามารถเขียนเป็น:

```swift
let sayings = [
    "Slow progress is still progress.",
    "Difficult roads lead to beautiful destinations.",
    "Awake minds are the strongest.",
    "Long-term thinking wins over short-term tricks.",
    "Master one thing, and do it well."
    ]
```

**คำแนะนำ: ตามหลักการตั้งชื่อในงานเขียนโปรแกรม ตัวแปรอาร์เรย์ที่เก็บหลายองค์ประกอบมักใช้ชื่อพหูพจน์ เช่น sayings เพื่อสื่อว่ามันเป็นชุดข้อมูล**

## อาร์เรย์

ใน Swift อาร์เรย์คือคอลเลกชันขององค์ประกอบที่มีลำดับ โดยใช้วงเล็บเหลี่ยม `[]` แทน

```swift
[]
```

ภายในอาร์เรย์สามารถมีองค์ประกอบหลายตัวที่เป็นชนิดเดียวกัน โดยคั่นแต่ละองค์ประกอบด้วยเครื่องหมายจุลภาคภาษาอังกฤษ `,`

ตัวอย่างเช่น:

```swift
[101, 102, 103, 104, 105]
```

เราสามารถเข้าใจอาร์เรย์แบบง่าย ๆ ได้ว่าเป็นขบวนรถไฟ:

![Array](../../RESOURCE/011_array1.png)

ขบวนรถไฟทั้งขบวนแทนออบเจ็กต์อาร์เรย์นี้ และแต่ละตู้รถไฟจะเรียงลำดับตามก่อนหลัง

### ดัชนีและการเข้าถึงองค์ประกอบ

เพราะอาร์เรย์เป็นข้อมูลที่มีลำดับ ระบบจึงสามารถระบุตำแหน่งขององค์ประกอบแต่ละตัวตามลำดับได้ กลไกนี้เรียกว่า ดัชนี (Index)

ใน Swift (รวมถึงภาษาโปรแกรมส่วนใหญ่) ดัชนีของอาร์เรย์เริ่มนับจาก 0 ไม่ใช่ 1 นั่นหมายความว่าองค์ประกอบตัวแรกมีดัชนีเป็น 0 ตัวที่สองเป็น 1 และเรียงต่อไปตามลำดับ

![Array](../../RESOURCE/011_array2.png)

หากต้องการเข้าถึงองค์ประกอบเฉพาะในอาร์เรย์ เพียงเขียนวงเล็บเหลี่ยมต่อท้ายชื่ออาร์เรย์ และใส่ค่าดัชนีขององค์ประกอบเป้าหมายลงไป

ตัวอย่างเช่น:

```swift
sayings[0]   // 101
sayings[1]   // 102
```

หากพยายามเข้าถึงดัชนีที่อยู่นอกขอบเขตของอาร์เรย์ จะเกิดปัญหา “ดัชนีเกินขอบเขต (Index Out of Range)” ดังนั้นเวลาเข้าถึงอาร์เรย์ต้องระวังให้ดัชนีอยู่ในช่วงที่ถูกต้อง

**ดัชนีเกินขอบเขต**

ตัวอย่างเช่น เมื่ออาร์เรย์มี 5 องค์ประกอบ ช่วงดัชนีที่ถูกต้องคือ 0 ถึง 4 ถ้าเราพยายามเข้าถึง `sayings[5]` โปรแกรมจะไม่พบ “ตู้รถไฟ” ที่ตรงกัน และจะเกิดข้อผิดพลาด “ดัชนีเกินขอบเขต” ทำให้แอปปิดตัวลง

![Array](../../RESOURCE/011_array3.png)

### การจัดการอาร์เรย์

อาร์เรย์ไม่ได้รองรับแค่การประกาศแบบคงที่เท่านั้น แต่ยังรองรับการเพิ่ม ลบ แก้ไของค์ประกอบ รวมถึงการดูความยาวของอาร์เรย์ด้วย

คำแนะนำ: ถ้าต้องการแก้ไขอาร์เรย์ ต้องประกาศด้วย `var` ไม่ใช่ `let`

**1. เพิ่มองค์ประกอบ**

สามารถใช้เมธอด `append` เพื่อเพิ่มองค์ประกอบที่ท้ายอาร์เรย์:

```swift
var num = [0, 1, 2]
num.append(3)   // [0, 1, 2, 3]
```

**2. ลบองค์ประกอบ**

ใช้เมธอด `remove(at:)` เพื่อลบองค์ประกอบตามตำแหน่งในอาร์เรย์:

```swift
var num = [0, 1, 2]
num.remove(at: 0)   // [1, 2]
```

**3. แก้ไของค์ประกอบ**

แก้ไของค์ประกอบโดยตรงผ่านดัชนี:

```swift
var num = [0, 1, 2]
num[1] = 4   // [0, 4, 2]
```

**4. นับจำนวนองค์ประกอบ**

ใช้คุณสมบัติ `count` เพื่อดูจำนวนองค์ประกอบในอาร์เรย์:

```swift
var num = [0, 1, 2]
num.count   // 3
```

### ใช้อาร์เรย์เพื่อแสดงคำคม

เพื่อแสดงคำคมหลายข้อความ เราสามารถเก็บคำคมไว้ในอาร์เรย์ แล้วเข้าถึงและแสดงผ่านดัชนีได้

ก่อนอื่น ให้สร้างอาร์เรย์ `sayings` ใน `ContentView` เพื่อเก็บคำคม จากนั้นในมุมมอง `Text` ให้อ่านและแสดงคำคมตามดัชนี:

```swift
struct ContentView: View {
    let sayings = [
        "Slow progress is still progress.",
        "Difficult roads lead to beautiful destinations.",
        "Awake minds are the strongest.",
        "Long-term thinking wins over short-term tricks.",
        "Master one thing, and do it well."
    ]
    
    var body: some View {
        VStack {
            Text(sayings[0])
        }
        .padding()
    }
}
```

ที่นี่ `sayings[0]` หมายถึงคำคมตัวแรกในอาร์เรย์

ถ้าต้องการแสดงคำคมอื่น ก็เพียงแค่เปลี่ยนค่าดัชนีในวงเล็บเหลี่ยม:

```swift
Text(sayings[0]) // "Slow progress is still progress."
Text(sayings[1]) // "Difficult roads lead to beautiful destinations."
```

### กำหนดดัชนีของคำคม

เพื่อให้คำคมเปลี่ยนแบบไดนามิก เราไม่ควร “เขียนค่าดัชนีตายตัว” ไว้ในมุมมอง `Text`

เราต้องสร้างตัวแปรขึ้นมาเพื่อเก็บค่าดัชนีที่กำลังแสดงอยู่แยกต่างหาก

ใน SwiftUI เราสามารถใช้ `@State` เพื่อประกาศดัชนีที่เปลี่ยนแปลงได้:

```swift
@State private var index = 0
```

SwiftUI จะเฝ้าติดตามตัวแปรที่ถูกครอบด้วย `@State` เมื่อ `index` เปลี่ยน SwiftUI จะเรนเดอร์วิวใหม่และแสดงคำคมที่ตรงกัน

จากนั้นเราใช้ `sayings[index]` เพื่อดึงคำคมจากอาร์เรย์แบบไดนามิก:

```swift
struct ContentView: View {
    @State private var index = 0
    let sayings = [
        "Slow progress is still progress.",
        "Difficult roads lead to beautiful destinations.",
        "Awake minds are the strongest.",
        "Long-term thinking wins over short-term tricks.",
        "Master one thing, and do it well."
    ]
    
    var body: some View {
        VStack {
            Text(sayings[index])
        }
        .padding()
    }
}
```

เมื่อค่า `index` เปลี่ยน `Text` ก็จะแสดงคำคมที่ต่างออกไป

### ใช้ปุ่มควบคุมดัชนี

เพื่อควบคุมการเปลี่ยนคำคม เราสามารถใช้ `Button` เพื่อแก้ค่า `index` ทุกครั้งที่กดปุ่ม `index` จะเพิ่มขึ้น 1 โดยอัตโนมัติ:

```swift
Button("Next") {
    index += 1
}
```

เมื่อเรากดปุ่ม ค่า `index` จะเปลี่ยนจาก 0 เป็น 1 ทำให้วิวรีเฟรช และ `Text(sayings[index])` จะอ่านคำคมถัดไป

อย่างไรก็ตาม ตรงนี้มีปัญหาที่อาจเกิดขึ้นได้: หากกดปุ่มต่อเนื่อง `index` จะเพิ่มไปจนเกินช่วงของอาร์เรย์ และทำให้เกิดข้อผิดพลาดดัชนีเกินขอบเขต ตัวอย่างเช่น เมื่อ `index` กลายเป็น 5 (ขณะที่ช่วงดัชนีของอาร์เรย์คือ 0 ถึง 4) โปรแกรมจะล่ม

เพื่อป้องกันไม่ให้ดัชนีเกินขอบเขตของอาร์เรย์ เราจำเป็นต้องใช้การควบคุมด้วยเงื่อนไข เพื่อให้แน่ใจว่า `index` จะไม่เกินช่วงของอาร์เรย์ ซึ่งสามารถทำได้ด้วยคำสั่ง `if-else` เพื่อตรวจสอบว่า `index` น้อยกว่าความยาวของอาร์เรย์หรือไม่

## การควบคุมเงื่อนไข: คำสั่ง if-else

คำสั่ง `if-else` เป็นคำสั่งแตกแขนงตามเงื่อนไขที่ใช้บ่อยที่สุดใน Swift ใช้สำหรับตรวจสอบว่าเงื่อนไขเป็นจริงหรือไม่ และทำงานต่างกันตามผลลัพธ์จริงหรือเท็จของเงื่อนไข

โครงสร้างพื้นฐาน:

```swift
if condition {
    // โค้ดที่จะทำงานเมื่อ condition เป็น true
} else {
    // โค้ดที่จะทำงานเมื่อ condition เป็น false
}
```

ในคำสั่ง `if` นั้น เงื่อนไข `condition` จะเป็นค่าบูลีน (`Bool`) ซึ่งอาจเป็น `true` หรือ `false` เมื่อเงื่อนไขเป็น `true` โค้ดในส่วน `if` จะถูกทำงาน มิฉะนั้นจะทำงานในส่วน `else`

ตัวอย่างเช่น:

```swift
let age = 25
if age > 18 {
    print("Big Boy")
} else {
    print("Little Body")
}
```

ในตัวอย่างนี้ ค่า `age` คือ 25 คำสั่ง `if` จะตรวจสอบว่า `age > 18` เป็นจริงหรือไม่ เมื่อเงื่อนไขเป็นจริง ก็จะแสดง `"Big Boy"`

ถ้าไม่ต้องการส่วน `else` ก็สามารถละไว้ได้:

```swift
if condition {
    // โค้ดที่จะทำงานเมื่อ condition เป็น true
}
```

### ใช้เงื่อนไขควบคุมช่วงของดัชนี

เพื่อป้องกันไม่ให้ดัชนีเกินขอบเขตของอาร์เรย์ เราสามารถใช้คำสั่ง `if` เพื่อให้แน่ใจว่า `index` จะไม่เกินช่วงของอาร์เรย์:

```swift
Button("Next") {
    if index < sayings.count - 1 {
        index += 1
    }
}
```

วิเคราะห์ตรรกะ: ค่า `sayings.count` คือ 5 (มีคำคมทั้งหมด 5 ข้อ) ดังนั้น `sayings.count - 1` จึงมีค่าเป็น 4 ซึ่งก็คือดัชนีสุดท้ายที่ถูกต้องของอาร์เรย์

เมื่อ `index` น้อยกว่า 4 การกดปุ่มเพื่อเพิ่มค่า 1 ยังคงปลอดภัย; แต่เมื่อ `index` เท่ากับ 4 แล้ว เงื่อนไขจะไม่เป็นจริงอีก การกดปุ่มก็จะไม่เกิดการเปลี่ยนแปลงใด ๆ

ตอนนี้โค้ดสามารถสลับคำคมได้แล้ว:

```swift
struct ContentView: View {
    @State private var index = 0
    let sayings = [
        "Slow progress is still progress.",
        "Difficult roads lead to beautiful destinations.",
        "Awake minds are the strongest.",
        "Long-term thinking wins over short-term tricks.",
        "Master one thing, and do it well."
    ]
    var body: some View {
        VStack {
            Text(sayings[index])
            Button("Next") {
                if index < sayings.count - 1 {
                    index += 1
                }
            }
        }
        .padding()
    }
}
```

### คำคมแบบวนลูป

ถ้าเราต้องการให้เมื่อแสดงคำคมสุดท้ายแล้ว พอกดปุ่มอีกครั้งกลับไปแสดงคำคมแรกใหม่ เพื่อให้คำคมแสดงวนลูปได้ เราสามารถใช้ส่วน `else` ได้ดังนี้:

```swift
Button("Next") {
    if index < sayings.count - 1 {
        index += 1
    } else {
        index = 0
    }
}
```

เมื่อกดปุ่ม ถ้า `index` ไปถึงองค์ประกอบสุดท้ายของอาร์เรย์แล้ว `index` จะถูกรีเซ็ตกลับเป็น 0 และเริ่มแสดงคำคมวนใหม่

## ปรับปรุงมุมมองคำคม

ตอนนี้ตรรกะการเลื่อนคำคมของเราเสร็จแล้ว แต่เรายังสามารถปรับหน้าจอให้ดูสวยขึ้นได้อีก

โค้ดแบบสมบูรณ์:

```swift
struct ContentView: View {
    @State private var index = 0
    let sayings = [
        "Slow progress is still progress.",
        "Difficult roads lead to beautiful destinations.",
        "Awake minds are the strongest.",
        "Long-term thinking wins over short-term tricks.",
        "Master one thing, and do it well."
    ]
    var body: some View {
        VStack(spacing: 10) {
            Text(sayings[index])
                .padding(20)
                .background(.white)
                .cornerRadius(10)
                .opacity(0.8)
            Button("Next") {
                if index < sayings.count - 1 {
                    index += 1
                } else {
                    index = 0
                }
            }
            .buttonStyle(.borderedProminent)
        }
        .padding()
        .background(
            Image("001")
        )
    }
}
```

ในตัวอย่างนี้ มุมมอง `Text` ถูกเพิ่มพื้นหลังสีขาวแบบโปร่งแสงและมุมโค้ง ปุ่ม `Button` ใช้สไตล์ `.borderedProminent` และ `VStack` ถูกกำหนดพื้นหลังเป็นรูปภาพ

เสริมความรู้: เมื่อใช้ตัวปรับแต่ง `background()` กับภาพเป็นพื้นหลัง พฤติกรรมเริ่มต้นของมันคือพยายามเติมเต็มพื้นที่เลย์เอาต์ของวิวปัจจุบันให้มากที่สุด ในหลายกรณี มันสามารถขยายไปครอบคลุมพื้นที่ปลอดภัย (Safe Area) ได้อย่างเป็นธรรมชาติ

ตอนนี้เราก็สร้างมุมมองการเลื่อนแสดงคำคมเสร็จแล้ว

## สรุป

ผ่านบทเรียนนี้ เราได้เรียนรู้วิธีใช้อาร์เรย์เพื่อเก็บคำคมหลายข้อความ และใช้คำสั่ง `if` กับ `if-else` เพื่อสร้างการเลื่อนแสดงคำคม

เรายังได้เข้าใจการใช้งานพื้นฐานของอาร์เรย์ เช่น การเพิ่ม การลบ การแก้ไของค์ประกอบ และวิธีป้องกันไม่ให้ดัชนีของอาร์เรย์เกินขอบเขต

บทเรียนนี้ไม่เพียงอธิบายวิธีสร้างฟังก์ชันเลื่อนคำคมเท่านั้น แต่ยังผสานการใช้งานพื้นฐานของอาร์เรย์และคำสั่งเงื่อนไขเข้าไว้ด้วยกัน ช่วยให้เราพัฒนาความสามารถในการจัดการข้อมูลและควบคุมลำดับการทำงานของโปรแกรม

## ความรู้เพิ่มเติม - การตรวจสอบหลายเงื่อนไข: คำสั่ง if-else if-else

ในการพัฒนาจริง เรามักต้องจัดการกับหลายเงื่อนไข ตัวอย่างเช่น ในเกมหนึ่ง เมื่อคะแนนเท่ากับ 1 ให้เกิดเหตุการณ์ A เท่ากับ 2 ให้เกิดเหตุการณ์ B เท่ากับ 3 ให้เกิดเหตุการณ์ C และเป็นแบบนี้ต่อไป

เมื่อเจอสถานการณ์ที่มีทางแยกมากกว่าสองเงื่อนไข เราจำเป็นต้องใช้คำสั่ง `if-else if-else` เพื่อทำการตรวจสอบหลายระดับ

ไวยากรณ์พื้นฐาน:

```swift
if conditionA {
    // โค้ดที่จะทำงานเมื่อ conditionA เป็น true
} else if conditionB {
    // โค้ดที่จะทำงานเมื่อ conditionB เป็น true
} else if conditionC {
    // โค้ดที่จะทำงานเมื่อ conditionC เป็น true
} else {
    // โค้ดที่จะทำงานเมื่อไม่มีเงื่อนไขใดเป็นจริง
}
```

ในกรณีนี้ โปรแกรมจะตรวจสอบเงื่อนไขตามลำดับ และทำงานตามเงื่อนไขแรกที่เป็นจริง หากไม่มีเงื่อนไขใดเป็นจริง ก็จะทำงานในส่วน `else`

ในระบบเลื่อนคำคม เราก็สามารถใช้คำสั่ง `if-else if-else` เพื่อตรวจสอบหลายเงื่อนไขได้เช่นกัน:

```swift
Button("Next") {
    if index == 0 {
        index += 1
    } else if index == 1 {
        index += 1
    } else if index == 2 {
        index += 1
    } else if index == 3 {
        index += 1
    } else if index == 4 {
        index = 0
    } else {
        index = 0
    }
}
```

เมื่อ `index` เป็น 0, 1, 2, 3 ทุกครั้งที่กดปุ่มจะทำ `index += 1` และเมื่อ `index` เป็น 4 (องค์ประกอบสุดท้าย) จะรีเซ็ต `index` กลับเป็น 0 เพื่อให้เกิดการวนลูป

ส่วน `else` ใช้เป็นกรณีสำรอง เพื่อป้องกันค่าที่ไม่ถูกต้อง (เช่น `index` ถูกแก้ไขผิดพลาด)

สิ่งที่ต้องระวังคือ `==` ที่ใช้ตรงนี้หมายถึงการตรวจสอบว่า “เท่ากันหรือไม่” ถ้าคำสั่ง `if` ตรวจสอบว่า `index` เท่ากับตัวเลขใดตัวเลขหนึ่ง ก็จะได้ค่า `true` และทำงานในบล็อกโค้ดถัดไป ถ้าไม่เท่ากัน ก็จะไปตรวจสอบเงื่อนไข `if` ถัดไป

การตรวจสอบหลายเงื่อนไขแบบนี้มีประโยชน์มาก เพราะช่วยให้เราสามารถรันโค้ดที่แตกต่างกันได้ในแต่ละสถานการณ์ของเงื่อนไขต่าง ๆ
