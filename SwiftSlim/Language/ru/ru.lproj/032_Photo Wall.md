# Фотогалерея

В этом уроке мы создадим страницу фотогалереи.

![view](../../../Resource/032_view13.png)

В этом примере мы повторим знания об `extension`, `self` и экземплярах, а также изучим часто используемые в SwiftUI графические типы `Shape`.

С помощью этих знаний мы сможем обрезать обычные изображения в разные формы, например круг, скруглённый прямоугольник, капсулу и эллипс, а также добавлять к фотографиям эффект рамки.

В этом уроке будут использоваться несколько новых понятий: `Shape`, `clipShape`, `strokeBorder` и `overlay`.

Среди них `clipShape` используется для обрезки формы представления, `strokeBorder` — для рисования рамки фигуры, а `overlay` — для наложения нового представления поверх текущего представления.

## Отображение фотографий

Сначала отобразим 5 фотографий в `ContentView`:

```swift
struct ContentView: View {
    var body: some View {
        ScrollView {
            VStack(spacing: 50) {
                Image("1")
                    .resizable()
                    .scaledToFit()
                    .frame(width: 300)

                Image("2")
                    .resizable()
                    .scaledToFit()
                    .frame(width: 300)

                Image("3")
                    .resizable()
                    .scaledToFit()
                    .frame(width: 300)

                Image("4")
                    .resizable()
                    .scaledToFit()
                    .frame(width: 300)

                Image("5")
                    .resizable()
                    .scaledToFit()
                    .frame(width: 300)
            }
            .padding(.vertical, 100)
        }
        .ignoresSafeArea()
    }
}
```

Результат отображения:

![view](../../../Resource/032_view.png)

В этом коде мы используем `ScrollView`, чтобы сделать список фотографий прокручиваемым, и используем `VStack`, чтобы расположить несколько изображений сверху вниз.

Каждое изображение использует `.resizable()`, `.scaledToFit()` и `.frame(width: 300)` для настройки отображения: изображение может масштабироваться, сохранять пропорции и иметь ограниченную ширину.

`.padding(.vertical, 100)` у `VStack` используется для увеличения верхнего и нижнего отступа, чтобы первая и последняя фотографии не были слишком близко к краям экрана.

Последний `.ignoresSafeArea()` означает, что прокручиваемое представление игнорирует безопасную область, поэтому фотографии при прокрутке могут доходить до верхнего и нижнего края экрана, а страница выглядит более цельной.

Однако сейчас все 5 изображений используют одинаковые модификаторы:

```swift
.resizable()
.scaledToFit()
.frame(width: 300)
```

Когда мы видим повторяющийся код, можно рассмотреть использование `extension` для его упорядочивания.

## Использование extension для упорядочивания стиля изображений

Мы можем расширить `Image` специальным методом для фотогалереи:

```swift
extension Image {
    func photoGalleryStyle() -> some View {
        self
            .resizable()
            .scaledToFit()
            .frame(width: 300)
    }
}
```

Здесь мы расширяем тип `Image` методом `photoGalleryStyle`.

Этот метод содержит ранее повторявшиеся `.resizable()`, `.scaledToFit()` и `.frame(width: 300)`. Другими словами, он объединяет код масштабирования изображения, пропорционального отображения и настройки ширины.

После использования этого расширения исходный код изображения:

```swift
Image("1")
    .resizable()
    .scaledToFit()
    .frame(width: 300)
```

можно упростить до:

```swift
Image("1")
    .photoGalleryStyle()
```

Таким образом, каждому изображению достаточно вызвать `.photoGalleryStyle()`, чтобы применить одинаковый стиль фотогалереи. Код становится чище, и его удобнее единообразно изменять позже.

## Понимание self

В методе расширения мы написали `self`:

```swift
extension Image {
    func photoGalleryStyle() -> some View {
        self
            .resizable()
            .scaledToFit()
            .frame(width: 300)
    }
}
```

Здесь `self` обозначает текущий экземпляр изображения, который вызывает этот метод.

Например:

```swift
Image("1")
    .photoGalleryStyle()
```

В этом коде `photoGalleryStyle()` вызывается объектом `Image("1")`, поэтому `self` внутри метода обозначает именно это изображение `Image("1")`.

Проще можно понять так: кто вызывает этот метод, того и обозначает `self`.

Важно заметить, что модификаторы SwiftUI постоянно возвращают новые результаты представления. Поэтому `photoGalleryStyle()` возвращает изменённое представление, а не просто исходный `Image` сам по себе.

## Формы фотографий

Теперь фотографии уже могут нормально отображаться. Однако по умолчанию они прямоугольные и выглядят довольно обычно.

Если нужно только добавить фотографиям скруглённые углы, можно напрямую использовать `.cornerRadius()`:

```swift
Image("1")
    .photoGalleryStyle()
    .cornerRadius(20)
```

Результат отображения:

![color](../../../Resource/032_view1.png)

`.cornerRadius(20)` означает добавление изображению скругления углов на `20 pt`. Для обычных изображений со скруглёнными углами такой записи уже достаточно.

Однако `.cornerRadius()` может обрабатывать только эффект скругления углов и подходит для создания обычных изображений со скруглёнными углами. Если мы хотим придать фотографиям больше разных форм, одного этого модификатора будет недостаточно.

В этот момент нужно использовать `Shape` в SwiftUI. `Shape` может представлять разные фигуры, а в сочетании с `clipShape` позволяет обрезать изображение по соответствующей форме.

## Знакомство с Shape

В SwiftUI `Shape` представляет фигуру. Как и `View`, это очень часто используемый тип в SwiftUI.

К распространённым типам `Shape` относятся круг, прямоугольник, скруглённый прямоугольник, капсула и эллипс. Чтобы нагляднее наблюдать внешний вид разных фигур, в следующих примерах каждой фигуре заданы разные цвета и размеры.

В этих примерах `.fill()` используется для заливки фигуры цветом, а `.frame()` — для настройки размера отображения фигуры. Цвета нужны только для различения разных фигур и не являются фиксированными цветами самих фигур.

### Circle Круг

`Circle` представляет круг. Он часто используется для аватаров, круглых кнопок, круглых изображений и других интерфейсных эффектов.

```swift
Circle()
    .fill(Color.red)
    .frame(width: 200, height: 200)
```

![circle](../../../Resource/032_view3.png)

### Rectangle Прямоугольник

`Rectangle` представляет прямоугольник. Это одна из самых базовых фигур, которую также можно использовать для создания фонов, разделённых областей или обычных рамок.

```swift
Rectangle()
    .fill(Color.green)
    .frame(width: 150, height: 100)
```

![rectangle](../../../Resource/032_view4.png)

### RoundedRectangle Скруглённый прямоугольник

`RoundedRectangle` представляет скруглённый прямоугольник. `cornerRadius` используется для настройки радиуса скругления.

```swift
RoundedRectangle(cornerRadius: 20)
    .fill(Color.blue)
    .frame(width: 150, height: 100)
```

![rectangle](../../../Resource/032_view5.png)

### Capsule Капсула

`Capsule` представляет форму капсулы. Её два конца имеют дугообразную форму, и она часто используется для капсульных кнопок, фонов меток и других интерфейсных эффектов.

```swift
Capsule()
    .fill(Color.orange)
    .frame(width: 150, height: 100)
```

![rectangle](../../../Resource/032_view6.png)

### Ellipse Эллипс

`Ellipse` представляет эллипс. Он похож на `Circle`, но когда ширина и высота различаются, отображается как эллипс.

```swift
Ellipse()
    .fill(Color.purple)
    .frame(width: 150, height: 100)
```

![rectangle](../../../Resource/032_view7.png)

Из этих примеров видно, что `Shape` сам по себе может напрямую отображаться как фигура. Далее мы будем использовать эти типы `Shape` вместе с `clipShape`, чтобы обрезать фотографии и показывать их в разных формах.

## Использование clipShape для обрезки фотографий

Теперь мы можем использовать `clipShape`, чтобы обрезать фотографии в разные формы.

Например, обрежем первую фотографию в круг:

```swift
Image("1")
    .photoGalleryStyle()
    .clipShape(Circle())
```

Этот код означает: сначала отобразить изображение, а затем с помощью `Circle()` обрезать его в круг.

![view](../../../Resource/032_view2.png)

Базовая запись `clipShape` выглядит так:

```swift
.clipShape(форма)
```

Добавьте `.clipShape(...)` после представления, а в скобках укажите фигуру, в которую нужно обрезать представление.

Например:

```swift
.clipShape(Circle())
.clipShape(Rectangle())
.clipShape(RoundedRectangle(cornerRadius: 20))
.clipShape(Capsule())
.clipShape(Ellipse())
```

Теперь обрежем 5 фотографий в разные формы:

```swift
struct ContentView: View {
    var body: some View {
        ScrollView {
            VStack(spacing: 50) {
                Image("1")
                    .photoGalleryStyle()
                    .clipShape(Circle())

                Image("2")
                    .photoGalleryStyle()
                    .clipShape(Rectangle())

                Image("3")
                    .photoGalleryStyle()
                    .clipShape(RoundedRectangle(cornerRadius: 20))

                Image("4")
                    .photoGalleryStyle()
                    .clipShape(Capsule())

                Image("5")
                    .photoGalleryStyle()
                    .clipShape(Ellipse())
            }
            .padding(.vertical, 100)
        }
        .ignoresSafeArea()
    }
}
```

Результат отображения:

![view](../../../Resource/032_view8.png)

Среди них результат после обрезки с помощью `Rectangle()` близок к обычной прямоугольной фотографии, поэтому визуальное изменение неочевидно. Он главным образом используется для сравнения с другими формами.

Теперь фотографии больше не являются просто обычными прямоугольниками: у них появились разные формы.

## Добавление рамок к фотографиям

Если мы хотим добавить рамку к круглой фотографии, можно подумать об использовании `border`:

```swift
Image("1")
    .photoGalleryStyle()
    .clipShape(Circle())
    .border(Color.gray, width: 5)
```

Однако обычно в результате получается не круглая рамка, а прямоугольная рамка.

![view](../../../Resource/032_view9.png)

Это происходит потому, что `border` добавляет рамку по прямоугольной области представления, а не по форме после обрезки `clipShape`.

Поэтому, если нам нужна круглая рамка, нельзя напрямую использовать `border`.

## Использование strokeBorder для рисования рамок фигур

В SwiftUI `strokeBorder` может рисовать внутренние рамки для этих распространённых типов `Shape`.

Например, нарисуем круглую рамку:

```swift
Circle()
    .strokeBorder(Color.brown, lineWidth: 10)
    .frame(width: 240, height: 240)
```

Этот код означает: нарисовать для `Circle` коричневую рамку шириной `10 pt`.

![view](../../../Resource/032_view12.png)

Здесь нужно обратить внимание: `strokeBorder` не добавляет рамку непосредственно к фотографии, а рисует рамку для фигуры `Circle()`.

Другими словами, этот код создаёт только независимую круглую рамку, и пока она не связана с фотографией.

Если мы хотим, чтобы эта круглая рамка отображалась поверх фотографии, нужно продолжить использовать `overlay`, чтобы наложить рамку на фотографию.

## Использование overlay для наложения рамки

`overlay` — это модификатор представления, который может накладывать новое представление поверх текущего представления.

Его базовую структуру можно понять так:

```swift
Текущее представление
    .overlay {
        Накладываемое представление
    }
```

В этом примере текущее представление — это фотография, уже обрезанная в круг:

```swift
Image("1")
    .photoGalleryStyle()
    .clipShape(Circle())
```

![view](../../../Resource/032_view2.png)

Новое представление, которое мы хотим наложить, — это круглая рамка:

```swift
Circle()
    .strokeBorder(Color.brown, lineWidth: 10)
```

![view](../../../Resource/032_view12.png)

Поэтому можно объединить фотографию и рамку:

```swift
Image("1")
    .photoGalleryStyle()
    .clipShape(Circle())
    .overlay {
        Circle()
            .strokeBorder(Color.brown, lineWidth: 10)
    }
```

Здесь `overlay` означает наложение нового представления поверх текущего представления.

В `overlay` рамка накладывается в соответствии с областью отображения текущей фотографии, поэтому отдельно задавать `frame` не нужно. Если форма рамки совпадает с формой обрезки, рамка будет точно прилегать к фотографии.

![view](../../../Resource/032_view10.png)

Итоговый эффект: поверх круглой фотографии располагается круглая рамка.

По сравнению с `ZStack`, `overlay` больше подходит для сценария «добавить оформление к текущему представлению». Фотография является основным объектом, а рамка — только дополнительным эффектом, поэтому использование `overlay` делает код понятнее.

## Завершение фотогалереи

Теперь добавим каждой фотографии соответствующую форму и рамку:

```swift
import SwiftUI

struct ContentView: View {
    var body: some View {
        ScrollView {
            VStack(spacing: 50) {
                Image("1")
                    .photoGalleryStyle()
                    .clipShape(Circle())
                    .overlay {
                        Circle()
                            .strokeBorder(Color.brown, lineWidth: 20)
                    }

                Image("2")
                    .photoGalleryStyle()
                    .clipShape(Rectangle())
                    .overlay {
                        Rectangle()
                            .strokeBorder(Color.green, lineWidth: 20)
                    }

                Image("3")
                    .photoGalleryStyle()
                    .clipShape(RoundedRectangle(cornerRadius: 20))
                    .overlay {
                        RoundedRectangle(cornerRadius: 20)
                            .strokeBorder(Color.blue, lineWidth: 20)
                    }

                Image("4")
                    .photoGalleryStyle()
                    .clipShape(Capsule())
                    .overlay {
                        Capsule()
                            .strokeBorder(Color.orange, lineWidth: 20)
                    }

                Image("5")
                    .photoGalleryStyle()
                    .clipShape(Ellipse())
                    .overlay {
                        Ellipse()
                            .strokeBorder(Color.yellow, lineWidth: 20)
                    }
            }
            .padding(.vertical, 100)
        }
        .ignoresSafeArea()
    }
}
```

Результат отображения:

![view](../../../Resource/032_view13.png)

Так базовая страница фотогалереи завершена.

На этой странице мы используем `ScrollView` для реализации прокручиваемого отображения, `Image` — для показа фотографий, `clipShape` — для обрезки формы фотографий, а `overlay` и `strokeBorder` — для добавления рамок.

## Полный код

```swift
import SwiftUI

struct ContentView: View {
    var body: some View {
        ScrollView {
            VStack(spacing: 50) {
                Image("1")
                    .photoGalleryStyle()
                    .clipShape(Circle())
                    .overlay {
                        Circle()
                            .strokeBorder(Color.brown, lineWidth: 20)
                    }

                Image("2")
                    .photoGalleryStyle()
                    .clipShape(Rectangle())
                    .overlay {
                        Rectangle()
                            .strokeBorder(Color.green, lineWidth: 20)
                    }

                Image("3")
                    .photoGalleryStyle()
                    .clipShape(RoundedRectangle(cornerRadius: 20))
                    .overlay {
                        RoundedRectangle(cornerRadius: 20)
                            .strokeBorder(Color.blue, lineWidth: 20)
                    }

                Image("4")
                    .photoGalleryStyle()
                    .clipShape(Capsule())
                    .overlay {
                        Capsule()
                            .strokeBorder(Color.orange, lineWidth: 20)
                    }

                Image("5")
                    .photoGalleryStyle()
                    .clipShape(Ellipse())
                    .overlay {
                        Ellipse()
                            .strokeBorder(Color.yellow, lineWidth: 20)
                    }
            }
            .padding(.vertical, 100)
        }
        .ignoresSafeArea()
    }
}

extension Image {
    func photoGalleryStyle() -> some View {
        self
            .resizable()
            .scaledToFit()
            .frame(width: 300)

}

```

## Итоги

В этом уроке мы завершили страницу фотогалереи.

В этом примере мы сначала использовали `Image` для отображения фотографий, а затем с помощью `extension` расширили `Image` методом `photoGalleryStyle`, чтобы упорядочить повторяющийся код стиля изображений.

Затем мы изучили распространённые типы `Shape` в SwiftUI, например `Circle`, `Rectangle`, `RoundedRectangle`, `Capsule` и `Ellipse`. Эти фигуры можно не только напрямую отображать, но и использовать вместе с `clipShape` для обрезки представлений.

Например:

```swift

```

означает обрезку представления в круг.

В конце мы использовали `overlay` и `strokeBorder`, чтобы добавить фотографиям рамки соответствующей формы. Важно отметить, что `border` обычно рисует рамку по прямоугольной области представления. Если нужно, чтобы рамка следовала кругу, капсуле или эллипсу, больше подходит наложение такого же `Shape`.

С помощью этого урока мы не только завершили эффект фотогалереи, но и поняли распространённый способ совместного использования `Shape`, `clipShape`, `strokeBorder` и `overlay` в интерфейсах.
