# משתנים וקבועים

בשיעור הזה נלמד על משתנים (`variables`) וקבועים (`constants`) ב-Swift, ונכיר גם כמה סוגי נתונים נפוצים ואופרטורים בסיסיים.

אלה היסודות הבסיסיים ביותר בתכנות, והם גם בסיס חשוב מאוד להמשך הלמידה של SwiftUI.

## משתנים וקבועים בחיי היום־יום

אפשר להבין משתנים וקבועים דרך דברים שקורים בחיים האמיתיים.

משתנים בחיים:

- התוכן שמוקרן בכל תוכנית בטלוויזיה שונה
- מזג האוויר משתנה מיום ליום
- מחוגי השעון זזים בכל שנייה

המאפיין המשותף לכל הדברים האלה הוא: הם משתנים.

אם בטלוויזיה היה תמיד אותו מסך, מזג האוויר היה תמיד בהיר, ומחוגי השעון לא היו זזים לעולם, הדברים הללו היו נחשבים לקבועים.

אחד עשוי להשתנות, והשני לא.

## להבין משתנים וקבועים

בפיתוח אפליקציות, לעיתים קרובות צריך שהמשתמש ימלא או ישמור מידע כלשהו.

למשל:

- שם החשבון
- תאריך לידה
- פרטי קשר
- כתובת

המידע הזה נשמר ולאחר מכן מוצג.

לדוגמה, אם משתמש מזין שם באפליקציה:

```
FangJunyu
```

אנחנו צריכים לשמור את השם הזה, כדי שנוכל להציג אותו באפליקציה.

אפשר להבין את תהליך השמירה בצורה פשוטה כמעין הנחת חפץ בתוך מגירה.

כאשר אנחנו שומרים שם, זה כאילו הכנסנו את השם למגירה.

יכול להיות שיש הרבה פריטים שצריך לשמור, ולכן יהיו גם הרבה מגירות. כדי לדעת מה יש בתוך כל מגירה, צריך לתת לה שם.

למשל:

```
name
```

בדוגמה הזאת, `name` הוא השם, ו-`FangJunyu` הוא המידע שנשמר.

![Var](../../RESOURCE/007_var.png)

**ב-Swift, אם רוצים לשמור נתונים, חייבים להצהיר עליהם באמצעות משתנה (`var`) או קבוע (`let`).**

מכיוון ששם הוא בדרך כלל משהו שיכול להשתנות, נשתמש כאן במשתנה.

```swift
var name = "FangJunyu"
```

כאן הוצהר משתנה בשם `name`, מסוג `String`, ובעל ערך `"FangJunyu"`.

### ההבדל בין משתנים לקבועים

משתנים מוכרזים בעזרת `var`:

```swift
var
```

קבועים מוכרזים בעזרת `let`:

```swift
let
```

למשל:

```swift
var name = "FangJunyu"
let id = 123456
```

אם מגדירים ערך כמשתנה, אפשר לשנות אותו אחר כך; אם מגדירים אותו כקבוע, אי אפשר לשנות אותו.

לכן ההבדל המרכזי בין משתנה לקבוע הוא: האם מותר לערך להשתנות.

### שינוי קבוע

אם הקוד מנסה לשנות את הערך של קבוע:

```swift
let name = "Sam"
name = "Bob"
```

Swift תציג שגיאה שאומרת שלא ניתן לבצע השמה, כי זהו קבוע.

```
Cannot assign to property: 'name' is a 'let' constant
```

המנגנון הזה עוזר למפתחים להימנע משינוי בטעות של נתונים חשובים.

### הצגה ב-SwiftUI

פתחו את הקובץ `ContentView.swift` והצהירו על משתנה בתוך ה-`View`:

```swift
struct ContentView: View {
    var name = "FangJunyu"

    var body: some View {
        Text(name)
    }
}
```

כאן הוצהר משתנה בשם `name`, ובאמצעות `Text` הוא מוצג על המסך.

אם נשנה את תוכן המשתנה:

```swift
struct ContentView: View {
    var name = "Sam"
    var body: some View {
        Text(name)
    }
}
```

גם הטקסט שמוצג ב-`Text` ישתנה.

בדרך הזאת אפשר להשתמש במשתנים כדי לשלוט במה שמוצג בממשק, בלי לשנות בכל פעם ידנית את הטקסט שבתוך `Text`.

אם במקום אחר בקוד נצטרך לקבל את המידע הזה, נוכל גם להעביר לשם את המשתנה, כאילו העברנו את התוכן של “המגירה” למקום אחר לשימוש.

### מיקום משתנים וקבועים ב-SwiftUI

ב-SwiftUI נהוג לכתוב משתנים וקבועים מחוץ ל-`body`:

```swift
struct ContentView: View {
    var name = "FangJunyu"

    var body: some View {
        Text(name)
    }
}
```

כמובן, אפשר גם לכתוב משתנים וקבועים בתוך ה-`body`:

```swift
struct ContentView: View {
    var body: some View {
        var name = "FangJunyu"
        Text(name)
    }
}
```

אבל יש ביניהם הבדל חשוב:

- אם המשתנים והקבועים כתובים בתוך ה-`body`, הם ייווצרו מחדש בכל פעם שהתצוגה תחושב מחדש
- אם הם כתובים מחוץ ל-`body`, הם קיימים כתכונות של מבנה התצוגה, ולכן מבנה הקוד ברור יותר

לכן, בפיתוח בפועל נהוג בדרך כלל לכתוב משתנים וקבועים מחוץ ל-`body`.

## סוגי נתונים

משתנים יכולים לשמור הרבה סוגים שונים של נתונים, והסוגים האלה נקראים סוגי נתונים (`data types`).

למשל, קודם השתמשנו ב-`Text` כדי להציג טקסט:

```swift
Text("Hello, World")
```

כאן `"Hello, World"` הוא מסוג מחרוזת (`String`).

מחרוזת משמשת לייצוג תוכן טקסטואלי, וב-Swift חייבים לעטוף אותה במרכאות כפולות `""`.

למשל:

```swift
var hello = "Hello, World"
```

מלבד מחרוזות, יש ב-Swift עוד הרבה סוגי נתונים נוספים.

בשלב ההתחלתי, ארבעת הסוגים הנפוצים ביותר הם:

- `String`
- `Int`
- `Double`
- `Bool`

ארבעת הסוגים האלו מספיקים כדי לבצע את רוב עיבוד הנתונים הבסיסי.

**`String`**

`String` מייצג תוכן טקסטואלי, למשל:

```swift
var name = "FangJunyu"
var city = "Rizhao"
```

מחרוזות משמשות בדרך כלל לייצוג שמות משתמש, כותרות ותוכן טקסטואלי.

**`Int`**

`Int` מייצג מספר שלם, כלומר מספר בלי חלק עשרוני.

למשל:

```swift
var age = 26
var count = 100
```

מספרים שלמים משמשים בדרך כלל לייצוג גיל, כמות וספירה.

**`Double`**

`Double` מייצג מספר שיש בו חלק עשרוני.

למשל:

```swift
var weight = 74.5
var height = 185.0
```

ב-Swift, מספרים עשרוניים הם בדרך כלל מסוג `Double` כברירת מחדל.

יש גם סוג דומה בשם `Float`, אבל הטווח שהוא מסוגל לייצג קטן יותר, ולכן בפיתוח בפועל נהוג יותר להשתמש ב-`Double`.

**`Bool`**

הסוג `Bool` משמש לייצוג שתי אפשרויות: `true` ו-`false`.

למשל:

```swift
var isShowAlert = true
var isLogin = false
```

הסוג `Bool` משמש לעיתים קרובות לבדיקות תנאי. למשל, האם להציג חלון התראה.

למצבים שיש בהם רק שתי תוצאות אפשריות, `Bool` הוא בחירה מצוינת.

### הצגה ב-SwiftUI

כאשר מציגים ערך מסוג `String`, אפשר להשתמש ישירות ב-`Text`:

```swift
Text(name)
```

אבל טיפוסים כמו `Int` ו-`Double` אינם מחרוזות, ולכן אי אפשר להשתמש בהם ישירות כחלק מתוכן טקסטואלי.

אם רוצים להציג אותם בתוך `Text`, אפשר להשתמש באינטרפולציה של מחרוזות (`String Interpolation`):

```swift
\()
```

אינטרפולציה של מחרוזות צריכה להופיע בתוך מחרוזת, ולשים את המשתנה או הקבוע בתוך `\()`.

למשל:

```swift
var num = 1

Text("\(num)")
Text("DoubleNum: \(num * 2)")
```

בקוד למעלה, `""` מייצג מחרוזת, ו-`\()` משמש להכנסת משתנה או קבוע לתוך המחרוזת.

למשל:

```swift
"DoubleNum: \(num * 2)"
```

התוכן שיוצג יהיה:

```swift
DoubleNum: 2
```

כך אפשר לשלב משתנים וקבועים בתוך מחרוזת, ואז להציג אותם באמצעות `Text`.

טיפ: `\()` ניתן לשימוש רק בתוך מחרוזת `""`.

## אופרטורים

כאשר מצהירים על משתנה או קבוע, אפשר לראות כתיבה כמו זו:

```swift
var num = 1
```

הסימן `=` נקרא אופרטור השמה.

התפקיד שלו הוא: לקחת את הערך שבצד ימין ולהציב אותו במשתנה שבצד שמאל.

בדוגמה הזאת, `1` הוצב במשתנה `num`.

מלבד אופרטור ההשמה, יש גם כמה אופרטורים נפוצים לחישובים מספריים:

- `+`
- `-`
- `*`
- `/`

כאשר מבצעים חישובים על מספרים, למשל על נתונים מסוג `Int` או `Double`, משתמשים באופרטורים האלו.

למשל:

```swift
var age = 21 + 3 // 24
var value = 30 - 2 // 28
var num = 1 * 3 // 3
var weight = 88.5 / 2 // 44.25
```

אחרי ביצוע החישוב, התוצאה נשמרת בתוך המשתנה.

## סיכום

משתנים, קבועים ואופרטורים הם מהמושגים הבסיסיים ביותר בתכנות.

באמצעות משתנים וקבועים אפשר לשמור נתונים מסוגים שונים בתוך התוכנית; באמצעות סוגי נתונים אפשר להבהיר באיזה סוג מידע מדובר; ובאמצעות אופרטורים אפשר לבצע חישובים ועיבוד על הנתונים.

הידע הזה הוא כמו ארגז הכלים הבסיסי של עולם התכנות. שליטה בו תניח יסוד מוצק להמשך הלמידה של Swift ושל SwiftUI.
