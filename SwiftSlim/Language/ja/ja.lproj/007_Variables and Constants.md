# 変数と定数

このレッスンでは、主に Swift における変数と定数を学び、よく使われるデータ型や基本的な演算子についても理解します。

これらの内容はプログラミングにおける最も基礎的な知識であり、今後 SwiftUI を学ぶ上でも重要な土台になります。

## 日常生活における変数と定数

変数と定数は、日常の物事を通して理解できます。

日常の変数の例：

- テレビでは番組ごとに放送内容が異なる
- 毎日の天気は違う
- 時計は毎秒変化する

これらの共通点は、「変化する」ということです。

もしテレビがずっと同じ画面しか映さず、天気がずっと晴れのままで、時計の針がまったく動かなければ、これらは定数になります。

一方は変化する可能性があり、もう一方は変化しません。

## 変数と定数を理解する

App 開発では、通常ユーザーがいくつかの情報を入力したり保存したりする必要があります。

たとえば：

- アカウント名
- 誕生日
- 連絡先
- 住所

これらの情報は保存され、表示されます。

たとえば、ユーザーが App の中で名前を入力したとします：

```
FangJunyu
```

私たちはこの名前を保存し、App の中で表示できるようにする必要があります。

この保存の過程は、簡単に言えば「引き出しの中に物を入れること」だと考えられます。

名前を保存するというのは、名前を引き出しの中に入れるようなものです。

保存するものはたくさんあるかもしれず、引き出しもたくさん必要になります。各引き出しに何が入っているか分かるようにするには、引き出しに名前を付ける必要があります。

たとえば：

```
name
```

この例では、`name` が名前であり、`FangJunyu` が保存された情報です。

![Var](../../RESOURCE/007_var.png)

**Swift では、データを保存したい場合、必ず変数（`var`）または定数（`let`）で宣言する必要があります。**

名前は通常変更可能なので、ここでは変数を使うべきです。

```swift
var name = "FangJunyu"
```

ここでは `name` という名前の変数が宣言され、型は `String`、値は `"FangJunyu"` です。

### 変数と定数の違い

変数は `var` で宣言します：

```swift
var
```

定数は `let` で宣言します：

```swift
let
```

たとえば：

```swift
var name = "FangJunyu"
let id = 123456
```

変数として宣言した場合、その値は変更できます。定数として宣言した場合は、あとから変更できません。

したがって、変数と定数の主な違いは「値の変更が許されるかどうか」です。

### 定数を変更する

もしコードが定数の値を変更しようとすると：

```swift
let name = "Sam"
name = "Bob"
```

Swift は、それが定数なので代入を完了できないと表示します。

```
Cannot assign to property: 'name' is a 'let' constant
```

この仕組みによって、重要なデータが誤って変更されるのを防ぐことができます。

### SwiftUI で表示する

ContentView.swift ファイルを開き、View の内部で変数を宣言します：

```swift
struct ContentView: View {
    var name = "FangJunyu"

    var body: some View {
        Text(name)
    }
}
```

ここでは `name` という変数が宣言され、それを `Text` を通して画面に表示しています。

もし変数の内容を変更すると：

```swift
struct ContentView: View {
    var name = "Sam"
    var body: some View {
        Text(name)
    }
}
```

`Text` に表示される内容も変わります。

このように、変数を使って画面に表示する内容を制御できるため、毎回 `Text` の中の文字列を直接変更する必要がありません。

もし他の場所でこの情報が必要になれば、変数をそこへ渡すこともできます。つまり「引き出し」の中身を他の場所に渡して使えるということです。

### SwiftUI における変数と定数の位置

SwiftUI では、変数と定数は通常 `body` の外側に書きます：

```swift
struct ContentView: View {
    var name = "FangJunyu"

    var body: some View {
        Text(name)
    }
}
```

もちろん、変数や定数を `body` の内側に書くこともできます：

```swift
struct ContentView: View {
    var body: some View {
        var name = "FangJunyu"
        Text(name)
    }
}
```

ただし、この 2 つには重要な違いがあります：

- 変数や定数を `body` の内側に書くと、ビューが再計算されるたびにその変数が作り直されます。
- `body` の外側に書くと、変数はビュー構造体のプロパティとして存在し、コード構造がより明確になります。

そのため、実際の開発では、変数と定数は通常 `body` の外側に書きます。

## データ型

変数にはさまざまな種類のデータを保存できます。これらの種類をデータ型と呼びます。

たとえば、以前 `Text` を使って文字を表示しました：

```swift
Text("Hello, World")
```

ここでの `"Hello, World"` は文字列（`String`）型です。

文字列はテキスト内容を表し、Swift では `""` のダブルクォーテーションで囲む必要があります。

たとえば：

```swift
var hello = "Hello, World"
```

文字列以外にも、Swift には多くのデータ型があります。

初心者段階で最もよく使う 4 つの型は次の通りです：

- `String`
- `Int`
- `Double`
- `Bool`

この 4 つの型で、大部分の基本的なデータ処理を行えます。

**String**

`String` はテキスト内容を表します。たとえば：

```swift
var name = "FangJunyu"
var city = "Rizhao"
```

文字列は通常、ユーザー名、タイトル、文章内容などを表すために使われます。

**Int**

`Int` は整数を表します。つまり、小数点のない整数です。

たとえば：

```swift
var age = 26
var count = 100
```

整数は通常、年齢、数量、カウントを表すために使われます。

**Double**

`Double` は小数を含む数値を表します。

たとえば：

```swift
var weight = 74.5
var height = 185.0
```

Swift では、小数は通常デフォルトで `Double` 型が使われます。

似た型に `Float` もありますが、表現できる数値範囲がより小さいため、実際の開発では `Double` の方がよく使われます。

**Bool**

`Bool` 型は、`true` と `false` の 2 つの状態を表します。

たとえば：

```swift
var isShowAlert = true
var isLogin = false
```

`Bool` 型は条件判断でよく使われます。たとえば、通知ダイアログを表示するかどうかなどです。

結果が 2 通りしかない状況を表すのに、`Bool` 型はとても適しています。

### SwiftUI で表示する

`String` 型を表示する場合は、直接 `Text` を使えます：

```swift
Text(name)
```

しかし、`Int` や `Double` などの型は文字列ではないため、そのままテキスト内容の一部として表示することはできません。

もし `Text` の中にこれらのデータを表示したいなら、文字列補間（`String Interpolation`）を使います：

```swift
\()
```

文字列補間は文字列の中で使い、変数や定数を `\()` で囲みます。

たとえば：

```swift
var num = 1

Text("\(num)")
Text("DoubleNum: \(num * 2)")
```

上のコードでは、`""` は文字列を表し、`\()` は変数や定数を文字列の中に差し込むために使われています。

たとえば：

```swift
"DoubleNum: \(num * 2)"
```

表示内容は：

```swift
DoubleNum: 2
```

このように、文字列の中に変数や定数を差し込み、それを `Text` で画面に表示できます。

ヒント：`\()` は文字列 `""` の内部でしか使えません。

## 演算子

変数や定数を宣言するとき、次のような書き方を見かけます：

```swift
var num = 1
```

ここでの `=` は代入演算子です。

その役割は、右側の値を左側の変数へ代入することです。

この例では、1 が変数 `num` に代入されています。

代入演算子のほかにも、いくつかよく使う数値演算子があります：

- `+`
- `-`
- `*`
- `/`

数値を計算するとき、たとえば `Int` や `Double` 型のデータに対して、これらの演算子を使います。

たとえば：

```swift
var age = 21 + 3 // 24
var value = 30 - 2 // 28
var num = 1 * 3 // 3
var weight = 88.5 / 2 // 44.25
```

計算が終わると、その結果は変数に保存されます。

## まとめ

変数、定数、そして演算子は、プログラミングにおける最も基本的な概念です。

変数と定数によって、プログラムの中にさまざまなデータを保存できます。データ型によって、そのデータが何であるかを明確にできます。演算子によって、そのデータに対して計算や処理を行えます。

これらの知識は、プログラミング世界における基礎ツールのようなものです。しっかり身につけることで、その後の Swift と SwiftUI 学習に堅実な土台を築けます。
