# மாறிகள் மற்றும் மாறாத மாறிகள்

இந்தப் பாடத்தில், Swift-இல் உள்ள மாறிகள் மற்றும் மாறாத மாறிகளைப் பற்றி முதன்மையாகக் கற்போம். அதோடு, பொதுவாகப் பயன்படுத்தப்படும் சில data type-களையும் அடிப்படை operator-களையும் அறிந்துகொள்வோம்.

இவை அனைத்தும் programming-இன் மிகவும் அடிப்படையான அறிவுகள்; மேலும், அடுத்தடுத்த SwiftUI கற்றலுக்கான முக்கியமான அடித்தளமாகவும் உள்ளன.

## வாழ்கையில் உள்ள மாறிகள் மற்றும் மாறாத மாறிகள்

நாம் வாழ்கையில் உள்ள சில பொருட்களை வைத்து, variable மற்றும் constant என்ன என்பதைப் புரிந்துகொள்ளலாம்.

வாழ்கையில் உள்ள மாறிகளின் உதாரணங்கள்:

* தொலைக்காட்சியில் ஒவ்வொரு நிகழ்ச்சியும் வேறு உள்ளடக்கத்தை ஒளிபரப்புகிறது
* ஒவ்வொரு நாளும் வானிலை மாறுபடும்
* கடிகாரத்தின் ஊசிகள் ஒவ்வொரு விநாடியும் மாறுகின்றன

இந்த விஷயங்களின் பொதுவான தன்மை என்னவென்றால்: அவை மாறக்கூடியவை.

ஒரு தொலைக்காட்சி எப்போதும் ஒரே படத்தைக் காட்டினால், வானிலை எப்போதும் வெயிலாகவே இருந்தால், கடிகாரத்தின் ஊசிகள் ஒருபோதும் நகராவிட்டால், அவை constant போன்றவை ஆகும்.

ஒன்று மாறக்கூடியது; மற்றொன்று மாறாதது.

## மாறிகள் மற்றும் மாறாத மாறிகளைப் புரிந்துகொள்வது

App development-இல், பொதுவாக பயனர் சில தகவல்களை உள்ளிடவோ சேமிக்கவோ வேண்டும்.

உதாரணமாக:

* கணக்கு பெயர்
* பிறந்த நாள்
* தொடர்பு விபரம்
* முகவரி

இந்தத் தகவல்கள் சேமிக்கப்பட்டு, பின்னர் காட்டப்படும்.

உதாரணமாக, பயனர் App-இல் ஒரு பெயரை உள்ளிடுகிறார்:

```id="gign42"
FangJunyu
```

அந்தப் பெயரை App-இல் காட்டுவதற்காக, நாங்கள் அதைச் சேமிக்க வேண்டும்.

இந்தச் சேமிப்பு செயல்முறையை எளிதாகப் புரிந்துகொள்ள, “ஒரு அலமாரி çek்ட்டியில் பொருளை வைப்பது” போல நினைக்கலாம்.

நாம் பெயரைச் சேமிக்கும் போது, அது அந்தப் பெயரை ஒரு çek்ட்டிக்குள் வைப்பதுபோன்றது.

சேமிக்க வேண்டிய பொருட்கள் பல இருக்கும்; அதனால் çek்ட்டிகளும் பல இருக்கும். ஒவ்வொரு çek்ட்டியிலும் என்ன இருக்கிறது என்பதை அறிய, அந்த çek்ட்டிக்கு ஒரு பெயர் தேவைப்படும்.

உதாரணமாக:

```id="g8gcl5"
name
```

இந்த உதாரணத்தில், `name` என்பது பெயர்; `FangJunyu` என்பது சேமிக்கப்பட்ட தகவல்.

![Var](../../RESOURCE/007_var.png)

**Swift-இல், தரவைச் சேமிக்க வேண்டுமெனில், கட்டாயமாக variable (`var`) அல்லது constant (`let`) கொண்டு அதை அறிவிக்க வேண்டும்.**

பெயர் பொதுவாக மாற்றப்படக்கூடியது என்பதால், இங்கு variable பயன்படுத்துவது சரியானது.

```swift id="2s86mf"
var name = "FangJunyu"
```

இங்கே `name` என்ற பெயருடைய ஒரு variable அறிவிக்கப்பட்டுள்ளது. அதன் type `String`, அதன் value `"FangJunyu"` ஆகும்.

### மாறிகள் மற்றும் மாறாத மாறிகளுக்கிடையேயான வித்தியாசம்

மாறிகள் `var` மூலம் அறிவிக்கப்படுகின்றன:

```swift id="3rpj4g"
var
```

மாறாத மாறிகள் `let` மூலம் அறிவிக்கப்படுகின்றன:

```swift id="fkcfxh"
let
```

உதாரணமாக:

```swift id="xhputj"
var name = "FangJunyu"
let id = 123456
```

ஒரு value variable ஆக அறிவிக்கப்பட்டால், அதை பின்னர் மாற்றலாம்; constant ஆக அறிவிக்கப்பட்டால், அதை மாற்ற முடியாது.

அதனால், variable மற்றும் constant-இன் முக்கிய வித்தியாசம்: அதன் value மாற்றப்படலாமா இல்லையா என்பதுதான்.

### constant-ஐ மாற்றுதல்

குறியீடு constant-இன் value-ஐ மாற்ற முயன்றால்:

```swift id="pvartp"
let name = "Sam"
name = "Bob"
```

Swift, assignment செய்ய முடியாது என்று பிழை காட்டும்; ஏனெனில் அது ஒரு constant ஆகும்.

```id="4k2jck"
Cannot assign to property: 'name' is a 'let' constant
```

இந்த முறை, முக்கியமான சில தரவுகள் தவறுதலாக மாற்றப்படுவதைத் தடுக்க developers-க்கு உதவுகிறது.

### SwiftUI-இல் காட்டுதல்

`ContentView.swift` கோப்பைத் திறந்து, View-க்குள் ஒரு variable அறிவிக்கலாம்:

```swift id="gts19m"
struct ContentView: View {
    var name = "FangJunyu"

    var body: some View {
        Text(name)
    }
}
```

இங்கே `name` என்ற variable அறிவிக்கப்பட்டுள்ளது; அதை `Text` மூலம் interface-இல் காட்டுகிறோம்.

அந்த variable-இன் value-ஐ மாற்றினால்:

```swift id="gne38j"
struct ContentView: View {
    var name = "Sam"
    var body: some View {
        Text(name)
    }
}
```

`Text`-இல் காட்டப்படும் உரையும் மாறிவிடும்.

இந்த முறையில், நாம்ச் variable-ஐப் பயன்படுத்தி interface-இல் காட்டப்படும் உள்ளடக்கத்தை கட்டுப்படுத்தலாம்; ஒவ்வொரு முறையும் நேரடியாக `Text`-இன் உரையை மாற்றத் தேவையில்லை.

வேறு இடங்களில் இந்தத் தகவல் தேவைப்பட்டாலும், அந்த variable-ஐ அங்கு அனுப்பலாம். அது, çek்ட்டிக்குள் வைத்துள்ள பொருளை வேறு இடத்தில் பயன்படுத்தக் கொடுப்பது போன்றது.

### SwiftUI-இல் variable மற்றும் constant எழுதும் இடம்

SwiftUI-யில், variable-களும் constant-களும் பொதுவாக `body`-க்கு வெளியே எழுதப்படுகின்றன:

```swift id="9pxglr"
struct ContentView: View {
    var name = "FangJunyu"

    var body: some View {
        Text(name)
    }
}
```

நிச்சயமாக, அவற்றை `body`-க்குள் கூட எழுதலாம்:

```swift id="3eb4pw"
struct ContentView: View {
    var body: some View {
        var name = "FangJunyu"
        Text(name)
    }
}
```

ஆனால், இந்த இரண்டுக்கும் ஒரு முக்கியமான வேறுபாடு உள்ளது:

* variable அல்லது constant `body`-க்குள் எழுதப்பட்டால், view ஒவ்வொரு முறையும் மறுகணக்கிடப்படும் போது அது மீண்டும் புதிதாக உருவாக்கப்படும்.
* `body`-க்கு வெளியே எழுதப்பட்டால், அது view structure-இன் property ஆக இருக்கும்; இதனால் code structure இன்னும் தெளிவாக இருக்கும்.

அதனால், நடைமுறை development-இல், variable-களையும் constant-களையும் பொதுவாக `body`-க்கு வெளியே எழுதுவார்கள்.

## தரவுவகைகள்

மாறிகள் பல்வேறு வகையான தரவுகளைச் சேமிக்க முடியும். அந்த வகைகள் data type-கள் என்று அழைக்கப்படுகின்றன.

உதாரணமாக, முன்பு நாம் `Text` மூலம் உரையை காட்டினோம்:

```swift id="adwhjs"
Text("Hello, World")
```

இங்கே `"Hello, World"` என்பது string (`String`) type ஆகும்.

String என்பது உரை உள்ளடக்கத்தைக் குறிக்கும். Swift-இல் அது கட்டாயமாக இரட்டை குறிக்கோள்கள் `""` உள்ளே எழுதப்பட வேண்டும்.

உதாரணமாக:

```swift id="i4b8w6"
var hello = "Hello, World"
```

String தவிர, Swift-இல் இன்னும் பல data type-கள் உள்ளன.

தொடக்கநிலையில் அதிகமாக பயன்படுத்தப்படும் நான்கு முக்கிய type-கள்:

* String
* Int
* Double
* Bool

இந்த நான்கு type-களே பெரும்பாலான அடிப்படை data handling-ஐ முடிக்கக்கூடியவை.

**String**

`String` என்பது உரை உள்ளடக்கத்தைக் குறிக்கும். உதாரணமாக:

```swift id="xnrpig"
var name = "FangJunyu"
var city = "Rizhao"
```

String பொதுவாக பயன்படுத்தப்படுவது: username, தலைப்பு, உரை உள்ளடக்கம் ஆகியவற்றைக் காட்ட.

**Int**

`Int` என்பது integer, அதாவது தசம புள்ளி இல்லாத முழு எண்.

உதாரணமாக:

```swift id="2knn7u"
var age = 26
var count = 100
```

முழு எண்கள் பொதுவாக பயன்படுத்தப்படுவது: வயது, எண்ணிக்கை, கணக்கீடு போன்றவற்றுக்கு.

**Double**

`Double` என்பது தசம எண்களைக் குறிக்கும்.

உதாரணமாக:

```swift id="kja3sa"
var weight = 74.5
var height = 185.0
```

Swift-இல், decimal number-கள் பொதுவாக இயல்பாகவே `Double` type ஆக கருதப்படும்.

இதற்கு ஒத்த இன்னொரு type `Float` என்றாலும், அது குறைந்த range-ஐ மட்டுமே தரும். அதனால், நடைமுறை development-இல் `Double` அதிகமாகப் பயன்படுத்தப்படுகிறது.

**Bool**

`Bool` type இரண்டு நிலைகளை மட்டும் குறிக்கும்: `true` மற்றும் `false`.

உதாரணமாக:

```swift id="3gx0d1"
var isShowAlert = true
var isLogin = false
```

`Bool` type பொதுவாக condition checking-க்கு பயன்படுத்தப்படுகிறது. உதாரணமாக, alert box காட்ட வேண்டுமா இல்லையா என்பதைக் குறிக்க.

இரண்டு முடிவுகளே இருக்கும் சூழலில், `Bool` மிகவும் பொருத்தமான type ஆகும்.

### SwiftUI-இல் காட்டுதல்

நாம் `String` type-ஐ காட்டும்போது, அதை நேரடியாக `Text`-இல் பயன்படுத்தலாம்:

```swift id="tpo38q"
Text(name)
```

ஆனால் `Int`, `Double` போன்றவை string அல்ல; அதனால், அவற்றை நேரடியாக text content ஆக எழுத முடியாது.

அவற்றை `Text`-இல் காட்ட வேண்டும் என்றால், string interpolation பயன்படுத்த வேண்டும்:

```swift id="e8itqb"
\()
```

String interpolation என்பது string-க்குள் எழுதப்பட வேண்டும்; மேலும் variable அல்லது constant `\()` உள்ளே எழுதப்பட வேண்டும்.

உதாரணமாக:

```swift id="svjwv6"
var num = 1

Text("\(num)")
Text("DoubleNum: \(num * 2)")
```

மேலுள்ள code-இல், `""` என்பது string; `\()` என்பது variable அல்லது constant-ஐ string-க்குள் சேர்க்கப் பயன்படுகிறது.

உதாரணமாக:

```swift id="7plqks"
"DoubleNum: \(num * 2)"
```

அதில் காட்டப்படும் உள்ளடக்கம்:

```id="19nqhe"
DoubleNum: 2
```

இந்த முறையில், நாம் string-க்குள் variable அல்லது constant-ஐச் சேர்த்து, பின்னர் `Text` மூலம் interface-இல் காட்டலாம்.

குறிப்பு: `\()` என்பது string `""`-க்குள் மட்டுமே பயன்படுத்தப்பட வேண்டும்.

## Operator-கள்

நாம் variable அல்லது constant அறிவிக்கும் போது, கீழே உள்ளபடி ஒரு எழுதும் முறையைப் பார்க்கலாம்:

```swift id="lvqawy"
var num = 1
```

இங்கே உள்ள `=` என்பது assignment operator என்று அழைக்கப்படுகிறது.

இதன் பணி: வலப்புறத்தில் உள்ள value-ஐ இடப்புற variable-க்கு ஒதுக்குவது.

இந்த உதாரணத்தில், `1` என்பது `num` என்ற variable-க்கு ஒதுக்கப்படுகிறது.

Assignment operator தவிர, இன்னும் சில பொதுவான arithmetic operator-களும் உள்ளன:

* `+`
* `-`
* `*`
* `/`

நாம் எண்களை வைத்து கணக்கிடும் போது, உதாரணமாக `Int` அல்லது `Double` type-களில், இந்த operator-கள் பயன்படுத்தப்படும்.

உதாரணமாக:

```swift id="8ikff6"
var age = 21 + 3 // 24
var value = 30 - 2 // 28
var num = 1 * 3 // 3
var weight = 88.5 / 2 // 44.25
```

கணக்கீடு முடிந்த பிறகு, அதன் விளைவு variable-இல் சேமிக்கப்படும்.

## சுருக்கம்

Variable, constant, operator ஆகியவை programming-இன் மிகவும் அடிப்படையான கருத்துகள்.

Variable மற்றும் constant மூலம், நாங்கள் program-இல் பல்வேறு தரவுகளைச் சேமிக்க முடியும்; data type மூலம், அந்தத் தரவு எந்த வகையைச் சேர்ந்தது என்பதைத் தெளிவாக அறிய முடியும்; operator மூலம், அந்தத் தரவை கணக்கிடவும் செயல்படுத்தவும் முடியும்.

இந்த அறிவுகள் programming உலகின் அடிப்படை கருவிகளைப் போன்றவை. இவற்றை நன்றாகப் புரிந்துகொண்டால், அடுத்தடுத்த Swift மற்றும் SwiftUI கற்றலுக்கான உறுதியான அடித்தளத்தை அமைக்க முடியும்.
