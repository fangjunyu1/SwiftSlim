# Въртележка с цитати

В този урок ще реализираме функция „въртележка с цитати“ и ще изучим по-задълбочено основни знания от Swift, като масиви (`Array`) и условни изрази (`if-else`).

Ще покажем как да съхраняваме множество цитати и как чрез взаимодействие с бутон да реализираме циклично превъртане на цитатите.

## Показване на цитат

Първо трябва да покажем един цитат в SwiftUI.

Най-простият начин е да използваме изгледа `Text`:

```swift
Text("Slow progress is still progress.")
```

Този код може да показва само един фиксиран цитат. Ако искаме да показваме повече цитати и да превключваме между тях, трябва да ги запазим.

Но обикновената променлива от тип низ може да пази само един цитат:

```swift
let sayings = "Slow progress is still progress."
```

Ако искаме да запазим няколко цитата, трябва да декларираме по една променлива за всеки:

```swift
let saying1 = "Slow progress is still progress."
let saying2 = "Difficult roads lead to beautiful destinations."
let saying3 = "Awake minds are the strongest."
let saying4 = "Long-term thinking wins over short-term tricks."
let saying5 = "Master one thing, and do it well."
```

Но в реалната разработка този подход не само е тромав, а и понеже всяка променлива е независима, не можем гъвкаво да реализираме въртележка.

За да управляваме удобно няколко цитата, трябва да използваме структура от данни, която ги съхранява заедно. Това е масивът (`Array`).

След като използваме масив, горният код може да бъде записан така:

```swift
let sayings = [
    "Slow progress is still progress.",
    "Difficult roads lead to beautiful destinations.",
    "Awake minds are the strongest.",
    "Long-term thinking wins over short-term tricks.",
    "Master one thing, and do it well."
    ]
```

**Подсказка: според добрите практики в програмирането, имената на масивите, които съхраняват множество елементи, обикновено са в множествено число, като `sayings`, за да отразяват, че представляват колекция.**

## Масиви

В Swift масивът е колекция от подредени елементи и се представя чрез квадратни скоби `[]`.

```swift
[]
```

Вътре в масива може да има няколко елемента от един и същи тип, като те се разделят със запетая `,`.

Например:

```swift
[101, 102, 103, 104, 105]
```

Можем да разбираме масива като композиция от влак:

![Array](../../RESOURCE/011_array1.png)

Цялата композиция представлява обекта масив, а всеки вагон е подреден последователно.

### Индекси и достъп до елементи

Понеже масивът е подреден, системата може да намира конкретните елементи според техния ред. Този механизъм се нарича индекс (`Index`).

В Swift, както и в повечето езици за програмиране, индексите на масива започват от `0`, а не от `1`. Това означава, че първият елемент има индекс `0`, вторият `1` и така нататък.

![Array](../../RESOURCE/011_array2.png)

Ако искаме да достъпим конкретен елемент от масива, трябва само да добавим квадратни скоби след името на масива и вътре да запишем индекса на желания елемент.

Например:

```swift
sayings[0]   // 101
sayings[1]   // 102
```

Ако опитаме да достъпим индекс извън валидния диапазон на масива, ще възникне проблем „излизане извън границите на индекса“ (`Index Out of Range`). Затова трябва да внимаваме индексът винаги да е в допустимите граници.

**Излизане извън границите**

Например, ако масивът има само 5 елемента, валидният диапазон на индексите е от `0` до `4`. Ако опитаме да достъпим `sayings[5]`, програмата няма да намери съответния „вагон“, което ще доведе до грешка `Index Out of Range` и срив на приложението.

![Array](../../RESOURCE/011_array3.png)

### Операции с масиви

Масивите не поддържат само статично дефиниране. Те позволяват също така добавяне, изтриване, промяна на елементи и вземане на дължината на масива.

Подсказка: ако искаме да променяме масива, трябва задължително да използваме `var`, а не `let`.

**1. Добавяне на елемент**

Можем да използваме метода `append`, за да добавим елемент в края на масива:

```swift
var num = [0, 1, 2]
num.append(3)   // [0, 1, 2, 3]
```

**2. Изтриване на елемент**

Чрез метода `remove(at:)` можем да изтрием конкретен елемент от масива:

```swift
var num = [0, 1, 2]
num.remove(at: 0)   // [1, 2]
```

**3. Промяна на елемент**

Можем директно да променим елемент чрез неговия индекс:

```swift
var num = [0, 1, 2]
num[1] = 4   // [0, 4, 2]
```

**4. Брой елементи**

Използваме свойството `count`, за да получим броя на елементите:

```swift
var num = [0, 1, 2]
num.count   // 3
```

### Показване на цитати чрез масив

За да показваме няколко цитата, можем да ги съхраним в масив и след това да достъпваме и показваме конкретен цитат чрез индекс.

Първо създаваме масив `sayings` в `ContentView`, за да съхраняваме цитатите, а след това в `Text` използваме индекс, за да прочетем и покажем съответния цитат:

```swift
struct ContentView: View {
    let sayings = [
        "Slow progress is still progress.",
        "Difficult roads lead to beautiful destinations.",
        "Awake minds are the strongest.",
        "Long-term thinking wins over short-term tricks.",
        "Master one thing, and do it well."
    ]
    
    var body: some View {
        VStack {
            Text(sayings[0])
        }
        .padding()
    }
}
```

Тук `sayings[0]` означава първия цитат в масива.

Ако искаме да покажем различен цитат, трябва само да променим индекса в квадратните скоби:

```swift
Text(sayings[0]) // "Slow progress is still progress."
Text(sayings[1]) // "Difficult roads lead to beautiful destinations."
```

### Дефиниране на индекс за цитатите

За да реализираме динамично превключване между цитатите, не можем да „заключим“ индекса директно вътре в `Text`.

Трябва да създадем отделна променлива, която да пази текущия показван индекс.

В SwiftUI можем да използваме `@State`, за да декларираме променлив индекс:

```swift
@State private var index = 0
```

SwiftUI ще следи променливата, обвита с `@State`, и когато `index` се промени, ще прерисува изгледа и ще покаже съответния цитат.

След това използваме `sayings[index]`, за да вземем динамично цитата от масива:

```swift
struct ContentView: View {
    @State private var index = 0
    let sayings = [
        "Slow progress is still progress.",
        "Difficult roads lead to beautiful destinations.",
        "Awake minds are the strongest.",
        "Long-term thinking wins over short-term tricks.",
        "Master one thing, and do it well."
    ]
    
    var body: some View {
        VStack {
            Text(sayings[index])
        }
        .padding()
    }
}
```

Когато стойността на `index` се промени, `Text` ще показва различен цитат.

### Управление на индекса чрез бутон

За да управляваме превключването на цитатите, можем да използваме `Button`, който да променя стойността на `index`. При всяко натискане бутонът ще увеличава `index` с `1`:

```swift
Button("Next") {
    index += 1
}
```

Когато натиснем бутона, стойността на `index` ще се промени от `0` на `1`, което ще доведе до обновяване на изгледа и `Text(sayings[index])` ще прочете следващия цитат.

Но тук има потенциален проблем: ако натискаме бутона непрекъснато, `index` може да нарасне до стойност извън диапазона на масива, което ще доведе до грешка за индекс извън граници. Например, когато `index` стане `5` (а валидният диапазон на индекси е от `0` до `4`), приложението ще се срине.

За да предотвратим излизане извън границите, трябва да въведем условен контрол и да се уверим, че `index` няма да надхвърля допустимия диапазон. Това може да стане чрез `if-else`.

## Условен контрол: `if-else`

Изразът `if-else` е едно от най-често използваните условни разклонения в Swift. Той се използва, за да проверява дали дадено условие е изпълнено и според това да изпълнява различен блок код.

Основна структура:

```swift
if condition {
    // Кодът, който се изпълнява, когато condition е true
} else {
    // Кодът, който се изпълнява, когато condition е false
}
```

В `if` израза условието `condition` е булева стойност (`Bool`) и може да бъде `true` или `false`. Когато условието е `true`, се изпълнява кодът в `if`, иначе се изпълнява кодът в `else`.

Например:

```swift
let age = 25
if age > 18 {
    print("Big Boy")
} else {
    print("Little Body")
}
```

В този пример стойността на `age` е `25`, а `if` проверява дали `age > 18`. Понеже условието е изпълнено, ще се изведе `"Big Boy"`.

Ако не ни е нужен кодът в `else`, можем просто да го пропуснем:

```swift
if condition {
    // Кодът, който се изпълнява, когато condition е true
}
```

### Използване на условия за контрол на индекса

За да предотвратим излизане извън границите на масива, можем да използваме `if`, за да гарантираме, че `index` няма да излезе извън допустимия диапазон:

```swift
Button("Next") {
    if index < sayings.count - 1 {
        index += 1
    }
}
```

Логиката е следната: `sayings.count` има стойност `5` (има общо 5 цитата), така че `sayings.count - 1` е `4`, което е и последният валиден индекс в масива.

Когато `index` е по-малък от `4`, натискането на бутона и увеличаването с `1` е безопасно. Щом `index` достигне `4`, условието вече няма да е изпълнено и бутонът няма да прави нищо.

Сега кодът вече реализира превключване между цитатите:

```swift
struct ContentView: View {
    @State private var index = 0
    let sayings = [
        "Slow progress is still progress.",
        "Difficult roads lead to beautiful destinations.",
        "Awake minds are the strongest.",
        "Long-term thinking wins over short-term tricks.",
        "Master one thing, and do it well."
    ]
    var body: some View {
        VStack {
            Text(sayings[index])
            Button("Next") {
                if index < sayings.count - 1 {
                    index += 1
                }
            }
        }
        .padding()
    }
}
```

### Циклични цитати

Ако искаме след последния цитат при натискане на бутона отново да се показва първият, така че да реализираме циклично показване, можем да направим това чрез частта `else`:

```swift
Button("Next") {
    if index < sayings.count - 1 {
        index += 1
    } else {
        index = 0
    }
}
```

Когато бутонът бъде натиснат и `index` е достигнал последния елемент в масива, `index` ще бъде върнат обратно на `0` и цитатите ще започнат да се въртят отначало.

## Подобряване на изгледа на цитатите

Сега логиката на въртележката вече е реализирана, но можем допълнително да подобрим интерфейса, така че да изглежда по-красиво.

Пълен код:

```swift
struct ContentView: View {
    @State private var index = 0
    let sayings = [
        "Slow progress is still progress.",
        "Difficult roads lead to beautiful destinations.",
        "Awake minds are the strongest.",
        "Long-term thinking wins over short-term tricks.",
        "Master one thing, and do it well."
    ]
    var body: some View {
        VStack(spacing: 10) {
            Text(sayings[index])
                .padding(20)
                .background(.white)
                .cornerRadius(10)
                .opacity(0.8)
            Button("Next") {
                if index < sayings.count - 1 {
                    index += 1
                } else {
                    index = 0
                }
            }
            .buttonStyle(.borderedProminent)
        }
        .padding()
        .background(
            Image("001")
        )
    }
}
```

В този пример към `Text` е добавен бял полупрозрачен фон и заоблени ъгли, `Button` използва стила `.borderedProminent`, а `VStack` има фоново изображение.

Допълнителна бележка: когато използваме модификатора `background()` за фон с изображение, неговото стандартно поведение е да запълни колкото се може повече от областта за подредба на текущия изглед. В много случаи той естествено може да се разшири и върху `Safe Area`.

С това вече реализирахме изглед на въртележка с цитати.

## Допълнителни знания - многократна условна проверка: `if-else if-else`

В реалната разработка често се налага да обработваме повече от две условия. Например в някаква игра, когато точките са `1`, се задейства събитие A, когато са `2`, се задейства събитие B, когато са `3`, се задейства събитие C и така нататък.

Когато имаме повече от два клона на условие, трябва да използваме `if-else if-else`.

Основен синтаксис:

```swift
if conditionA {
    // Кодът, който се изпълнява, когато conditionA е true
} else if conditionB {
    // Кодът, който се изпълнява, когато conditionB е true
} else if conditionC {
    // Кодът, който се изпълнява, когато conditionC е true
} else {
    // Кодът, който се изпълнява, когато никое условие не е изпълнено
}
```

В този случай програмата ще проверява условията последователно и ще изпълни първото, което е изпълнено. Ако нито едно условие не е изпълнено, ще изпълни кода в `else`.

Въртележката с цитати също може да бъде написана чрез `if-else if-else`:

```swift
Button("Next") {
    if index == 0 {
        index += 1
    } else if index == 1 {
        index += 1
    } else if index == 2 {
        index += 1
    } else if index == 3 {
        index += 1
    } else if index == 4 {
        index = 0
    } else {
        index = 0
    }
}
```

Когато `index` е `0`, `1`, `2` или `3`, при всяко натискане се изпълнява `index += 1`. Когато `index` е `4` (последният елемент), `index` се връща на `0`, с което се реализира цикълът.

Клонът `else` служи като защита, в случай че се появи невалидна стойност (например ако `index` е бил променен погрешно).

Важно е да отбележим, че тук използваното `==` означава „дали е равно на“. Ако `if` проверява дали `index` е равен на дадено число, резултатът ще бъде `true` и ще се изпълни съответният код. Ако не е равно, ще се премине към следващата проверка.

Този вид многократна проверка е полезен, когато при различни условия трябва да се изпълнява различен код.
