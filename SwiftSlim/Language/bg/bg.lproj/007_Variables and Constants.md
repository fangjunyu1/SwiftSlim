# Променливи и константи

Този урок е посветен основно на променливите и константите в Swift, както и на някои често използвани типове данни и базови оператори.

Това са едни от най-основните знания в програмирането и също така са важна основа за по-нататъшното изучаване на SwiftUI.

## Променливи и константи в ежедневието

Можем да разберем променливите и константите чрез неща от ежедневието.

Примери за променливи в живота:

- съдържанието на всяко телевизионно предаване е различно
- времето всеки ден е различно
- часовникът се променя всяка секунда

Общото между тези неща е, че те се променят.

Ако телевизорът винаги показваше една и съща картина, времето винаги беше слънчево и стрелките на часовника никога не се движеха, тогава тези неща щяха да бъдат константи.

Едното може да се променя, а другото не.

## Разбиране на променливите и константите

При разработката на приложения често се налага потребителят да въведе или запази определена информация.

Например:

- име на акаунт
- рожден ден
- данни за контакт
- адрес

Тази информация трябва да бъде съхранена и показана.

Например, ако потребител въведе име в приложение:

```text
FangJunyu
```

трябва да запазим това име, за да може да се показва в приложението.

Можеш да мислиш за процеса на запазване като за поставяне на нещо в чекмедже.

Когато запазваме име, все едно поставяме това име в чекмедже.

Нещата, които трябва да бъдат запазени, може да са много, което означава, че и чекмеджетата може да са много. За да знаем какво има във всяко чекмедже, трябва да дадем име на всяко от тях.

Например:

```text
name
```

В този пример `name` е името на чекмеджето, а `FangJunyu` е съхранената информация.

![Var](../../RESOURCE/007_var.png)

**В Swift, ако искаш да съхраняваш данни, трябва да ги декларираш чрез променлива (`var`) или константа (`let`).**

Тъй като името обикновено може да се променя, трябва да използваме променлива.

```swift
var name = "FangJunyu"
```

Това декларира променлива с име `name`. Нейният тип е `String`, а стойността ѝ е `"FangJunyu"`.

### Разликата между променливи и константи

Променливите се декларират с `var`:

```swift
var
```

Константите се декларират с `let`:

```swift
let
```

Например:

```swift
var name = "FangJunyu"
let id = 123456
```

Ако нещо е декларирано като променлива, неговата стойност може да се променя. Ако е декларирано като константа, след това вече не може да се променя.

Затова основната разлика между променлива и константа е дали стойността може да се променя.

### Промяна на константа

Ако кодът се опита да промени стойността на константа:

```swift
let name = "Sam"
name = "Bob"
```

Swift ще покаже, че присвояването не може да бъде изпълнено, защото това е константа.

```text
Cannot assign to property: 'name' is a 'let' constant
```

Този механизъм помага на разработчиците да избегнат случайна промяна на важни данни.

### Показване в SwiftUI

Отвори файла `ContentView.swift` и декларирай променлива вътре в изгледа:

```swift
struct ContentView: View {
    var name = "FangJunyu"

    var body: some View {
        Text(name)
    }
}
```

Тук е декларирана променлива с име `name` и нейната стойност се показва с `Text`.

Ако променим съдържанието на променливата:

```swift
struct ContentView: View {
    var name = "Sam"
    var body: some View {
        Text(name)
    }
}
```

текстът, показван от `Text`, също ще се промени.

По този начин можем да използваме променливи, за да управляваме какво се показва в интерфейса, вместо всеки път директно да редактираме текста в `Text`.

Ако друга част от приложението има нужда от тази информация, можем също да предадем променливата нататък, което е като да извадим съдържанието от „чекмеджето“ и да го дадем на друго място за употреба.

### Къде се пишат променливи и константи в SwiftUI

В SwiftUI променливите и константите обикновено се пишат извън `body`:

```swift
struct ContentView: View {
    var name = "FangJunyu"

    var body: some View {
        Text(name)
    }
}
```

Разбира се, можем да пишем променливи и константи и вътре в `body`:

```swift
struct ContentView: View {
    var body: some View {
        var name = "FangJunyu"
        Text(name)
    }
}
```

Но има една важна разлика между двата подхода:

- ако променливите и константите са написани вътре в `body`, те се създават отново всеки път, когато изгледът бъде преизчислен
- ако са написани извън `body`, те съществуват като свойства на структурата на изгледа и структурата на кода е по-ясна

Затова в реалната разработка променливите и константите обикновено се пишат извън `body`.

## Типове данни

Променливите могат да съхраняват много различни видове данни и тези видове се наричат типове данни.

Например, по-рано използвахме `Text`, за да покажем текст:

```swift
Text("Hello, World")
```

Тук `"Hello, World"` е от тип `String`.

`String` се използва за представяне на текст и в Swift задължително трябва да бъде ограден с двойни кавички.

Например:

```swift
var hello = "Hello, World"
```

Освен низовете, в Swift има много други типове данни.

За начинаещите четирите най-често срещани типа са:

- `String`
- `Int`
- `Double`
- `Bool`

Тези четири типа са достатъчни, за да покрият повечето базови обработки на данни.

**String**

`String` представя текстово съдържание, например:

```swift
var name = "FangJunyu"
var city = "Rizhao"
```

Низовете обикновено се използват за потребителски имена, заглавия и текстово съдържание.

**Int**

`Int` представя цели числа, тоест числа без десетични знаци.

Например:

```swift
var age = 26
var count = 100
```

Целите числа често се използват за възраст, количество и броене.

**Double**

`Double` представя числа с десетична точка.

Например:

```swift
var weight = 74.5
var height = 185.0
```

В Swift десетичните числа по подразбиране обикновено са от тип `Double`.

Има и друг подобен тип, наречен `Float`, но той може да представя по-малък диапазон от стойности, затова в реалната разработка `Double` се използва по-често.

**Bool**

Типът `Bool` се използва за представяне на две състояния: `true` и `false`.

Например:

```swift
var isShowAlert = true
var isLogin = false
```

`Bool` често се използва в условна логика, например дали да се показва alert.

За ситуации с само два възможни резултата, `Bool` е особено подходящ.

### Показване в SwiftUI

Когато показваме тип `String`, можем директно да използваме `Text`:

```swift
Text(name)
```

Но типове като `Int` и `Double` не са низове, затова не могат директно да бъдат използвани като текстово съдържание.

Ако искаме да покажем такива стойности в `Text`, можем да използваме интерполация на низове (`String Interpolation`):

```swift
\()
```

Интерполацията на низове трябва да бъде написана вътре в низ и използва `\()` за обграждане на променлива или константа.

Например:

```swift
var num = 1

Text("\(num)")
Text("DoubleNum: \(num * 2)")
```

В кода по-горе `""` означава низ, а `\()` се използва за вмъкване на променливи или константи в него.

Например:

```swift
"DoubleNum: \(num * 2)"
```

Показваното съдържание е:

```text
DoubleNum: 2
```

По този начин можем да вмъкнем променливи или константи в низ и след това да ги покажем чрез `Text`.

Подсказка: `\()` може да се използва само вътре в низ `""`.

## Оператори

Когато декларираме променливи или константи, често виждаме код като този:

```swift
var num = 1
```

Тук `=` се нарича оператор за присвояване.

Неговата задача е да присвои стойността отдясно на променливата отляво.

В този пример `1` се присвоява на променливата `num`.

Освен оператора за присвояване има и някои често използвани числови оператори:

- `+`
- `-`
- `*`
- `/`

Когато извършваме изчисления с числа, например данни от тип `Int` или `Double`, използваме тези оператори.

Например:

```swift
var age = 21 + 3 // 24
var value = 30 - 2 // 28
var num = 1 * 3 // 3
var weight = 88.5 / 2 // 44.25
```

След като изчислението бъде направено, резултатът се съхранява в променливата.

## Обобщение

Променливите, константите и операторите са едни от най-базовите понятия в програмирането.

Чрез променливите и константите можем да съхраняваме различни данни в програмата; чрез типовете данни можем ясно да определим какъв вид са тези данни; чрез операторите можем да извършваме изчисления и обработка върху тях.

Тези знания са като основните инструменти в света на програмирането. Ако ги овладеем, ще изградим здрава основа за по-нататъшното изучаване на Swift и SwiftUI.
