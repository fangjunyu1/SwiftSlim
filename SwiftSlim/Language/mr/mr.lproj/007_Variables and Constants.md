# चल आणि स्थिरांक

या धड्यात आपण Swift मधील चल (variables) आणि स्थिरांक (constants) शिकणार आहोत, तसेच काही सामान्य data types आणि मूलभूत operators यांची ओळख करून घेणार आहोत.

ही सामग्री प्रोग्रामिंगमधील सर्वात मूलभूत ज्ञान आहे आणि पुढे SwiftUI शिकण्यासाठीही ती अत्यंत महत्त्वाची पायाभूत गोष्ट आहे.

## दैनंदिन जीवनातील चल आणि स्थिरांक

आपण दैनंदिन जीवनातील गोष्टींमधून चल आणि स्थिरांक समजू शकतो.

जीवनातील चल:

- टीव्हीवर प्रत्येक कार्यक्रमातील सामग्री वेगळी असते
- रोजचे हवामान बदलते
- घड्याळ प्रत्येक सेकंदाला बदलते

या गोष्टींचे सामायिक वैशिष्ट्य असे की: त्या बदलतात.

जर टीव्हीवर नेहमी एकच चित्र दिसले असते, हवामान नेहमीच स्वच्छ असते, आणि घड्याळाच्या सुया कधीच हलल्या नसत्या, तर त्या गोष्टी स्थिरांक ठरल्या असत्या.

एक गोष्ट बदलू शकते, आणि दुसरी बदलत नाही.

## चल आणि स्थिरांक समजून घेणे

App development मध्ये वापरकर्त्याने काही माहिती भरावी किंवा जतन करावी लागते.

उदाहरणार्थ:

- account नाव
- जन्मदिनांक
- संपर्क माहिती
- पत्ता

ही माहिती जतन केली जाईल आणि नंतर दाखवली जाईल.

उदाहरणार्थ, वापरकर्ता App मध्ये एक नाव भरतो:

```
FangJunyu
```

हे नाव आपल्याला जतन करावे लागेल, जेणेकरून ते App मध्ये दाखवता येईल.

जतन करण्याची प्रक्रिया आपण सोप्या भाषेत अशी समजू शकतो: जणू एखाद्या drawer मध्ये वस्तू ठेवणे.

जेव्हा आपण नाव जतन करतो, तेव्हा जणू आपण ते नाव एका drawer मध्ये ठेवतो.

जतन करण्याच्या गोष्टी खूप असू शकतात, त्यामुळे drawers सुद्धा अनेक असतील. प्रत्येक drawer मध्ये काय आहे हे ओळखण्यासाठी, आपल्याला त्याला एक नाव द्यावे लागेल.

उदाहरणार्थ:

```
name
```

या उदाहरणात, `name` हे drawer चे नाव आहे, आणि `FangJunyu` ही जतन केलेली माहिती आहे.

![Var](../../RESOURCE/007_var.png)

**Swift मध्ये डेटा जतन करायचा असेल, तर `var` किंवा `let` वापरून declaration करणे आवश्यक आहे.**

नाव ही गोष्ट सामान्यतः बदलू शकते, म्हणून आपण चल वापरला पाहिजे.

```swift
var name = "FangJunyu"
```

इथे `name` नावाचा एक चल घोषित केला आहे. त्याचा प्रकार `String` आहे आणि त्याची किंमत `"FangJunyu"` आहे.

### चल आणि स्थिरांक यांतील फरक

चल `var` ने घोषित करतात:

```swift
var
```

स्थिरांक `let` ने घोषित करतात:

```swift
let
```

उदाहरणार्थ:

```swift
var name = "FangJunyu"
let id = 123456
```

जर declaration चल म्हणून केले, तर त्याची value बदलता येते; जर स्थिरांक म्हणून केले, तर नंतर ती बदलता येत नाही.

म्हणून, चल आणि स्थिरांक यांच्यातील मुख्य फरक असा आहे: value बदलण्याची परवानगी आहे का नाही.

### स्थिरांक बदलणे

जर कोड स्थिरांकाची value बदलण्याचा प्रयत्न करत असेल:

```swift
let name = "Sam"
name = "Bob"
```

Swift सूचित करेल की assignment होऊ शकत नाही, कारण तो स्थिरांक आहे.

```
Cannot assign to property: 'name' is a 'let' constant
```

अशी यंत्रणा developers ना काही महत्त्वाचा डेटा चुकून बदलला जाण्यापासून वाचवते.

### SwiftUI मध्ये दाखवणे

`ContentView.swift` फाइल उघडा आणि `View` च्या आत एक चल घोषित करा:

```swift
struct ContentView: View {
    var name = "FangJunyu"

    var body: some View {
        Text(name)
    }
}
```

इथे `name` नावाचा एक चल declare केला आहे आणि `Text` च्या मदतीने तो interface वर दाखवला आहे.

जर आपण चलातील सामग्री बदलली:

```swift
struct ContentView: View {
    var name = "Sam"
    var body: some View {
        Text(name)
    }
}
```

`Text` मध्ये दाखवला जाणारा मजकूरही बदलेल.

या पद्धतीने, आपण variables वापरून interface वर काय दिसेल ते नियंत्रित करू शकतो, आणि प्रत्येकवेळी थेट `Text` मधील मजकूर बदलण्याची गरज राहत नाही.

जर इतर ठिकाणीही ही माहिती लागली, तर आपण तो variable पुढे पास करू शकतो. म्हणजेच जणू drawer मधली सामग्री दुसऱ्या ठिकाणी वापरण्यास दिली.

### SwiftUI मध्ये चल आणि स्थिरांक कुठे लिहायचे

SwiftUI मध्ये variables आणि constants सहसा `body` च्या बाहेर लिहिले जातात:

```swift
struct ContentView: View {
    var name = "FangJunyu"

    var body: some View {
        Text(name)
    }
}
```

नक्कीच, variables आणि constants `body` च्या आतही लिहू शकतो:

```swift
struct ContentView: View {
    var body: some View {
        var name = "FangJunyu"
        Text(name)
    }
}
```

पण या दोन्हीत एक महत्त्वाचा फरक आहे:

- जर variables आणि constants `body` च्या आत लिहिले, तर view पुन्हा compute झाला की हा variable पुन्हा तयार होईल.
- जर `body` च्या बाहेर लिहिले, तर variable हा view struct चा property म्हणून राहतो आणि code structure अधिक स्पष्ट होते.

म्हणून, प्रत्यक्ष development मध्ये variables आणि constants साधारणपणे `body` च्या बाहेर लिहिले जातात.

## डेटा प्रकार

variables मध्ये अनेक वेगवेगळ्या प्रकारचा डेटा साठवता येतो. त्या प्रकारांना data types म्हणतात.

उदाहरणार्थ, आपण आधी `Text` वापरून मजकूर दाखवला:

```swift
Text("Hello, World")
```

येथे `"Hello, World"` हा string (`String`) प्रकार आहे.

String हा text content दर्शवण्यासाठी वापरला जातो, आणि Swift मध्ये तो double quotes `""` मध्ये लिहावा लागतो.

उदाहरणार्थ:

```swift
var hello = "Hello, World"
```

String व्यतिरिक्त, Swift मध्ये अनेक data types आहेत.

नवशिक्यांच्या टप्प्यावर सर्वात सामान्य चार प्रकार हे आहेत:

- String
- Int
- Double
- Bool

हे चार प्रकार बहुतेक मूलभूत डेटा प्रक्रिया पूर्ण करू शकतात.

**String**

`String` म्हणजे text content. उदाहरणार्थ:

```swift
var name = "FangJunyu"
var city = "Rizhao"
```

Strings सामान्यतः username, title, text content यांसाठी वापरले जातात.

**Int**

`Int` म्हणजे integer, म्हणजेच दशांशांशिवाय पूर्णांक.

उदाहरणार्थ:

```swift
var age = 26
var count = 100
```

पूर्णांकांचा वापर साधारणपणे वय, संख्या, मोजणी यांसाठी होतो.

**Double**

`Double` म्हणजे दशांश असलेली संख्या.

उदाहरणार्थ:

```swift
var weight = 74.5
var height = 185.0
```

Swift मध्ये decimal numbers सहसा डीफॉल्टने `Double` प्रकारात घेतले जातात.

यासारखाच `Float` हा प्रकारही आहे, पण त्याची numeric range कमी असल्यामुळे प्रत्यक्ष development मध्ये `Double` अधिक वापरला जातो.

**Bool**

`Bool` हा प्रकार दोन स्थिती दाखवतो: `true` आणि `false`.

उदाहरणार्थ:

```swift
var isShowAlert = true
var isLogin = false
```

`Bool` प्रकार condition checks मध्ये सामान्यतः वापरला जातो. उदाहरणार्थ, alert दाखवायचा का नाही.

ज्या परिस्थितीत फक्त दोनच निष्कर्ष असतात, तिथे `Bool` खूपच योग्य ठरतो.

### SwiftUI मध्ये दाखवणे

जेव्हा आपण `String` प्रकार दाखवतो, तेव्हा तो थेट `Text` मध्ये वापरता येतो:

```swift
Text(name)
```

पण `Int`, `Double` यांसारखे प्रकार strings नाहीत, त्यामुळे ते थेट मजकूराचा भाग म्हणून वापरता येत नाहीत.

जर `Text` मध्ये अशी values दाखवायची असतील, तर string interpolation वापरावे लागेल:

```swift
\()
```

String interpolation हे string च्या आत लिहिले जाते आणि `\\()` मध्ये variable किंवा constant ठेवला जातो.

उदाहरणार्थ:

```swift
var num = 1

Text("\(num)")
Text("DoubleNum: \(num * 2)")
```

वरील कोडमध्ये `""` म्हणजे string, आणि `\\()` वापरून variable किंवा constant string मध्ये insert केला जातो.

उदाहरणार्थ:

```swift
"DoubleNum: \(num * 2)"
```

दाखवला जाणारा परिणाम:

```swift
DoubleNum: 2
```

अशा प्रकारे आपण string मध्ये variables किंवा constants insert करून `Text` मध्ये ते दाखवू शकतो.

सूचना: `\\()` फक्त string `""` च्या आतच वापरता येते.

## operators

चल किंवा स्थिरांक declare करताना, आपल्याला पुढील syntax दिसते:

```swift
var num = 1
```

येथील `=` ला assignment operator म्हणतात.

त्याचे काम असे आहे: उजवीकडील value डावीकडील variable ला देणे.

या उदाहरणात, `1` ही value `num` या variable ला assign झाली आहे.

assignment operator व्यतिरिक्त, काही सामान्य numeric operators असे आहेत:

- `+`
- `-`
- `*`
- `/`

जेव्हा आपण numbers वर गणिती क्रिया करतो, उदाहरणार्थ `Int` किंवा `Double` प्रकाराच्या डेटावर, तेव्हा हे operators वापरले जातात.

उदाहरणार्थ:

```swift
var age = 21 + 3 // 24
var value = 30 - 2 // 28
var num = 1 * 3 // 3
var weight = 88.5 / 2 // 44.25
```

गणना पूर्ण झाल्यानंतर, निकाल variable मध्ये जतन होतो.

## सारांश

चल, स्थिरांक आणि operators हे प्रोग्रामिंगमधील सर्वात मूलभूत संकल्पना आहेत.

variables आणि constants च्या मदतीने आपण प्रोग्राममध्ये विविध डेटा जतन करू शकतो; data types मुळे डेटा कोणत्या प्रकारचा आहे हे स्पष्ट होते; operators च्या मदतीने आपण डेटा गणना आणि प्रक्रिया करू शकतो.

हे ज्ञान म्हणजे प्रोग्रामिंग जगातील मूलभूत साधने आहेत. ही सामग्री चांगली समजली, तर पुढे Swift आणि SwiftUI शिकण्यासाठी मजबूत पाया तयार होतो.
