# Μετρητής

Στα δύο προηγούμενα μαθήματα μάθαμε μεταβλητές, σταθερές, buttons και συναρτήσεις.

Σε αυτό το μάθημα θα συνδυάσουμε αυτές τις γνώσεις για να φτιάξουμε μια απλή εφαρμογή μετρητή.

Μέσα από αυτό το παράδειγμα θα μάθουμε πώς να αποθηκεύουμε και να αλλάζουμε μεταβλητές, καθώς και πώς να χρησιμοποιούμε το `@State` για τη διαχείρισή τους.

## Υλοποίηση μετρητή

Οι μετρητές χρησιμοποιούνται συνήθως για να καταγράφουν ποσότητες που αυξάνονται συνεχώς, όπως π.χ. πόσες φορές έκανε κάποιος σχοινάκι ή πόσους γύρους έτρεξε.

Ας υλοποιήσουμε λοιπόν έναν απλό μετρητή: θα εμφανίζεται ένας αριθμός και ένα button, και κάθε φορά που πατάμε το button ο αριθμός θα αυξάνεται.

![Num](../../RESOURCE/009_num.png)

### Εμφάνιση αριθμού

Αρχικά χρησιμοποιούμε `Text` για να εμφανίσουμε έναν αριθμό.

```swift
Text("0")
```

Αν όμως ο αριθμός πρέπει να αλλάζει, δεν μπορούμε να γράψουμε μόνιμα το `"0"`, αλλά πρέπει να αποθηκεύσουμε τον αριθμό σε μεταβλητή.

```swift
var num = 0
```

Και μετά να εμφανίσουμε τη μεταβλητή μέσα σε `Text`:

```swift
Text("\(num)")
```

Εδώ χρησιμοποιούμε string interpolation `\()` για να μετατρέψουμε τον αριθμό σε string και να τον εμφανίσουμε.

### Αύξηση αριθμού με button

Στη συνέχεια προσθέτουμε ένα button.

```swift
Button("+") {

}
.buttonStyle(.borderedProminent)
```

Όταν πατιέται το button, θέλουμε ο αριθμός να αυξάνεται.

Άρα μπορούμε να αλλάζουμε τη μεταβλητή μέσα στο button:

```swift
Button("+") {
    num = num + 1
}
.buttonStyle(.borderedProminent)
```

Αυτό σημαίνει ότι όταν πατηθεί το button, θα υπολογιστεί το `num + 1` και το αποτέλεσμα θα ανατεθεί πάλι στο `num`.

### Πιο σύντομη γραφή

Η Swift προσφέρει και πιο σύντομη σύνταξη:

```swift
num += 1
```

Αυτό είναι ισοδύναμο με:

```swift
num = num + 1
```

Αυτή η γραφή ονομάζεται σύνθετος τελεστής ανάθεσης.

Συνηθισμένοι σύνθετοι τελεστές ανάθεσης στη Swift είναι:

```text
+=   ανάθεση πρόσθεσης
-=   ανάθεση αφαίρεσης
*=   ανάθεση πολλαπλασιασμού
/=   ανάθεση διαίρεσης
%=   ανάθεση υπολοίπου
```

Για παράδειγμα:

```swift
num -= 2   // num = num - 2
num *= 3   // num = num * 3
```

### Πλήρης κώδικας

Τώρα μπορούμε να γράψουμε έναν απλό μετρητή:

```swift
struct ContentView: View {
    var num = 0
    var body: some View {
        VStack {
            Text("\(num)")
            Button("+") {
                num += 1
            }
            .buttonStyle(.borderedProminent)
        }
    }
}
```

Ωστόσο, σε αυτό το σημείο ο κώδικας δεν λειτουργεί σωστά.

### Το σφάλμα που εμφανίζεται

Το Xcode θα εμφανίσει:

```text
Cannot assign to property: 'self' is immutable
```

Δηλαδή δεν είναι δυνατό να τροποποιηθεί το property, επειδή η view θεωρείται immutable.

## Πώς λειτουργεί το SwiftUI

Στο SwiftUI: **η διεπαφή καθοδηγείται από τα δεδομένα**. Αυτό σημαίνει ότι όταν αλλάζουν τα δεδομένα, η διεπαφή ανανεώνεται αυτόματα.

Όμως υπάρχει κάτι σημαντικό: μόνο όταν αλλάζει κάποια κατάσταση (`state`) όπως `@State` ή `@Binding`, το SwiftUI παρακολουθεί την αλλαγή και ανανεώνει τη view.

Αν χρησιμοποιήσουμε μια απλή μεταβλητή, το SwiftUI διαβάζει την τιμή της μόνο μία φορά κατά τη δημιουργία της view. Ακόμη κι αν η μεταβλητή αλλάξει μετά, η διεπαφή δεν θα ενημερωθεί.

Για παράδειγμα:

```swift
var num = 0   // 0
num = 1   // 0
```

Στο παραπάνω παράδειγμα, το `num` είναι μια κανονική μεταβλητή. Παρόλο που η τιμή της αλλάζει, το SwiftUI δεν ενημερώνει τη view, και η διεπαφή εξακολουθεί να εμφανίζει το αρχικό `0`.

![Num](../../RESOURCE/009_state.png)

Επομένως, μόνο όταν τα δεδομένα χαρακτηρίζονται ως κατάσταση, για παράδειγμα με property wrappers όπως `@State` ή `@Binding`, το SwiftUI μπορεί να παρακολουθεί αυτόματα τις αλλαγές και να ενημερώνει τη view.

## Ο property wrapper @State

Στο SwiftUI, αν θέλουμε να αλλάζουμε μια μεταβλητή και να ανανεώνεται η view, πρέπει να χρησιμοποιήσουμε `@State`.

```swift
@State var num = 0
```

Το `@State` κάνει το SwiftUI να παρακολουθεί αυτή τη μεταβλητή.

Όταν η μεταβλητή αλλάζει, το SwiftUI υπολογίζει ξανά τη view και την ενημερώνει.

Πλήρης κώδικας:

```swift
struct ContentView: View {
    @State var num = 0
    var body: some View {
        VStack {
            Text("\(num)")
            Button("+") {
                num += 1
            }
            .buttonStyle(.borderedProminent)
        }
    }
}
```

Τώρα, όταν πατηθεί το button:

```swift
num += 1
```

το SwiftUI θα ανιχνεύσει ότι το `num` άλλαξε και θα ανανεώσει αυτόματα τη view.

![Num](../../RESOURCE/009_state1.png)

## Κανόνες χρήσης του @State

Στην πράξη, το `@State` συνήθως ακολουθεί δύο βασικούς κανόνες.

### 1. Το @State συνήθως χρησιμοποιείται μαζί με private

Οι μεταβλητές `@State` συνήθως χρησιμοποιούνται μόνο μέσα στον τρέχοντα τύπο, όπως το τρέχον struct της View, οπότε συνήθως γράφονται έτσι:

```swift
@State private var num = 0
```

Έτσι αποφεύγουμε άλλα views να προσπελάσουν ή να τροποποιήσουν απευθείας τη συγκεκριμένη μεταβλητή.

Δεν έχουμε μάθει ακόμη το `private`, αλλά προς το παρόν μπορείς να το σκεφτείς ως εξής:

**Ό,τι δηλώνεται με `private` μπορεί να χρησιμοποιηθεί μόνο μέσα στην τρέχουσα view και όχι από έξω.**

### 2. Το @State χρησιμοποιείται μόνο για την κατάσταση της τρέχουσας view

Το `@State` είναι κατάλληλο για δεδομένα που διαχειρίζεται η ίδια η τρέχουσα view.

Για παράδειγμα, για το περιεχόμενο ενός input field ή για την κατάσταση ενός switch:

```swift
@State private var isOn = false
@State private var text = ""
```

Αν τα δεδομένα χρειάζεται να μοιράζονται ανάμεσα σε πολλά views, τότε πρέπει να χρησιμοποιηθούν άλλοι τύποι state, όπως `@Binding` ή `@Observable`.

Αυτά θα τα μάθουμε σε επόμενα μαθήματα.

### Button μείωσης

Ο μετρητής δεν χρειάζεται μόνο να αυξάνεται, αλλά και να μειώνεται.

Μπορούμε λοιπόν να προσθέσουμε ένα button `-`.

```swift
Button("-") {
    num -= 1
}
.buttonStyle(.borderedProminent)
```

Στη συνέχεια τοποθετούμε τα buttons οριζόντια με `HStack`:

```swift
HStack {
    Button("+") {
        num += 1
    }
    .buttonStyle(.borderedProminent)
    Button("-") {
        num -= 1
    }
    .buttonStyle(.borderedProminent)
}
```

Αποτέλεσμα:

![Num](../../RESOURCE/009_num1.png)

Όταν πατάμε το `+`, ο αριθμός αυξάνεται κατά 1· όταν πατάμε το `-`, μειώνεται κατά 1.

### Button επαναφοράς

Μπορούμε επίσης να προσθέσουμε ένα button επαναφοράς που θα μηδενίζει τον αριθμό.

```swift
Button("0") {
    num = 0
}
.buttonStyle(.borderedProminent)
```

Όταν πατιέται το button επαναφοράς, στο `num` ανατίθεται η τιμή `0`.

Αποτέλεσμα:

![Num](../../RESOURCE/009_num2.png)

Όταν πατηθεί το button `0`, ο αριθμός γίνεται `0`.

### Προσαρμοσμένα buttons

Μέχρι τώρα, τα buttons μας μπορούν να εμφανίζουν μόνο κείμενο:

```swift
Button("+") {

}
```

Ένα button σε αυτή τη μορφή μπορεί να εμφανίζει μόνο κείμενο, όπως `+` ή `-`.

Στην πράξη όμως συνήθως θέλουμε η εμφάνιση του button να είναι πιο πλούσια, για παράδειγμα με εικονίδια ή άλλα views.

Το SwiftUI μάς επιτρέπει να προσαρμόσουμε το περιεχόμενο που εμφανίζει το button. Τότε μπορούμε να χρησιμοποιήσουμε μια άλλη μορφή γραφής:

```swift
Button(action: {

}, label: {

})
```

Σε αυτή τη σύνταξη, το `action` είναι ο κώδικας που εκτελείται όταν πατηθεί το button, ενώ το `label` είναι η view που εμφανίζεται μέσα στο button.

Για παράδειγμα, μπορούμε να κάνουμε το button να εμφανίζει ένα εικονίδιο SF Symbols:

```swift
Button(action: {
    num += 1
}, label: {
    Image(systemName: "plus.app")
})
```

Έτσι το button δεν εμφανίζει πια κείμενο αλλά ένα εικονίδιο.

### Βελτίωση της διεπαφής του μετρητή

Για να κάνουμε τη διεπαφή του μετρητή πιο καθαρή, μπορούμε επίσης να ρυθμίσουμε καλύτερα την εμφάνιση του αριθμού, για παράδειγμα να τον μεγεθύνουμε:

```swift
Text("\(num)")
    .font(.largeTitle)
    .fontWeight(.bold)
```

Στη συνέχεια μπορούμε να αλλάξουμε και το περιεχόμενο των buttons σε εικονίδια, μεγαλώνοντας λίγο και το μέγεθος των εικονιδίων:

```swift
Button(action: {
    num += 1
}, label: {
    Image(systemName: "plus.app")
        .imageScale(.large)
})
```

Με τον ίδιο τρόπο μπορούμε να μετατρέψουμε και το button μείωσης και το button επαναφοράς σε icon buttons:

```swift
Button(action: {
    num -= 1
}, label: {
    Image(systemName: "minus.square")
        .imageScale(.large)
})

Button(action: {
    num = 0
}, label: {
    Image(systemName: "arrow.trianglehead.2.clockwise.rotate.90")
        .imageScale(.large)
})
```

Τώρα τα buttons εμφανίζουν εικονίδια SF Symbols και η διεπαφή γίνεται πιο ξεκάθαρη και πιο άμεση οπτικά.

## Σύνοψη

Σε αυτό το μάθημα, μέσα από την υλοποίηση ενός απλού μετρητή, εξασκηθήκαμε μαζί στη χρήση μεταβλητών, `Text` και `Button`. Χρησιμοποιήσαμε μεταβλητή για να αποθηκεύουμε τον αριθμό και buttons για να ελέγχουμε την αύξηση και τη μείωσή του, ενώ ταυτόχρονα μάθαμε σύνθετους τελεστές ανάθεσης όπως `num += 1` και `num -= 1`.

Κατά τη διάρκεια της υλοποίησης κατανοήσαμε επίσης πώς λειτουργεί το SwiftUI: η διεπαφή καθοδηγείται από δεδομένα. Όταν αλλάζουν τα δεδομένα, τότε ανανεώνεται και η διεπαφή. Για αυτό χρειάζεται να χρησιμοποιούμε `@State` για να αποθηκεύουμε μεταβαλλόμενη κατάσταση. Όταν αλλάζει μια μεταβλητή `@State`, το SwiftUI ενημερώνει αυτόματα τη view.

Τέλος, βελτιώσαμε λίγο τη διεπαφή, προσαρμόσαμε το στιλ της γραμματοσειράς του `Text` και χρησιμοποιήσαμε `Button(action:label:)` για να ορίσουμε προσαρμοσμένο περιεχόμενο στα buttons, ώστε να μπορούν να εμφανίζουν εικονίδια. Έτσι η διεπαφή του μετρητή έγινε πιο καθαρή και πιο όμορφη.

Πλέον έχουμε έρθει σε επαφή με τον βασικό τρόπο χρήσης των SwiftUI views. Στο SwiftUI, μεγάλο μέρος της διεπαφής καθοδηγείται από μεταβλητές. Όταν οι μεταβλητές αλλάζουν, το SwiftUI ανανεώνει αυτόματα τη view. Γι’ αυτό, κατά την ανάπτυξη εφαρμογών SwiftUI, συνήθως χρειάζεται πρώτα να σχεδιάσουμε ποια δεδομένα θέλουμε να αποθηκεύσουμε και μετά, πάνω σε αυτά, να χτίσουμε τη διεπαφή και τη λογική αλληλεπίδρασης.

## Πλήρης κώδικας

```swift
struct ContentView: View {
    @State var num = 0
    var body: some View {
        VStack(spacing: 20) {
            Text("\(num)")
                .font(.largeTitle)
                .fontWeight(.bold)
            HStack {
                Button(action: {
                    num += 1
                }, label: {
                    Image(systemName: "plus.app")
                        .imageScale(.large)
                })
                
                Button(action: {
                    num -= 1
                }, label: {
                    Image(systemName: "minus.square")
                        .imageScale(.large)
                })
                
                Button(action: {
                    num = 0
                }, label: {
                    Image(systemName: "arrow.trianglehead.2.clockwise.rotate.90")
                        .imageScale(.large)
                })
            }
        }
    }
}
```
