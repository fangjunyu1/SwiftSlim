# Τοίχος φωτογραφιών

Σε αυτό το μάθημα, θα δημιουργήσουμε μια σελίδα τοίχου φωτογραφιών.

![view](../../../Resource/032_view13.png)

Σε αυτό το παράδειγμα, θα επαναλάβουμε τις γνώσεις για τα `extension`, `self` και τις παρουσίες, και θα μάθουμε τα συχνά χρησιμοποιούμενα γραφικά `Shape` στο SwiftUI.

Με αυτές τις γνώσεις, μπορούμε να περικόψουμε απλές εικόνες σε διαφορετικά στυλ, όπως κύκλο, στρογγυλεμένο ορθογώνιο, κάψουλα και έλλειψη, και να προσθέσουμε εφέ περιγράμματος στις φωτογραφίες.

Σε αυτό το μάθημα θα χρησιμοποιήσουμε μερικά νέα σημεία γνώσης: `Shape`, `clipShape`, `strokeBorder` και `overlay`.

Μεταξύ αυτών, το `clipShape` χρησιμοποιείται για την περικοπή του σχήματος μιας προβολής, το `strokeBorder` χρησιμοποιείται για τη σχεδίαση περιγράμματος σχήματος και το `overlay` χρησιμοποιείται για την επικάλυψη μιας νέας προβολής πάνω από την τρέχουσα προβολή.

## Εμφάνιση φωτογραφιών

Αρχικά, εμφανίζουμε 5 φωτογραφίες στο `ContentView`:

```swift
struct ContentView: View {
    var body: some View {
        ScrollView {
            VStack(spacing: 50) {
                Image("1")
                    .resizable()
                    .scaledToFit()
                    .frame(width: 300)

                Image("2")
                    .resizable()
                    .scaledToFit()
                    .frame(width: 300)

                Image("3")
                    .resizable()
                    .scaledToFit()
                    .frame(width: 300)

                Image("4")
                    .resizable()
                    .scaledToFit()
                    .frame(width: 300)

                Image("5")
                    .resizable()
                    .scaledToFit()
                    .frame(width: 300)
            }
            .padding(.vertical, 100)
        }
        .ignoresSafeArea()
    }
}
```

Αποτέλεσμα εμφάνισης:

![view](../../../Resource/032_view.png)

Σε αυτόν τον κώδικα, χρησιμοποιούμε το `ScrollView` για να υλοποιήσουμε την κυλιόμενη εμφάνιση της λίστας φωτογραφιών και χρησιμοποιούμε το `VStack` ώστε πολλές εικόνες να τοποθετούνται από πάνω προς τα κάτω.

Κάθε εικόνα ρυθμίζει το αποτέλεσμα εμφάνισής της μέσω των `.resizable()`, `.scaledToFit()` και `.frame(width: 300)`, ώστε η εικόνα να μπορεί να κλιμακώνεται, να εμφανίζεται πλήρως διατηρώντας την αναλογία της και να περιορίζεται το πλάτος της.

Το `.padding(.vertical, 100)` στο `VStack` χρησιμοποιείται για να αυξήσει το επάνω και κάτω διάστημα, αποφεύγοντας η πρώτη και η τελευταία φωτογραφία να είναι υπερβολικά κοντά στην άκρη της οθόνης.

Το τελικό `.ignoresSafeArea()` σημαίνει ότι η προβολή κύλισης αγνοεί την ασφαλή περιοχή, ώστε οι φωτογραφίες να μπορούν να επεκτείνονται προς το επάνω και το κάτω μέρος της οθόνης κατά την κύλιση και η σελίδα να εμφανίζεται πιο ολοκληρωμένα.

Ωστόσο, τώρα και οι 5 εικόνες χρησιμοποιούν τους ίδιους τροποποιητές:

```swift
.resizable()
.scaledToFit()
.frame(width: 300)
```

Όταν βλέπουμε επαναλαμβανόμενο κώδικα, μπορούμε να εξετάσουμε τη χρήση `extension` για οργάνωση.

## Χρήση extension για οργάνωση του στυλ εικόνας

Μπορούμε να επεκτείνουμε το `Image` με μια ειδική μέθοδο για τον τοίχο φωτογραφιών:

```swift
extension Image {
    func photoGalleryStyle() -> some View {
        self
            .resizable()
            .scaledToFit()
            .frame(width: 300)
    }
}
```

Εδώ, επεκτείνουμε τον τύπο `Image` με μια μέθοδο `photoGalleryStyle`.

Το εσωτερικό αυτής της μεθόδου περιλαμβάνει τα `.resizable()`, `.scaledToFit()` και `.frame(width: 300)` που χρησιμοποιήσαμε επανειλημμένα παραπάνω. Δηλαδή, συγκεντρώνει μαζί τον κώδικα για την κλιμάκωση εικόνας, την αναλογική εμφάνιση και τη ρύθμιση πλάτους.

Μετά τη χρήση αυτής της επέκτασης, ο αρχικός κώδικας εικόνας:

```swift
Image("1")
    .resizable()
    .scaledToFit()
    .frame(width: 300)
```

μπορεί να απλοποιηθεί σε:

```swift
Image("1")
    .photoGalleryStyle()
```

Έτσι, κάθε εικόνα χρειάζεται μόνο να καλέσει `.photoGalleryStyle()` για να εφαρμόσει το ίδιο στυλ τοίχου φωτογραφιών. Ο κώδικας γίνεται πιο συνοπτικός και είναι πιο βολικό να τροποποιηθεί ενιαία αργότερα.

## Κατανόηση του self

Στη μέθοδο επέκτασης, γράψαμε `self`:

```swift
extension Image {
    func photoGalleryStyle() -> some View {
        self
            .resizable()
            .scaledToFit()
            .frame(width: 300)
    }
}
```

Εδώ, το `self` δηλώνει την τρέχουσα παρουσία εικόνας που καλεί αυτή τη μέθοδο.

Για παράδειγμα:

```swift
Image("1")
    .photoGalleryStyle()
```

Σε αυτόν τον κώδικα, το `photoGalleryStyle()` καλείται από το `Image("1")`, επομένως το `self` μέσα στη μέθοδο δηλώνει αυτή την εικόνα `Image("1")`.

Μπορεί να γίνει απλά κατανοητό ως εξής: όποιος καλεί αυτή τη μέθοδο, αυτόν δηλώνει το `self`.

Πρέπει να σημειωθεί ότι οι τροποποιητές του SwiftUI επιστρέφουν συνεχώς νέα αποτελέσματα προβολής. Επομένως, το `photoGalleryStyle()` επιστρέφει μια τροποποιημένη προβολή και όχι απλώς το αρχικό ίδιο το `Image`.

## Σχήματα φωτογραφιών

Τώρα, οι φωτογραφίες μπορούν ήδη να εμφανίζονται κανονικά. Ωστόσο, αυτές οι φωτογραφίες είναι από προεπιλογή ορθογώνιες και φαίνονται σχετικά συνηθισμένες.

Αν θέλουμε απλώς να δώσουμε στις φωτογραφίες στρογγυλεμένες γωνίες, μπορούμε να χρησιμοποιήσουμε απευθείας το `.cornerRadius()`:

```swift
Image("1")
    .photoGalleryStyle()
    .cornerRadius(20)
```

Αποτέλεσμα εμφάνισης:

![color](../../../Resource/032_view1.png)

Το `.cornerRadius(20)` σημαίνει ότι προσθέτουμε στην εικόνα στρογγυλεμένη γωνία `20 pt`. Για μια συνηθισμένη εικόνα με στρογγυλεμένες γωνίες, αυτή η γραφή είναι ήδη αρκετή.

Ωστόσο, το `.cornerRadius()` μπορεί να χειριστεί μόνο το εφέ στρογγυλεμένων γωνιών και είναι κατάλληλο για τη δημιουργία απλών εικόνων με στρογγυλεμένες γωνίες. Αν θέλουμε να μετατρέψουμε τις φωτογραφίες σε περισσότερα διαφορετικά σχήματα, αυτός ο τροποποιητής από μόνος του δεν είναι αρκετός.

Τότε χρειάζεται να χρησιμοποιήσουμε το `Shape` στο SwiftUI. Το `Shape` μπορεί να αναπαριστά διαφορετικά γραφικά, και σε συνδυασμό με το `clipShape`, μπορούμε να περικόψουμε την εικόνα στο αντίστοιχο σχήμα.

## Γνωριμία με το Shape

Στο SwiftUI, το `Shape` δηλώνει ένα γραφικό. Όπως και το `View`, είναι ένας πολύ συχνά χρησιμοποιούμενος τύπος στο SwiftUI.

Συνηθισμένα `Shape` είναι ο κύκλος, το ορθογώνιο, το στρογγυλεμένο ορθογώνιο, η κάψουλα και η έλλειψη. Για να παρατηρήσουμε πιο άμεσα την εμφάνιση διαφορετικών γραφικών, τα παρακάτω παραδείγματα θα ορίσουν διαφορετικά χρώματα και μεγέθη για κάθε γραφικό.

Σε αυτά τα παραδείγματα, το `.fill()` χρησιμοποιείται για να γεμίσει το γραφικό με χρώμα και το `.frame()` χρησιμοποιείται για να ορίσει το μέγεθος εμφάνισης του γραφικού. Το χρώμα είναι μόνο για τη διάκριση διαφορετικών γραφικών και δεν είναι το σταθερό χρώμα του ίδιου του γραφικού.

### Circle κύκλος

Το `Circle` δηλώνει κύκλο. Χρησιμοποιείται συχνά σε avatars, κυκλικά κουμπιά, κυκλικές εικόνες και άλλα εφέ διεπαφής.

```swift
Circle()
    .fill(Color.red)
    .frame(width: 200, height: 200)
```

![circle](../../../Resource/032_view3.png)

### Rectangle ορθογώνιο

Το `Rectangle` δηλώνει ορθογώνιο. Είναι ένα από τα πιο βασικά γραφικά και μπορεί επίσης να χρησιμοποιηθεί για φόντα, διαχωριστικές περιοχές ή απλά περιγράμματα.

```swift
Rectangle()
    .fill(Color.green)
    .frame(width: 150, height: 100)
```

![rectangle](../../../Resource/032_view4.png)

### RoundedRectangle στρογγυλεμένο ορθογώνιο

Το `RoundedRectangle` δηλώνει στρογγυλεμένο ορθογώνιο. Το `cornerRadius` χρησιμοποιείται για να ορίσει το μέγεθος της στρογγύλευσης.

```swift
RoundedRectangle(cornerRadius: 20)
    .fill(Color.blue)
    .frame(width: 150, height: 100)
```

![rectangle](../../../Resource/032_view5.png)

### Capsule σχήμα κάψουλας

Το `Capsule` δηλώνει σχήμα κάψουλας. Τα δύο άκρα του είναι κυκλικά τόξα και χρησιμοποιείται συχνά σε κουμπιά κάψουλας, φόντα ετικετών και άλλα εφέ διεπαφής.

```swift
Capsule()
    .fill(Color.orange)
    .frame(width: 150, height: 100)
```

![rectangle](../../../Resource/032_view6.png)

### Ellipse έλλειψη

Το `Ellipse` δηλώνει έλλειψη. Είναι παρόμοιο με το `Circle`, αλλά όταν το πλάτος και το ύψος δεν είναι ίσα, εμφανίζεται ως έλλειψη.

```swift
Ellipse()
    .fill(Color.purple)
    .frame(width: 150, height: 100)
```

![rectangle](../../../Resource/032_view7.png)

Μέσα από αυτά τα παραδείγματα μπορούμε να δούμε ότι το ίδιο το `Shape` μπορεί να εμφανιστεί απευθείας ως γραφικό. Στη συνέχεια, θα χρησιμοποιήσουμε αυτά τα `Shape` μαζί με το `clipShape` για να περικόψουμε τις φωτογραφίες και να τις εμφανίσουμε σε διαφορετικά σχήματα.

## Χρήση clipShape για περικοπή φωτογραφιών

Τώρα, μπορούμε να χρησιμοποιήσουμε το `clipShape` για να περικόψουμε τις φωτογραφίες σε διαφορετικά σχήματα.

Για παράδειγμα, περικόπτουμε την πρώτη φωτογραφία σε κύκλο:

```swift
Image("1")
    .photoGalleryStyle()
    .clipShape(Circle())
```

Αυτός ο κώδικας σημαίνει: πρώτα εμφανίζουμε μια εικόνα και έπειτα χρησιμοποιούμε το `Circle()` για να την περικόψουμε σε κύκλο.

![view](../../../Resource/032_view2.png)

Η βασική γραφή του `clipShape` είναι:

```swift
.clipShape(σχήμα)
```

Προσθέτουμε το `.clipShape(...)` μετά την προβολή και μέσα στις παρενθέσεις γράφουμε το γραφικό στο οποίο θέλουμε να γίνει η περικοπή.

Για παράδειγμα:

```swift
.clipShape(Circle())
.clipShape(Rectangle())
.clipShape(RoundedRectangle(cornerRadius: 20))
.clipShape(Capsule())
.clipShape(Ellipse())
```

Τώρα, περικόπτουμε τις 5 φωτογραφίες σε διαφορετικά σχήματα:

```swift
struct ContentView: View {
    var body: some View {
        ScrollView {
            VStack(spacing: 50) {
                Image("1")
                    .photoGalleryStyle()
                    .clipShape(Circle())

                Image("2")
                    .photoGalleryStyle()
                    .clipShape(Rectangle())

                Image("3")
                    .photoGalleryStyle()
                    .clipShape(RoundedRectangle(cornerRadius: 20))

                Image("4")
                    .photoGalleryStyle()
                    .clipShape(Capsule())

                Image("5")
                    .photoGalleryStyle()
                    .clipShape(Ellipse())
            }
            .padding(.vertical, 100)
        }
        .ignoresSafeArea()
    }
}
```

Αποτέλεσμα εμφάνισης:

![view](../../../Resource/032_view8.png)

Μεταξύ αυτών, το αποτέλεσμα μετά την περικοπή με `Rectangle()` είναι κοντά σε μια συνηθισμένη ορθογώνια φωτογραφία, οπότε η οπτική αλλαγή δεν είναι εμφανής και χρησιμοποιείται κυρίως για σύγκριση με τα άλλα σχήματα.

Τώρα, οι φωτογραφίες δεν είναι πλέον απλά ορθογώνια, αλλά έχουν διαφορετικά σχήματα.

## Προσθήκη περιγράμματος φωτογραφίας

Αν θέλουμε να προσθέσουμε περίγραμμα σε μια κυκλική φωτογραφία, μπορεί να σκεφτούμε να χρησιμοποιήσουμε το `border`:

```swift
Image("1")
    .photoGalleryStyle()
    .clipShape(Circle())
    .border(Color.gray, width: 5)
```

Όμως, το αποτέλεσμα που παίρνουμε συνήθως δεν είναι κυκλικό περίγραμμα, αλλά ένα ορθογώνιο περίγραμμα.

![view](../../../Resource/032_view9.png)

Αυτό συμβαίνει επειδή το `border` προσθέτει περίγραμμα σύμφωνα με την ορθογώνια περιοχή της προβολής και όχι σύμφωνα με το σχήμα μετά την περικοπή με `clipShape`.

Επομένως, αν θέλουμε κυκλικό περίγραμμα, δεν μπορούμε να χρησιμοποιήσουμε απευθείας το `border`.

## Χρήση strokeBorder για σχεδίαση περιγράμματος σχήματος

Στο SwiftUI, το `strokeBorder` μπορεί να σχεδιάσει εσωτερικό περίγραμμα για αυτά τα συνηθισμένα `Shape`.

Για παράδειγμα, σχεδιάζουμε ένα κυκλικό περίγραμμα:

```swift
Circle()
    .strokeBorder(Color.brown, lineWidth: 10)
    .frame(width: 240, height: 240)
```

Αυτός ο κώδικας σημαίνει: σχεδιάζουμε ένα καφέ περίγραμμα για το `Circle`, με πλάτος περιγράμματος `10 pt`.

![view](../../../Resource/032_view12.png)

Εδώ χρειάζεται προσοχή: το `strokeBorder` δεν προσθέτει άμεσα περίγραμμα στη φωτογραφία, αλλά σχεδιάζει περίγραμμα για αυτό το γραφικό `Circle()`.

Δηλαδή, αυτός ο κώδικας παράγει μόνο ένα ανεξάρτητο κυκλικό περίγραμμα, το οποίο δεν έχει ακόμη σχέση με τη φωτογραφία.

Αν θέλουμε αυτό το κυκλικό περίγραμμα να εμφανίζεται πάνω από τη φωτογραφία, πρέπει να συνεχίσουμε να χρησιμοποιούμε το `overlay` για να επικαλύψουμε το περίγραμμα πάνω στη φωτογραφία.

## Χρήση overlay για επικάλυψη περιγράμματος

Το `overlay` είναι ένας τροποποιητής προβολής που μπορεί να επικαλύψει μια νέα προβολή πάνω από την τρέχουσα προβολή.

Η βασική δομή του μπορεί να γίνει κατανοητή ως:

```swift
τρέχουσα προβολή
    .overlay {
        επικαλυπτόμενη προβολή
    }
```

Σε αυτό το παράδειγμα, η τρέχουσα προβολή είναι η φωτογραφία που έχει ήδη περικοπεί σε κύκλο:

```swift
Image("1")
    .photoGalleryStyle()
    .clipShape(Circle())
```

![view](../../../Resource/032_view2.png)

Η νέα προβολή που θέλουμε να επικαλύψουμε είναι ένα κυκλικό περίγραμμα:

```swift
Circle()
    .strokeBorder(Color.brown, lineWidth: 10)
```

![view](../../../Resource/032_view12.png)

Άρα, μπορούμε να συνδυάσουμε τη φωτογραφία και το περίγραμμα:

```swift
Image("1")
    .photoGalleryStyle()
    .clipShape(Circle())
    .overlay {
        Circle()
            .strokeBorder(Color.brown, lineWidth: 10)
    }
```

Εδώ, το `overlay` σημαίνει ότι επικαλύπτουμε μια νέα προβολή πάνω από την τρέχουσα προβολή.

Μέσα στο `overlay`, το περίγραμμα θα επικαλυφθεί ακολουθώντας το εύρος εμφάνισης της τρέχουσας φωτογραφίας, επομένως δεν χρειάζεται να ορίσουμε ξεχωριστό `frame`. Αρκεί το σχήμα του περιγράμματος να είναι ίδιο με το σχήμα περικοπής, και το περίγραμμα θα εφαρμόσει στη φωτογραφία.

![view](../../../Resource/032_view10.png)

Το τελικό αποτέλεσμα είναι ότι πάνω από την κυκλική φωτογραφία καλύπτεται ένα κυκλικό περίγραμμα.

Σε σύγκριση με το `ZStack`, το `overlay` είναι πιο κατάλληλο για τέτοιες περιπτώσεις «προσθήκης διακόσμησης στην τρέχουσα προβολή». Η φωτογραφία είναι το κύριο σώμα και το περίγραμμα είναι μόνο ένα πρόσθετο εφέ, επομένως η χρήση του `overlay` είναι πιο καθαρή.

## Ολοκλήρωση του τοίχου φωτογραφιών

Τώρα, προσθέτουμε σε κάθε φωτογραφία το αντίστοιχο σχήμα και περίγραμμα:

```swift
import SwiftUI

struct ContentView: View {
    var body: some View {
        ScrollView {
            VStack(spacing: 50) {
                Image("1")
                    .photoGalleryStyle()
                    .clipShape(Circle())
                    .overlay {
                        Circle()
                            .strokeBorder(Color.brown, lineWidth: 20)
                    }

                Image("2")
                    .photoGalleryStyle()
                    .clipShape(Rectangle())
                    .overlay {
                        Rectangle()
                            .strokeBorder(Color.green, lineWidth: 20)
                    }

                Image("3")
                    .photoGalleryStyle()
                    .clipShape(RoundedRectangle(cornerRadius: 20))
                    .overlay {
                        RoundedRectangle(cornerRadius: 20)
                            .strokeBorder(Color.blue, lineWidth: 20)
                    }

                Image("4")
                    .photoGalleryStyle()
                    .clipShape(Capsule())
                    .overlay {
                        Capsule()
                            .strokeBorder(Color.orange, lineWidth: 20)
                    }

                Image("5")
                    .photoGalleryStyle()
                    .clipShape(Ellipse())
                    .overlay {
                        Ellipse()
                            .strokeBorder(Color.yellow, lineWidth: 20)
                    }
            }
            .padding(.vertical, 100)
        }
        .ignoresSafeArea()
    }
}
```

Αποτέλεσμα εμφάνισης:

![view](../../../Resource/032_view13.png)

Έτσι, μια βασική σελίδα τοίχου φωτογραφιών ολοκληρώθηκε.

Σε αυτή τη σελίδα, χρησιμοποιούμε το `ScrollView` για να υλοποιήσουμε κυλιόμενη εμφάνιση, το `Image` για να εμφανίσουμε φωτογραφίες, το `clipShape` για να περικόψουμε το σχήμα των φωτογραφιών και το `overlay` μαζί με το `strokeBorder` για να προσθέσουμε περιγράμματα.

## Πλήρης κώδικας

```swift
import SwiftUI

struct ContentView: View {
    var body: some View {
        ScrollView {
            VStack(spacing: 50) {
                Image("1")
                    .photoGalleryStyle()
                    .clipShape(Circle())
                    .overlay {
                        Circle()
                            .strokeBorder(Color.brown, lineWidth: 20)
                    }

                Image("2")
                    .photoGalleryStyle()
                    .clipShape(Rectangle())
                    .overlay {
                        Rectangle()
                            .strokeBorder(Color.green, lineWidth: 20)
                    }

                Image("3")
                    .photoGalleryStyle()
                    .clipShape(RoundedRectangle(cornerRadius: 20))
                    .overlay {
                        RoundedRectangle(cornerRadius: 20)
                            .strokeBorder(Color.blue, lineWidth: 20)
                    }

                Image("4")
                    .photoGalleryStyle()
                    .clipShape(Capsule())
                    .overlay {
                        Capsule()
                            .strokeBorder(Color.orange, lineWidth: 20)
                    }

                Image("5")
                    .photoGalleryStyle()
                    .clipShape(Ellipse())
                    .overlay {
                        Ellipse()
                            .strokeBorder(Color.yellow, lineWidth: 20)
                    }
            }
            .padding(.vertical, 100)
        }
        .ignoresSafeArea()
    }
}

extension Image {
    func photoGalleryStyle() -> some View {
        self
            .resizable()
            .scaledToFit()
            .frame(width: 300)
    }
}

    ContentView()
}
```

## Σύνοψη

Σε αυτό το μάθημα, ολοκληρώσαμε μια σελίδα τοίχου φωτογραφιών.

![view](../../../Resource/032_view13.png)

Σε αυτό το παράδειγμα, πρώτα χρησιμοποιήσαμε το `Image` για να εμφανίσουμε φωτογραφίες και στη συνέχεια επεκτείναμε το `Image` μέσω `extension` με μια μέθοδο `photoGalleryStyle`, για να οργανώσουμε τον επαναλαμβανόμενο κώδικα στυλ εικόνας.

Έπειτα, μάθαμε τα συνηθισμένα `Shape` στο SwiftUI, όπως `Circle`, `Rectangle`, `RoundedRectangle`, `Capsule` και `Ellipse`. Αυτά τα γραφικά μπορούν όχι μόνο να εμφανίζονται απευθείας, αλλά και να χρησιμοποιούνται μαζί με το `clipShape` για την περικοπή προβολών.

Για παράδειγμα:

```swift
.clipShape(Circle())
```

σημαίνει ότι περικόπτουμε την προβολή σε κύκλο.

Τέλος, χρησιμοποιήσαμε το `overlay` και το `strokeBorder` για να προσθέσουμε στις φωτογραφίες περιγράμματα με αντίστοιχα σχήματα. Πρέπει να σημειωθεί ότι το `border` συνήθως σχεδιάζει περίγραμμα σύμφωνα με την ορθογώνια περιοχή της προβολής. Αν θέλουμε το περίγραμμα να ακολουθεί κύκλο, κάψουλα ή έλλειψη, είναι πιο κατάλληλο να χρησιμοποιήσουμε το ίδιο `Shape` για επικάλυψη.

Μέσα από αυτό το μάθημα, όχι μόνο ολοκληρώσαμε το εφέ τοίχου φωτογραφιών, αλλά κατανοήσαμε επίσης τους συχνούς συνδυασμούς των `Shape`, `clipShape`, `strokeBorder` και `overlay` στη διεπαφή.
