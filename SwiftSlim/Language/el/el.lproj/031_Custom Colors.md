# Προσαρμοσμένα χρώματα

Σε αυτό το μάθημα, θα μάθουμε πώς να δημιουργούμε προσαρμοσμένα χρώματα στο SwiftUI.

Στα προηγούμενα μαθήματα, έχουμε ήδη χρησιμοποιήσει τα προεπιλεγμένα χρώματα που παρέχει το SwiftUI, για παράδειγμα:

```swift
Color.blue
Color.red
Color.green
```

Αυτά τα χρώματα είναι πολύ εύκολα στη χρήση, αλλά στην πραγματική ανάπτυξη εφαρμογών, τα προεπιλεγμένα χρώματα συνήθως δεν είναι αρκετά ακριβή.

Για παράδειγμα, σε ένα σχέδιο μπορεί να χρησιμοποιούνται τέτοια χρώματα:

```text
#2c54c2
#4875ed
#213e8d
```

Αυτό το είδος χρώματος ονομάζεται χρώμα Hex.

Σε αυτό το μάθημα, πρώτα θα κάνουμε το SwiftUI να υποστηρίζει χρώματα Hex και έπειτα θα χρησιμοποιήσουμε το `static` για να οργανώσουμε τα συχνά χρησιμοποιούμενα χρώματα.

Τέλος, θα εφαρμόσουμε τα προσαρμοσμένα χρώματα στην προβολή της εγκυκλοπαίδειας ζώων και θα χρησιμοποιήσουμε περαιτέρω φόντο με διαβάθμιση, ώστε τα κουμπιά να φαίνονται πιο πολυεπίπεδα.

## Γιατί χρειαζόμαστε προσαρμοσμένα χρώματα;

Στο SwiftUI, μπορούμε να χρησιμοποιήσουμε απευθείας τα χρώματα συστήματος.

Για παράδειγμα:

```swift
Text("Fang Junyu")
    .foregroundStyle(.blue)
```

Εδώ, το `.blue` σημαίνει μπλε και στην πραγματικότητα είναι μια συντομευμένη γραφή του `Color.blue`.

Το πλεονέκτημα των προεπιλεγμένων χρωμάτων είναι ότι είναι απλά και βολικά, αλλά οι επιλογές χρωμάτων είναι σχετικά περιορισμένες.

Για παράδειγμα:

```swift
Color.blue
```

Μπορεί να εκφράσει μόνο το προεπιλεγμένο μπλε που παρέχει το SwiftUI.

![Color.blue](../../../Resource/031_color6.png)

Όμως στην πραγματική ανάπτυξη, συχνά χρειαζόμαστε πιο συγκεκριμένα χρώματα.

Για παράδειγμα, ακόμα και αν όλα είναι μπλε, μπορεί να υπάρχουν διαφορετικά αποτελέσματα όπως ανοιχτό μπλε, σκούρο μπλε, γκριζωπό μπλε και φωτεινό μπλε.

![More Blue](../../../Resource/031_color5.png)

Σε αυτή την περίπτωση, αν χρησιμοποιούμε μόνο το `Color.blue`, είναι δύσκολο να αναπαραγάγουμε το αποτέλεσμα του σχεδίου.

Γι’ αυτό πρέπει να κάνουμε το SwiftUI να υποστηρίζει προσαρμοσμένα χρώματα.

## Τι είναι ένα χρώμα Hex;

Τα χρώματα στην οθόνη συνήθως αποτελούνται από τρία κανάλια: κόκκινο, πράσινο και μπλε, δηλαδή RGB.

Το RGB σημαίνει αντίστοιχα:

```text
Red     // Κόκκινο
Green   // Πράσινο
Blue    // Μπλε
```

Το χρώμα Hex είναι ένας τρόπος αναπαράστασης ενός χρώματος RGB.

Για παράδειγμα:

```swift
#5479FF
```

Αυτή η τιμή χρώματος μπορεί να γίνει απλά κατανοητή ως τρία μέρη:

```text
54  // Δηλώνει το κόκκινο κανάλι
79  // Δηλώνει το πράσινο κανάλι
FF  // Δηλώνει το μπλε κανάλι
```

Σε αυτό το μάθημα, δεν χρειάζεται να υπολογίσουμε αυτές τις τιμές ούτε να κατανοήσουμε σε βάθος τους κανόνες του δεκαεξαδικού συστήματος.

Προς το παρόν, χρειάζεται μόνο να γνωρίζουμε ότι το `#5479FF` δηλώνει ένα συγκεκριμένο χρώμα.

Στο μέλλον, όταν βλέπουμε γραφές όπως `#2c54c2` ή `#4875ed`, μπορούμε πρώτα να τις κατανοούμε ως τιμές χρώματος.

Σε εργαλεία σχεδίασης όπως Sketch, Figma και Photoshop, βλέπουμε επίσης συχνά παρόμοιες τιμές χρώματος.

![color](../../../Resource/031_color.png)

Ωστόσο, το SwiftUI από προεπιλογή δεν μπορεί να γραφτεί απευθείας έτσι:

```swift
Color(hex: "#5479FF")
```

Γι’ αυτό πρέπει να επεκτείνουμε μόνοι μας τον τύπο `Color`, ώστε να υποστηρίζει τη δημιουργία χρωμάτων μέσω συμβολοσειρών Hex.

## Δημιουργία αρχείου Color+Hex.swift

Αρχικά, δημιουργούμε ένα νέο αρχείο Swift.

Το όνομα του αρχείου μπορεί να γραφτεί ως:

```text
Color.swift
```

Μπορεί επίσης να γραφτεί πιο καθαρά ως:

```text
Color+Hex.swift
```

Εδώ προτείνεται περισσότερο να χρησιμοποιήσουμε:

```text
Color+Hex.swift
```

Σε ένα έργο Swift, ονόματα αρχείων όπως `Color+Hex.swift` είναι πολύ συνηθισμένα.

Δηλώνει ότι αυτό το αρχείο είναι ένα αρχείο επέκτασης που προσθέτει δυνατότητα Hex στο `Color`.

Το όνομα του αρχείου δεν επηρεάζει άμεσα την εκτέλεση του κώδικα· απλώς μας βοηθά να κατανοούμε πιο εύκολα τον σκοπό του αρχείου.

## Προσθήκη κώδικα Color(hex:)

Στο αρχείο `Color+Hex.swift`, γράφουμε τον παρακάτω κώδικα:

```swift
import SwiftUI

extension Color {
    init(hex: String) {
        let scanner = Scanner(string: hex)
        
        if hex.hasPrefix("#") {
            scanner.currentIndex = hex.index(after: hex.startIndex)
        }
        
        var rgbValue: UInt64 = 0
        scanner.scanHexInt64(&rgbValue)
        
        let red = Double((rgbValue & 0xFF0000) >> 16) / 255.0
        let green = Double((rgbValue & 0x00FF00) >> 8) / 255.0
        let blue = Double(rgbValue & 0x0000FF) / 255.0
        
        self.init(red: red, green: green, blue: blue)
    }
}
```

Αυτός ο κώδικας χρησιμοποιεί το `extension` για να επεκτείνει τον τύπο `Color` και προσθέτει στο `Color` μια νέα μέθοδο αρχικοποίησης:

```swift
init(hex: String)
```

Αφού έχουμε αυτή τη μέθοδο αρχικοποίησης, μπορούμε να περάσουμε μια συμβολοσειρά Hex στο `Color`, δημιουργώντας έτσι ένα προσαρμοσμένο χρώμα:

```swift
Color(hex: "#5479FF")
```

Στο εσωτερικό του, αυτός ο κώδικας επέκτασης μετατρέπει τη συμβολοσειρά Hex σε χρώμα RGB που μπορεί να αναγνωρίσει το SwiftUI.

Σε αυτό το στάδιο, δεν χρειάζεται να κατανοήσουμε σε βάθος τη λογική μετατροπής κάθε γραμμής· αρκεί να γνωρίζουμε ότι η μέθοδος `Color(hex:)` που προσθέσαμε μάς επιτρέπει να δημιουργούμε προσαρμοσμένα χρώματα μέσω τιμών Hex.

## Χρήση προσαρμοσμένων χρωμάτων

Τώρα μπορούμε να δοκιμάσουμε τα προσαρμοσμένα χρώματα στο `ContentView`.

```swift
import SwiftUI

struct ContentView: View {
    var body: some View {
        VStack(spacing: 20) {
            Text("Color.blue")
                .font(.largeTitle)
                .fontWeight(.bold)
                .foregroundStyle(Color.blue)
            
            Text("#2c54c2")
                .font(.largeTitle)
                .fontWeight(.bold)
                .foregroundStyle(Color(hex: "#2c54c2"))
            
            Text("#4875ed")
                .font(.largeTitle)
                .fontWeight(.bold)
                .foregroundStyle(Color(hex: "#4875ed"))
            
            Text("#213e8d")
                .font(.largeTitle)
                .fontWeight(.bold)
                .foregroundStyle(Color(hex: "#213e8d"))
        }
        .padding()
    }
}
```

Εμφανιζόμενο αποτέλεσμα:

![color](../../../Resource/031_color1.png)

Σε αυτό το παράδειγμα, η πρώτη γραμμή χρησιμοποιεί χρώμα συστήματος:

```swift
.foregroundStyle(Color.blue)
```

Οι επόμενες τρεις γραμμές χρησιμοποιούν προσαρμοσμένα χρώματα Hex:

```swift
.foregroundStyle(Color(hex: "#2c54c2"))
.foregroundStyle(Color(hex: "#4875ed"))
.foregroundStyle(Color(hex: "#213e8d"))
```

Μέσα από τη σύγκριση, μπορούμε να διαπιστώσουμε ότι το μπλε του συστήματος έχει μόνο ένα προεπιλεγμένο αποτέλεσμα.

Αντίθετα, τα χρώματα Hex μπορούν να εκφράσουν πιο λεπτές παραλλαγές του μπλε.

Αυτή είναι η αξία των προσαρμοσμένων χρωμάτων: μπορούν να κάνουν τα χρώματα της διεπαφής πιο κοντά στο πραγματικό σχέδιο και να μας βοηθήσουν να ελέγχουμε πιο εύκολα το οπτικό ύφος της εφαρμογής.

## Χρήση του static για οργάνωση χρωμάτων

Τώρα μπορούμε ήδη να δημιουργούμε χρώματα μέσω συμβολοσειρών Hex:

```swift
Color(hex: "#2c54c2")
```

Αυτός ο τρόπος γραφής μπορεί να χρησιμοποιηθεί κανονικά, αλλά αν το ίδιο χρώμα εμφανίζεται επανειλημμένα σε πολλά σημεία, η μεταγενέστερη συντήρηση δεν θα είναι τόσο βολική.

Αν αυτή η τιμή χρώματος έχει γραφτεί σε 10 διαφορετικά σημεία, όταν θελήσουμε αργότερα να αλλάξουμε αυτό το μπλε, θα πρέπει να τα τροποποιήσουμε ένα προς ένα.

Σε αυτή την περίπτωση, μπορούμε να χρησιμοποιήσουμε το `static` για να οργανώσουμε ενιαία τα συχνά χρησιμοποιούμενα χρώματα.

Στο τέλος του αρχείου `Color+Hex.swift`, συνεχίζουμε προσθέτοντας τον παρακάτω κώδικα:

```swift
extension Color {
    static let animalBlue = Color(hex: "#2c54c2")
    static let animalLightBlue = Color(hex: "#4875ed")
    static let animalDarkBlue = Color(hex: "#213e8d")
}
```

Εδώ προσθέσαμε στο `Color` τρεις στατικές ιδιότητες: `animalBlue`, `animalLightBlue` και `animalDarkBlue`, οι οποίες αντιπροσωπεύουν διαφορετικές αποχρώσεις του μπλε.

Επειδή αυτές οι ιδιότητες χρησιμοποιούν το `static`, ανήκουν στον ίδιο τον τύπο `Color`.

Κατά τη χρήση, μπορούμε να τις προσπελάσουμε απευθείας μέσω του `Color.`:

```swift
Color.animalBlue
```

Αυτός ο τρόπος γραφής είναι πιο καθαρός από το να γράφουμε απευθείας συμβολοσειρές Hex.

Όταν βλέπουμε το `Color.animalBlue`, μπορούμε να γνωρίζουμε ότι δηλώνει το μπλε που χρησιμοποιείται στην εγκυκλοπαίδεια ζώων.

Όταν όμως βλέπουμε το `Color(hex: "#2c54c2")`, μπορούμε μόνο να γνωρίζουμε ότι είναι μια τιμή χρώματος, αλλά δεν είναι εύκολο να καταλάβουμε τη συγκεκριμένη χρήση της.

Η ενιαία διαχείριση χρωμάτων έχει ακόμα ένα πλεονέκτημα: κάνει τις μεταγενέστερες αλλαγές πιο βολικές.

Αν θέλουμε να προσαρμόσουμε το κύριο χρώμα της εγκυκλοπαίδειας ζώων, χρειάζεται μόνο να αλλάξουμε το σημείο ορισμού:

```swift
static let animalBlue = Color(hex: "#1f5cff")
```

Όλα τα σημεία που χρησιμοποιούν αυτό το χρώμα θα ενημερωθούν μαζί.

Αυτό είναι το νόημα της χρήσης του `static` για την οργάνωση χρωμάτων: κάνει τα ονόματα των χρωμάτων πιο καθαρά και τη μεταγενέστερη συντήρηση πιο εύκολη.

## Εφαρμογή στην εγκυκλοπαίδεια ζώων

Τώρα μπορούμε να εφαρμόσουμε τα προσαρμοσμένα χρώματα στην προηγούμενη προβολή της εγκυκλοπαίδειας ζώων.

Προηγουμένως, το φόντο του κουμπιού ζώου ήταν λευκό:

```swift
.background(Color.white)
```

![single white button](../../../Resource/031_color7.png)

Τώρα μπορούμε να το αλλάξουμε σε προσαρμοσμένο χρώμα:

```swift
.background(Color.animalBlue)
```

![single blue button](../../../Resource/031_color8.png)

Εδώ, το `Color.animalBlue` είναι το στατικό χρώμα που μόλις ορίσαμε στο `Color+Hex.swift`.

Το κείμενο του κουμπιού χρησιμοποιεί λευκό:

```swift
.foregroundStyle(Color.white)
```

Το φόντο του emoji ζώου χρησιμοποιεί ημιδιαφανές λευκό:

```swift
.background(Color.white.opacity(0.15))
```

Έτσι, το κουμπί θα σχηματίσει ένα ενιαίο μπλε οπτικό ύφος.

Το κύριο σημείο αυτού του βήματος δεν είναι να προσθέσουμε περίπλοκο κώδικα, αλλά να εφαρμόσουμε πραγματικά στη διεπαφή τα προσαρμοσμένα χρώματα που μάθαμε προηγουμένως.

## Χρήση φόντου με διαβάθμιση

Εκτός από τη χρήση ενός μόνο χρώματος, μπορούμε επίσης να συνδυάσουμε πολλά χρώματα για να δημιουργήσουμε εφέ διαβάθμισης.

Για παράδειγμα, προηγουμένως ορίσαμε αυτά τα χρώματα:

```swift
extension Color {
    static let animalBlue = Color(hex: "#2c54c2")
    static let animalLightBlue = Color(hex: "#4875ed")
    static let animalDarkBlue = Color(hex: "#213e8d")
}
```

Αυτά τα χρώματα μπορούν όχι μόνο να χρησιμοποιηθούν μεμονωμένα, αλλά και να συνδυαστούν σε φόντο με διαβάθμιση.

Στο SwiftUI, μπορούμε να χρησιμοποιήσουμε το `LinearGradient` για να δημιουργήσουμε γραμμική διαβάθμιση.

Για παράδειγμα:

```swift
LinearGradient(
    colors: [Color.animalBlue, Color.animalLightBlue],
    startPoint: .leading,
    endPoint: .trailing
)
```

Αυτός ο κώδικας θα δημιουργήσει ένα φόντο με διαβάθμιση από αριστερά προς τα δεξιά, όπου το χρώμα μεταβαίνει σταδιακά από `Color.animalBlue` σε `Color.animalLightBlue`.

Σε αυτό, το `colors` χρησιμοποιείται για να ορίσει τα χρώματα που συμμετέχουν στη διαβάθμιση, ενώ τα `startPoint` και `endPoint` χρησιμοποιούνται για να ελέγχουν την κατεύθυνση της διαβάθμισης.

### Δοκιμή του εφέ διαβάθμισης

Μπορούμε να χρησιμοποιήσουμε ένα απλό Text για να δοκιμάσουμε το φόντο με διαβάθμιση.

```swift
Text("Linear Gradient")
    .font(.title)
    .fontWeight(.bold)
    .padding()
    .foregroundStyle(Color.white)
    .background {
        LinearGradient(
            colors: [Color.animalBlue, Color.animalLightBlue],
            startPoint: .leading,
            endPoint: .trailing
        )
    }
    .cornerRadius(12)
```

Εμφανιζόμενο αποτέλεσμα:

![color](../../../Resource/031_color3.png)

Σε αυτό το παράδειγμα, το φόντο του `Text` δεν είναι πλέον ένα μόνο χρώμα, αλλά ένα χρώμα διαβάθμισης που αλλάζει σταδιακά από αριστερά προς τα δεξιά.

Σε σύγκριση με ένα συνηθισμένο χρώμα φόντου, το φόντο με διαβάθμιση έχει περισσότερα επίπεδα και μπορεί πιο εύκολα να δημιουργήσει οπτική έμφαση στη διεπαφή.

## Χρήση φόντου με διαβάθμιση στην εγκυκλοπαίδεια ζώων

Τώρα μπορούμε να πάρουμε το χρώμα φόντου του κουμπιού ζώου:

```swift
.background(Color.animalBlue)
```

Και να το αλλάξουμε σε φόντο με διαβάθμιση:

```swift
.background {
    LinearGradient(
        colors: [Color.animalBlue, Color.animalLightBlue],
        startPoint: .leading,
        endPoint: .trailing
    )
}
```

Ο πλήρης κώδικας είναι ο εξής:

```swift
import SwiftUI

struct ContentView: View {
    @State private var selectedAnimal: Animal? = nil
    
    let animals = Animal.animals
    
    var body: some View {
        VStack(alignment: .leading, spacing: 20) {
            ForEach(animals) { animal in
                Button(action: {
                    selectedAnimal = animal
                }, label: {
                    HStack(spacing: 10) {
                        Text(animal.avatarEmoji)
                            .font(.title2)
                            .frame(width: 40, height: 40)
                            .background(Color.white.opacity(0.15))
                            .cornerRadius(20)
                        
                        Text(animal.name)
                            .fontWeight(.bold)
                            .foregroundStyle(Color.white)
                        
                        Spacer()
                    }
                    .padding(15)
                    .frame(width: 200, height: 60)
                    .background {
                        LinearGradient(
                            colors: [Color.animalBlue, Color.animalLightBlue],
                            startPoint: .leading,
                            endPoint: .trailing
                        )
                    }
                    .cornerRadius(12)
                    .shadow(radius: 5)
                })
                .buttonStyle(.plain)
            }
        }
        .padding()
        .sheet(item: $selectedAnimal) { animal in
            AnimalDetailView(animal: animal)
        }
    }
}
```

Εμφανιζόμενο αποτέλεσμα:

![color](../../../Resource/031_color4.png)

Τώρα, το κουμπί ζώου δεν έχει πλέον μόνο ένα ενιαίο χρώμα φόντου, αλλά διαθέτει εφέ διαβάθμισης από αριστερά προς τα δεξιά.

Σε σύγκριση με το μονόχρωμο φόντο, το φόντο με διαβάθμιση μπορεί να κάνει τη διεπαφή να φαίνεται πιο πολυεπίπεδη και πιο κοντά στον οπτικό σχεδιασμό μιας πραγματικής εφαρμογής.

## Η διαφορά ανάμεσα στο μονόχρωμο φόντο και το φόντο με διαβάθμιση

Το μονόχρωμο φόντο χρησιμοποιεί μόνο ένα χρώμα.

Για παράδειγμα:

```swift
.background(Color.animalBlue)
```

Αυτός ο τρόπος γραφής είναι απλός και καθαρός και είναι κατάλληλος για τις περισσότερες βασικές διεπαφές.

Το φόντο με διαβάθμιση χρησιμοποιεί πολλά χρώματα.

Για παράδειγμα:

```swift
.background {
    LinearGradient(
        colors: [Color.animalBlue, Color.animalLightBlue],
        startPoint: .topLeading,
        endPoint: .bottomTrailing
    )
}
```

Αυτός ο τρόπος γραφής μπορεί να κάνει τη διεπαφή πιο πολυεπίπεδη, αλλά είναι επίσης πιο πιθανό να αυξήσει την οπτική πολυπλοκότητα.

Γι’ αυτό, στην πραγματική ανάπτυξη, μπορούμε να το κατανοούμε έτσι:

Το μονόχρωμο φόντο είναι κατάλληλο για συνηθισμένο κείμενο, συνηθισμένα κουμπιά, συνηθισμένες κάρτες και συνηθισμένα φόντα σελίδων.

Το φόντο με διαβάθμιση είναι κατάλληλο για κύρια κουμπιά, επάνω περιοχές, κάρτες εξωφύλλου, εισόδους λειτουργιών και άλλα σημεία που χρειάζονται έμφαση.

## Σύνοψη

Σε αυτό το μάθημα, μάθαμε πώς να χρησιμοποιούμε προσαρμοσμένα χρώματα στο SwiftUI.

Πρώτα, γνωρίσαμε τα χρώματα Hex.

Για παράδειγμα:

```text
#2c54c2
```

Δηλώνει ένα συγκεκριμένο χρώμα.

Έπειτα, χρησιμοποιήσαμε το `extension Color` για να επεκτείνουμε τον τύπο `Color`.

Έτσι, το SwiftUI μπορεί να δημιουργεί χρώματα με τον παρακάτω τρόπο:

```swift
Color(hex: "#2c54c2")
```

Στη συνέχεια, χρησιμοποιήσαμε το `static` για να οργανώσουμε συχνά χρησιμοποιούμενα χρώματα:

```swift
extension Color {
    static let animalBlue = Color(hex: "#2c54c2")
}
```

Έτσι, σε άλλες προβολές μπορούμε να το χρησιμοποιούμε απευθείας:

```swift
Color.animalBlue
```

Σε σύγκριση με το να γράφουμε κάθε φορά τη συμβολοσειρά Hex, αυτή η προσέγγιση είναι πιο καθαρή και διευκολύνει επίσης την ενιαία τροποποίηση των χρωμάτων αργότερα.

Τέλος, μάθαμε το `LinearGradient` και συνδυάσαμε τα προσαρμοσμένα χρώματα σε φόντο με διαβάθμιση:

```swift
LinearGradient(
    colors: [Color.animalBlue, Color.animalLightBlue],
    startPoint: .leading,
    endPoint: .trailing
)
```

Μέσα από αυτό το μάθημα, ολοκληρώσαμε τη διαδικασία από τη «δημιουργία προσαρμοσμένων χρωμάτων» έως τη «χρήση χρωμάτων σε πραγματική διεπαφή».

Στο μέλλον, όταν ένα χρώμα χρειάζεται να χρησιμοποιηθεί επανειλημμένα σε πολλά σημεία, μπορούμε να εξετάζουμε πρώτα το ενδεχόμενο να το οργανώσουμε στην επέκταση `Color`.

Έτσι ο κώδικας θα είναι πιο καθαρός και το ύφος της διεπαφής θα είναι επίσης πιο εύκολο να παραμείνει ενιαίο.
