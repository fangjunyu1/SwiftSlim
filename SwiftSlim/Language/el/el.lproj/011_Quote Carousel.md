# Καρουζέλ αποφθεγμάτων

Σε αυτό το μάθημα θα υλοποιήσουμε μια λειτουργία "καρουζέλ αποφθεγμάτων" και ταυτόχρονα θα εμβαθύνουμε σε βασικές έννοιες της Swift, όπως τα arrays (`Array`) και οι conditional statements τύπου `if-else`.

Θα δούμε πώς μπορούμε να αποθηκεύσουμε πολλά αποφθέγματα και να υλοποιήσουμε την εναλλαγή τους με κουμπιά, έτσι ώστε να εμφανίζονται κυκλικά.

![alt text](../../RESOURCE/011_word.png)

## Εμφάνιση αποφθέγματος

Αρχικά χρειάζεται να εμφανίσουμε ένα απόφθεγμα μέσα στο SwiftUI.

Ο πιο απλός τρόπος είναι να χρησιμοποιήσουμε ένα `Text` view:

```swift
Text("Slow progress is still progress.")
```

Αυτός ο κώδικας μπορεί να εμφανίσει μόνο ένα σταθερό απόφθεγμα. Αν θέλουμε να εμφανίσουμε πολλά αποφθέγματα και να αλλάζουμε ανάμεσά τους, πρέπει πρώτα να τα αποθηκεύσουμε.

Όμως μια συνηθισμένη μεταβλητή τύπου string μπορεί να αποθηκεύσει μόνο ένα απόφθεγμα:

```swift
let sayings = "Slow progress is still progress."
```

Αν θέλουμε να αποθηκεύσουμε περισσότερα, θα έπρεπε να δηλώσουμε μια ξεχωριστή μεταβλητή για το καθένα:

```swift
let saying1 = "Slow progress is still progress."
let saying2 = "Difficult roads lead to beautiful destinations."
let saying3 = "Awake minds are the strongest."
let saying4 = "Long-term thinking wins over short-term tricks."
let saying5 = "Master one thing, and do it well."
```

Στην πραγματική ανάπτυξη, όμως, αυτή η προσέγγιση είναι όχι μόνο κουραστική, αλλά και άκαμπτη. Επειδή κάθε μεταβλητή είναι ανεξάρτητη, δεν μπορούμε να υλοποιήσουμε εύκολα ευέλικτη εναλλαγή τύπου καρουζέλ.

Για να διαχειριστούμε πιο άνετα πολλά αποφθέγματα, χρειαζόμαστε μια δομή δεδομένων που να τα αποθηκεύει όλα μαζί. Αυτή είναι το array (`Array`).

Με τη χρήση array, ο παραπάνω κώδικας μπορεί να γραφτεί έτσι:

```swift
let sayings = [
    "Slow progress is still progress.",
    "Difficult roads lead to beautiful destinations.",
    "Awake minds are the strongest.",
    "Long-term thinking wins over short-term tricks.",
    "Master one thing, and do it well."
    ]
```

**Σημείωση: Στις συνήθεις coding conventions, τα arrays που αποθηκεύουν πολλά στοιχεία συχνά ονομάζονται στον πληθυντικό, όπως `sayings`, ώστε να φαίνεται ότι πρόκειται για συλλογή.**

## Arrays

Στη Swift, ένα array είναι μια συλλογή από στοιχεία με σειρά, και συμβολίζεται με αγκύλες `[]`.

```swift
[]
```

Μέσα σε ένα array μπορούν να υπάρχουν πολλά στοιχεία του ίδιου τύπου, χωρισμένα με αγγλικό κόμμα `,`.

Για παράδειγμα:

```swift
[101, 102, 103, 104, 105]
```

Μπορούμε να φανταστούμε ένα array σαν ένα τρένο:

![Array](../../RESOURCE/011_array1.png)

Ολόκληρο το τρένο αντιστοιχεί στο αντικείμενο array, ενώ κάθε βαγόνι είναι τοποθετημένο σε μια συγκεκριμένη σειρά.

### Index και πρόσβαση σε στοιχεία

Επειδή τα arrays έχουν σειρά, το σύστημα μπορεί να εντοπίζει συγκεκριμένα στοιχεία τους με βάση τη θέση τους. Αυτός ο μηχανισμός λέγεται index.

Στη Swift, όπως και στις περισσότερες γλώσσες προγραμματισμού, ο index των arrays ξεκινά από το `0` και όχι από το `1`. Αυτό σημαίνει ότι το πρώτο στοιχείο αντιστοιχεί στον index `0`, το δεύτερο στον `1` και ούτω καθεξής.

![Array](../../RESOURCE/011_array2.png)

Αν θέλουμε να προσπελάσουμε ένα συγκεκριμένο στοιχείο μέσα στο array, αρκεί να γράψουμε το όνομα του array και μέσα σε αγκύλες τον αντίστοιχο index.

Για παράδειγμα:

```swift
sayings[0]   // 101
sayings[1]   // 102
```

Αν προσπαθήσουμε να προσπελάσουμε index έξω από τα έγκυρα όρια του array, θα προκύψει σφάλμα "Index Out of Range". Γι’ αυτό, όταν δουλεύουμε με arrays, πρέπει πάντα να βεβαιωνόμαστε ότι ο index είναι έγκυρος.

**Index out of range**

Για παράδειγμα, αν ένα array έχει μόνο 5 στοιχεία, τότε οι έγκυροι indexes είναι από `0` έως `4`. Αν προσπαθήσουμε να προσπελάσουμε το `sayings[5]`, το πρόγραμμα δεν θα μπορέσει να βρει αντίστοιχο «βαγόνι» και θα προκύψει σφάλμα index out of range, με αποτέλεσμα να καταρρεύσει η εφαρμογή.

![Array](../../RESOURCE/011_array3.png)

### Διαχείριση arrays

Τα arrays δεν υποστηρίζουν μόνο στατικό ορισμό, αλλά και προσθήκη, διαγραφή, τροποποίηση στοιχείων, καθώς και ανάκτηση του μήκους τους.

Σημείωση: Αν θέλεις να τροποποιήσεις ένα array, πρέπει να το δηλώσεις με `var` και όχι με `let`.

**1. Προσθήκη στοιχείου**

Μπορούμε να χρησιμοποιήσουμε τη μέθοδο `append` για να προσθέσουμε στοιχείο στο τέλος του array:

```swift
var num = [0, 1, 2]
num.append(3)   // [0, 1, 2, 3]
```

**2. Διαγραφή στοιχείου**

Με τη μέθοδο `remove(at:)`, μπορούμε να αφαιρέσουμε ένα συγκεκριμένο στοιχείο:

```swift
var num = [0, 1, 2]
num.remove(at: 0)   // [1, 2]
```

**3. Τροποποίηση στοιχείου**

Μπορούμε να αλλάξουμε απευθείας ένα στοιχείο χρησιμοποιώντας τον index:

```swift
var num = [0, 1, 2]
num[1] = 4   // [0, 4, 2]
```

**4. Μέγεθος array**

Με το property `count`, μπορούμε να πάρουμε τον αριθμό των στοιχείων:

```swift
var num = [0, 1, 2]
num.count   // 3
```

### Εμφάνιση αποφθεγμάτων με array

Για να εμφανίσουμε πολλά αποφθέγματα, μπορούμε να τα αποθηκεύσουμε σε ένα array και να διαβάζουμε το κατάλληλο στοιχείο μέσω index.

Αρχικά, μέσα στο `ContentView`, δημιουργούμε ένα array `sayings` για να αποθηκεύσουμε τα αποφθέγματα και στη συνέχεια εμφανίζουμε το αντίστοιχο μέσω `Text`:

```swift
struct ContentView: View {
    let sayings = [
        "Slow progress is still progress.",
        "Difficult roads lead to beautiful destinations.",
        "Awake minds are the strongest.",
        "Long-term thinking wins over short-term tricks.",
        "Master one thing, and do it well."
    ]
    
    var body: some View {
        VStack {
            Text(sayings[0])
        }
        .padding()
    }
}
```

Εδώ, το `sayings[0]` σημαίνει το πρώτο απόφθεγμα του array.

Αν θέλουμε να εμφανίσουμε διαφορετικό απόφθεγμα, αρκεί να αλλάξουμε τον index μέσα στις αγκύλες:

```swift
Text(sayings[0]) // "Slow progress is still progress."
Text(sayings[1]) // "Difficult roads lead to beautiful destinations."
```

### Ορισμός του index του αποφθέγματος

Για να υλοποιήσουμε δυναμική εναλλαγή των αποφθεγμάτων, δεν μπορούμε να έχουμε τον index «καρφωμένο» μέσα στο `Text`.

Χρειάζεται να δημιουργήσουμε μια μεταβλητή που να αποθηκεύει ξεχωριστά τον τρέχοντα index.

Στο SwiftUI μπορούμε να δηλώσουμε έναν mutable index με `@State`:

```swift
@State private var index = 0
```

Το SwiftUI παρακολουθεί τις μεταβλητές που είναι τυλιγμένες με `@State`, και όταν αλλάξει το `index`, κάνει ξανά render τη view και εμφανίζει το αντίστοιχο απόφθεγμα.

Στη συνέχεια χρησιμοποιούμε το `sayings[index]` για να διαβάζουμε δυναμικά το απόφθεγμα από το array:

```swift
struct ContentView: View {
    @State private var index = 0
    let sayings = [
        "Slow progress is still progress.",
        "Difficult roads lead to beautiful destinations.",
        "Awake minds are the strongest.",
        "Long-term thinking wins over short-term tricks.",
        "Master one thing, and do it well."
    ]
    
    var body: some View {
        VStack {
            Text(sayings[index])
        }
        .padding()
    }
}
```

Όταν αλλάζει η τιμή του `index`, το `Text` εμφανίζει διαφορετικό απόφθεγμα.

### Έλεγχος του index με button

Για να ελέγχουμε την εναλλαγή των αποφθεγμάτων, μπορούμε να χρησιμοποιήσουμε `Button` ώστε να αυξάνουμε το `index` κατά 1 σε κάθε πάτημα:

```swift
Button("Next") {
    index += 1
}
```

Όταν πατήσουμε το button, το `index` αλλάζει από `0` σε `1`, η view ανανεώνεται και το `Text(sayings[index])` διαβάζει το επόμενο απόφθεγμα.

Ωστόσο εδώ υπάρχει ένα πιθανό πρόβλημα: αν πατάμε συνεχώς το button, το `index` θα συνεχίσει να αυξάνεται και μπορεί να βγει έξω από τα όρια του array. Αυτό θα προκαλέσει σφάλμα index out of range. Για παράδειγμα, όταν το `index` φτάσει στο `5` ενώ το array έχει έγκυρους indexes από `0` έως `4`, το πρόγραμμα θα καταρρεύσει.

Για να αποφύγουμε αυτό το πρόβλημα, πρέπει να εισάγουμε συνθήκη που θα ελέγχει ότι το `index` παραμένει εντός ορίων. Αυτό μπορούμε να το πετύχουμε με `if-else`.

## Έλεγχος συνθηκών: if-else

Η `if-else` είναι από τις πιο συνηθισμένες conditional statements στη Swift. Χρησιμοποιείται για να ελέγχει αν ισχύει μια συνθήκη και, ανάλογα με το αποτέλεσμα, να εκτελεί διαφορετικά blocks κώδικα.

Βασική δομή:

```swift
if condition {
    // Κώδικας που εκτελείται όταν το condition είναι true
} else {
    // Κώδικας που εκτελείται όταν το condition είναι false
}
```

Στο `if`, το `condition` είναι bool τιμή (`Bool`), δηλαδή είτε `true` είτε `false`. Αν η συνθήκη είναι `true`, εκτελείται ο κώδικας του `if`, αλλιώς εκτελείται ο κώδικας του `else`.

Για παράδειγμα:

```swift
let age = 25
if age > 18 {
    print("Big Boy")
} else {
    print("Little Body")
}
```

Σε αυτό το παράδειγμα, το `age` είναι `25`, και το `if` ελέγχει αν το `age > 18` είναι αληθές. Εφόσον η συνθήκη ισχύει, θα εκτελεστεί το `"Big Boy"`.

Αν δεν χρειάζεται `else`, μπορεί να παραλειφθεί:

```swift
if condition {
    // Κώδικας που εκτελείται όταν το condition είναι true
}
```

### Έλεγχος των ορίων του index με συνθήκη

Για να αποτρέψουμε index out of range, μπορούμε να χρησιμοποιήσουμε `if` ώστε να βεβαιωθούμε ότι το `index` δεν θα ξεπεράσει τα όρια του array:

```swift
Button("Next") {
    if index < sayings.count - 1 {
        index += 1
    }
}
```

Ανάλυση λογικής: το `sayings.count` είναι `5`, επειδή έχουμε 5 αποφθέγματα, άρα το `sayings.count - 1` είναι `4`, δηλαδή ο τελευταίος έγκυρος index του array.

Όσο το `index` είναι μικρότερο από `4`, είναι ασφαλές να το αυξήσουμε κατά `1`. Μόλις φτάσει στο `4`, η συνθήκη δεν θα ισχύει πλέον και το button δεν θα προκαλεί καμία αλλαγή.

Τώρα ο κώδικας ήδη υλοποιεί την εναλλαγή των αποφθεγμάτων:

```swift
struct ContentView: View {
    @State private var index = 0
    let sayings = [
        "Slow progress is still progress.",
        "Difficult roads lead to beautiful destinations.",
        "Awake minds are the strongest.",
        "Long-term thinking wins over short-term tricks.",
        "Master one thing, and do it well."
    ]
    var body: some View {
        VStack {
            Text(sayings[index])
            Button("Next") {
                if index < sayings.count - 1 {
                    index += 1
                }
            }
        }
        .padding()
    }
}
```

### Κυκλική εναλλαγή αποφθεγμάτων

Αν θέλουμε, μόλις φτάσουμε στο τελευταίο απόφθεγμα, με το επόμενο πάτημα να επιστρέψουμε στο πρώτο και να έχουμε έτσι κυκλική εναλλαγή, μπορούμε να το κάνουμε μέσω του `else`:

```swift
Button("Next") {
    if index < sayings.count - 1 {
        index += 1
    } else {
        index = 0
    }
}
```

Όταν πατιέται το button και το `index` έχει φτάσει στο τελευταίο στοιχείο, θα επανέλθει στο `0` και τα αποφθέγματα θα αρχίσουν να εμφανίζονται από την αρχή.

## Βελτίωση της view του αποφθέγματος

Η λογική του καρουζέλ αποφθεγμάτων έχει πλέον ολοκληρωθεί, αλλά μπορούμε να βελτιώσουμε και τη διεπαφή ώστε να φαίνεται πιο όμορφη.

Πλήρης κώδικας:

```swift
struct ContentView: View {
    @State private var index = 0
    let sayings = [
        "Slow progress is still progress.",
        "Difficult roads lead to beautiful destinations.",
        "Awake minds are the strongest.",
        "Long-term thinking wins over short-term tricks.",
        "Master one thing, and do it well."
    ]
    var body: some View {
        VStack(spacing: 10) {
            Text(sayings[index])
                .padding(20)
                .background(.white)
                .cornerRadius(10)
                .opacity(0.8)
            Button("Next") {
                if index < sayings.count - 1 {
                    index += 1
                } else {
                    index = 0
                }
            }
            .buttonStyle(.borderedProminent)
        }
        .padding()
        .background(
            Image("001")
        )
    }
}
```

Σε αυτό το παράδειγμα, το `Text` έχει λευκό ημιδιαφανές background και στρογγυλεμένες γωνίες, το `Button` χρησιμοποιεί το style `.borderedProminent` και το `VStack` έχει background image.

Ένα επιπλέον σημείο γνώσης: όταν χρησιμοποιούμε τον modifier `background()` με εικόνα, η προεπιλεγμένη του συμπεριφορά είναι να γεμίζει όσο περισσότερο γίνεται την περιοχή layout της τρέχουσας view. Σε πολλές περιπτώσεις μπορεί να επεκταθεί φυσικά ώστε να καλύψει ακόμη και το Safe Area.

## Σύνοψη

Μέσα από αυτό το μάθημα μάθαμε πώς να χρησιμοποιούμε arrays για να αποθηκεύουμε πολλά αποφθέγματα και πώς, με `if` και `if-else`, μπορούμε να υλοποιήσουμε την εναλλαγή τους.

Μάθαμε επίσης τις βασικές λειτουργίες πάνω στα arrays, όπως προσθήκη, διαγραφή και τροποποίηση στοιχείων, καθώς και τον τρόπο αποφυγής του index out of range.

Αυτό το μάθημα δεν εξηγεί μόνο τη λειτουργία του καρουζέλ αποφθεγμάτων, αλλά ταυτόχρονα συνδυάζει τη βασική χρήση arrays και conditionals, βοηθώντας μας να χειριζόμαστε δεδομένα και να ελέγχουμε τη ροή του προγράμματος.

## Επέκταση - πολλαπλές συνθήκες με if-else if-else

Στην πραγματική ανάπτυξη, συχνά χρειάζεται να χειριζόμαστε περισσότερες από μία συνθήκες. Για παράδειγμα, σε κάποιο παιχνίδι μπορεί όταν το σκορ είναι `1` να ενεργοποιείται event A, όταν είναι `2` να ενεργοποιείται event B, όταν είναι `3` να ενεργοποιείται event C και ούτω καθεξής.

Όταν έχουμε περισσότερα από δύο branches συνθηκών, χρησιμοποιούμε `if-else if-else`.

Βασική σύνταξη:

```swift
if conditionA {
    // Κώδικας που εκτελείται όταν το conditionA είναι true
} else if conditionB {
    // Κώδικας που εκτελείται όταν το conditionB είναι true
} else if conditionC {
    // Κώδικας που εκτελείται όταν το conditionC είναι true
} else {
    // Κώδικας που εκτελείται όταν δεν ισχύει καμία συνθήκη
}
```

Σε αυτήν την περίπτωση, το πρόγραμμα ελέγχει τις συνθήκες με τη σειρά και εκτελεί την πρώτη που είναι αληθής. Αν καμία δεν ισχύει, θα εκτελεστεί το block του `else`.

Στο καρουζέλ αποφθεγμάτων μπορούμε επίσης να χρησιμοποιήσουμε `if-else if-else` για πολλαπλούς ελέγχους:

```swift
Button("Next") {
    if index == 0 {
        index += 1
    } else if index == 1 {
        index += 1
    } else if index == 2 {
        index += 1
    } else if index == 3 {
        index += 1
    } else if index == 4 {
        index = 0
    } else {
        index = 0
    }
}
```

Όταν το `index` είναι `0`, `1`, `2` ή `3`, κάθε πάτημα αυξάνει το `index` κατά `1`. Όταν είναι `4`, δηλαδή το τελευταίο στοιχείο, το `index` επανέρχεται στο `0`, ώστε να δημιουργείται κύκλος.

Το branch του `else` λειτουργεί ως ασφάλεια για να καλύψει μη έγκυρες τιμές, για παράδειγμα αν το `index` άλλαξε λανθασμένα από κάπου αλλού.

Πρέπει επίσης να προσέξουμε ότι εδώ το `==` σημαίνει έλεγχο ισότητας. Αν το `if` διαπιστώσει ότι το `index` είναι ίσο με έναν συγκεκριμένο αριθμό, η συνθήκη επιστρέφει `true` και εκτελείται το αντίστοιχο block κώδικα. Αν δεν είναι ίσο, ο έλεγχος περνά στην επόμενη συνθήκη.

Αυτό το είδος πολλαπλού ελέγχου είναι χρήσιμο όταν θέλουμε να εκτελούμε διαφορετικό κώδικα ανάλογα με το σενάριο.
