# Εισαγωγή

## Τι είναι η Swift;

Πριν μάθουμε Swift, ας μιλήσουμε λίγο για τη γλώσσα προγραμματισμού Swift. Η Swift είναι μια σύγχρονη γλώσσα προγραμματισμού που αναπτύχθηκε από την Apple, η οποία χρησιμοποιείται κυρίως για την κατασκευή εφαρμογών σε πλατφόρμες της Apple όπως iOS, macOS, iPadOS, watchOS κ.λπ.

![Swift](000_swift.png)

Σε σύγκριση με άλλες γλώσσες προγραμματισμού, η σύνταξη της Swift είναι συνοπτική και δίνει μεγαλύτερη έμφαση στην ασφάλεια και την αναγνωσιμότητα. Σε συνδυασμό με το SwiftUI, είναι πιο εύκολη στην εκμάθηση και την κατανόηση. Φυσικά, δεν έχω μάθει άλλες γλώσσες προγραμματισμού στο παρελθόν, οπότε μπορεί να είναι λίγο πιο δύσκολη από την HTML.

Η Swift δεν χρησιμεύει μόνο για την ανάπτυξη εφαρμογών στις πλατφόρμες της Apple. Η κοινότητα της Swift εργάζεται σκληρά για να μεταφέρει τη Swift στην πλατφόρμα [Android](https://www.swift.org/blog/nightly-swift-sdk-for-android/), ακόμα και για να την τρέξει σε [διακομιστές (servers)](https://github.com/awslabs/swift-aws-lambda-runtime) και άλλα περιβάλλοντα. Από την άποψη της μακροπρόθεσμης ανάπτυξης, η Swift έχει μεγαλύτερο περιθώριο ανάπτυξης και στο μέλλον έχει την ευκαιρία να γίνει μια γλώσσα προγραμματισμού πολλαπλών πλατφορμών (cross-platform), παρόμοια με το Flutter και το React Native.

## Περιεχόμενο Εκμάθησης

Αυτό το εκπαιδευτικό πρόγραμμα επικεντρώνεται κυρίως στις βασικές γνώσεις της Swift και του SwiftUI.

Θα ξεκινήσουμε μαθαίνοντας απλές μεταβλητές και τύπους δεδομένων, θα χρησιμοποιήσουμε το SwiftUI για να χτίσουμε διεπαφές, θα κατανοήσουμε πώς αποθηκεύονται και διαβάζονται τα δεδομένα και τέλος, ίσως εμβαθύνουμε σε προχωρημένο περιεχόμενο όπως τα Swift Data και ο συγχρονισμός iCloud.

Κάθε κεφάλαιο θα περιστρέφεται γύρω από μια πραγματική περίπτωση, μαθαίνοντας γνώσεις Swift στην πράξη.

Κάποια εκπαιδευτικά προγράμματα μπορεί να επικεντρώνονται πρώτα στην εξήγηση μεγάλης ποσότητας σύνταξης και API πριν περάσουν στην πρακτική εφαρμογή σε έργα. Για τους αρχάριους, αυτό είναι σαν ένα παιδί που μαθαίνει λέξεις από ένα λεξικό· η απομνημόνευση μεγάλου όγκου περιεχομένου χωρίς σενάρια εφαρμογής θα οδηγήσει τελικά μόνο στην απώλεια ενδιαφέροντος από τον αρχάριο.

Σε αυτό το εκπαιδευτικό πρόγραμμα, θα μάθουμε γνώσεις σε πραγματικά σενάρια ανάπτυξης και θα χρησιμοποιήσουμε τις απαιτήσεις για να εισαγάγουμε τα σχετικά σημεία γνώσης.

Μετά την ολοκλήρωση αυτού του εκπαιδευτικού προγράμματος, θα είστε σε θέση να αναπτύξετε και να τρέξετε ανεξάρτητα μια βασική εφαρμογή, να την εγκαταστήσετε στη δική σας συσκευή iOS / macOS ή ακόμα και να δημοσιεύσετε την εφαρμογή στο App Store.

## Πώς να μάθετε από το μηδέν;

Και εγώ ξεκίνησα από το μηδέν. Στην αρχή, απλώς περιηγήθηκα σε μερικά [εκπαιδευτικά βίντεο Swift]( εφαρμογών iOS δεν είναι περίπλοκη, κάτι που μου προκάλεσε το ενδιαφέρον. Στη συνέχεια, άρχισα να προσπαθώ να αποκτήσω περισσότερες γνώσεις Swift. Για παράδειγμα, παίζοντας παιχνίδια προγραμματισμού στο [Swift Playgrounds](https://apps.apple.com/cn/app/swift-playgrounds/id1496833156?mt=12), διαβάζοντας την [επίσημη τεκμηρίωση εκμάθησης της Swift](https://docs.swift.org/swift-book/documentation/the-swift-programming-language/) και μελετώντας το [100 Days Of SwiftUI](https://www.hackingwithswift.com/100/swiftui).

Τώρα φαίνεται ότι κυρίως το [100 Days Of SwiftUI](https://www.hackingwithswift.com/100/swiftui) ήταν αυτό που μου παρείχε ένα πλαίσιο μάθησης, επιτρέποντάς μου να μάθω το SwiftUI συστηματικά. Ωστόσο, επειδή το εκπαιδευτικό υλικό είναι στα αγγλικά και μου έλειπαν οι βάσεις προγραμματισμού, αντιμετώπισα πολλές δυσκολίες στην κατανόηση. Για το σκοπό αυτό, χρησιμοποίησα AI όπως το [ChatGPT](https://chatgpt.com/) και το [Claude](https://claude.ai/?redirect=claude.com) για να βοηθηθώ στην κατανόηση των σημείων γνώσης.

Στη διαδικασία της μάθησης, είναι αναπόφευκτο να εμφανιστούν κάποιες γνώσεις που είναι δύσκολο να κατανοηθούν. Θυμάμαι ότι όταν μάθαινα τα UserDefaults, ήξερα μόνο πώς να αποθηκεύω δεδομένα, αλλά δεν ήξερα ότι τα δεδομένα έπρεπε να διαβαστούν χειροκίνητα, με αποτέλεσμα να υποψιάζομαι ότι υπήρχε πρόβλημα στον κώδικά μου. Ακόμα και ένα τόσο απλό σημείο γνώσης με έκανε να σπαταλήσω μερικές μέρες.

Ή για παράδειγμα τα closures (κλεισίματα), το main thread (κύριο νήμα), τα generics (γενικά) και άλλα σημεία. Καθώς δεν μπορούσα να τα κατανοήσω σε βάθος στο αρχικό στάδιο της μάθησης, όταν τα συναντούσα ξανά στην πραγματική ανάπτυξη, έπρεπε να αφιερώσω ενέργεια για να τα ξαναμάθω.

Επομένως, η πρότασή μου είναι:

**1. Συμπληρώστε τις βασικές γνώσεις**

Υπάρχει μια κινεζική παροιμία που λέει "Κάθε αρχή και δύσκολη". Για τους αρχάριους από το μηδέν, όταν ξεκινούν να μαθαίνουν Swift, ορισμένα σημεία γνώσης είναι συχνά δύσκολα στην κατανόηση. Αυτό δεν οφείλεται στο ότι το ίδιο το σημείο γνώσης είναι περίπλοκο, αλλά στο ότι κάτω από αυτή την πυραμίδα γνώσης υπάρχουν πιο βασικές γνώσεις που δεν έχετε μάθει ή γνωρίσει.

Σε αυτή την περίπτωση, θα πρέπει να ρωτήσετε AI όπως το [ChatGPT](https://chatgpt.com/) και το [Claude](https://claude.ai/?redirect=claude.com) για να συμπληρώσετε τις βασικές γνώσεις.

**2. Προσπαθήστε να κατανοήσετε κάθε σημείο γνώσης**

Στη διαδικασία της μάθησης, θα πρέπει να προσπαθήσετε να κατανοήσετε κάθε σημείο γνώσης, ακόμα κι αν χρειαστεί να ξοδέψετε μερικές μέρες.

Όταν επιλέγετε να παραλείψετε περιεχόμενο που δεν κατανοείτε, θα το συναντήσετε ξανά στη μετέπειτα ανάπτυξη και θα εμποδίσει την πρόοδό σας, αναγκάζοντάς σας τελικά να επενδύσετε ξανά χρόνο για να το μάθετε. Αντί να προσπαθείτε να διορθώσετε τα πράγματα όταν είναι αργά, είναι καλύτερο να κατανοήσετε τις γνώσεις καθαρά στο στάδιο της μάθησης.

Κάθε σημείο γνώσης είναι ουσιαστικά ένα εργαλείο – όσο περισσότερα εργαλεία κατέχετε, τόσο πιο ισχυρή είναι η ικανότητά σας να λύνετε προβλήματα.

Ακριβώς όπως το χτίσιμο ενός σπιτιού, αν έχετε μόνο ένα εργαλείο στο χέρι, αν και ίσως μπορέσετε με δυσκολία να ολοκληρώσετε το μεγαλύτερο μέρος της δουλειάς, θα φανείτε ανίσχυροι μπροστά σε απαιτήσεις υψηλότερης ποιότητας. Η κατοχή περισσότερων εργαλείων επιτρέπει το χτίσιμο ενός πιο σταθερού σπιτιού.

**3. Διατηρήστε συνεχή μάθηση**

Η μάθηση απαιτεί συνέπεια. Λίγοι άνθρωποι είναι πραγματικά πρόθυμοι να αφιερώσουν μερικούς μήνες για να μελετήσουν σοβαρά ένα εκπαιδευτικό πρόγραμμα. Η συνεχής επένδυση χρόνου στη μάθηση είναι πιο σημαντική από τη βραχυπρόθεσμη υψηλή ένταση.

Ακόμα κι αν ολοκληρώνετε μόνο το ένα τοις εκατό κάθε μέρα, μπορείτε να ολοκληρώσετε ολόκληρο τον στόχο μάθησης σε εκατό ημέρες.
