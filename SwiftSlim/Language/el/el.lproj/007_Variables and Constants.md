# Μεταβλητές και σταθερές

Σε αυτό το μάθημα θα μάθουμε κυρίως τις μεταβλητές και τις σταθερές στη Swift, καθώς και μερικούς συνηθισμένους τύπους δεδομένων και βασικούς τελεστές.

Αυτά είναι από τα πιο θεμελιώδη στοιχεία του προγραμματισμού και αποτελούν επίσης σημαντική βάση για τη συνέχεια στη SwiftUI.

## Μεταβλητές και σταθερές στην καθημερινή ζωή

Μπορούμε να κατανοήσουμε τις μεταβλητές και τις σταθερές μέσα από πράγματα της καθημερινότητας.

Παραδείγματα μεταβλητών στην καθημερινή ζωή:

- Κάθε πρόγραμμα στην τηλεόραση προβάλλει διαφορετικό περιεχόμενο
- Ο καιρός είναι διαφορετικός κάθε μέρα
- Το ρολόι αλλάζει κάθε δευτερόλεπτο

Το κοινό χαρακτηριστικό αυτών των πραγμάτων είναι ότι αλλάζουν.

Αν η τηλεόραση έδειχνε πάντα μόνο μία εικόνα, αν ο καιρός ήταν πάντα ηλιόλουστος και αν οι δείκτες του ρολογιού δεν κινούνταν ποτέ, τότε αυτά τα πράγματα θα ήταν σταθερές.

Το ένα μπορεί να αλλάξει, το άλλο όχι.

## Κατανόηση μεταβλητών και σταθερών

Κατά την ανάπτυξη μιας εφαρμογής, συνήθως χρειάζεται ο χρήστης να συμπληρώσει ή να αποθηκεύσει κάποιες πληροφορίες.

Για παράδειγμα:

- Όνομα λογαριασμού
- Ημερομηνία γέννησης
- Στοιχεία επικοινωνίας
- Διεύθυνση

Αυτές οι πληροφορίες θα αποθηκευτούν και θα εμφανιστούν στην εφαρμογή.

Για παράδειγμα, αν ο χρήστης εισάγει ένα όνομα:

```text
FangJunyu
```

χρειάζεται να αποθηκεύσουμε αυτό το όνομα ώστε να μπορεί να εμφανιστεί στην εφαρμογή.

Μπορούμε να φανταστούμε τη διαδικασία αποθήκευσης σαν να βάζουμε κάτι μέσα σε ένα συρτάρι.

Όταν αποθηκεύουμε ένα όνομα, είναι σαν να βάζουμε το όνομα μέσα σε ένα συρτάρι.

Τα πράγματα που χρειάζεται να αποθηκευτούν μπορεί να είναι πολλά, και άρα και τα συρτάρια θα είναι πολλά. Για να ξέρουμε τι περιέχει κάθε συρτάρι, πρέπει να του δώσουμε ένα όνομα.

Για παράδειγμα:

```text
name
```

Σε αυτό το παράδειγμα, το `name` είναι το όνομα και το `FangJunyu` είναι η αποθηκευμένη πληροφορία.

![Var](../../RESOURCE/007_var.png)

**Στη Swift, αν θέλουμε να αποθηκεύσουμε δεδομένα, πρέπει να τα δηλώσουμε είτε με μεταβλητή (`var`) είτε με σταθερά (`let`).**

Επειδή ένα όνομα συνήθως μπορεί να αλλάξει, εδώ θα πρέπει να χρησιμοποιήσουμε μεταβλητή.

```swift
var name = "FangJunyu"
```

Εδώ δηλώνεται μια μεταβλητή με όνομα `name`, τύπου `String`, με τιμή `"FangJunyu"`.

### Η διαφορά ανάμεσα σε μεταβλητές και σταθερές

Οι μεταβλητές δηλώνονται με `var`:

```swift
var
```

Οι σταθερές δηλώνονται με `let`:

```swift
let
```

Για παράδειγμα:

```swift
var name = "FangJunyu"
let id = 123456
```

Αν κάτι δηλωθεί ως μεταβλητή, η τιμή του μπορεί να αλλάξει. Αν δηλωθεί ως σταθερά, η τιμή του δεν μπορεί πια να αλλάξει.

Επομένως, η βασική διαφορά ανάμεσα σε μεταβλητή και σταθερά είναι το αν επιτρέπεται να αλλάξει η τιμή.

### Τροποποίηση σταθεράς

Αν ο κώδικας προσπαθήσει να αλλάξει την τιμή μιας σταθεράς:

```swift
let name = "Sam"
name = "Bob"
```

η Swift θα εμφανίσει ότι δεν μπορεί να ολοκληρωθεί η ανάθεση, επειδή πρόκειται για σταθερά.

```text
Cannot assign to property: 'name' is a 'let' constant
```

Αυτός ο μηχανισμός βοηθά τους προγραμματιστές να αποφεύγουν την τυχαία αλλαγή σημαντικών δεδομένων.

### Εμφάνιση στη SwiftUI

Άνοιξε το αρχείο `ContentView.swift` και δήλωσε μια μεταβλητή μέσα στη View:

```swift
struct ContentView: View {
    var name = "FangJunyu"

    var body: some View {
        Text(name)
    }
}
```

Εδώ δηλώνεται μια μεταβλητή με όνομα `name` και εμφανίζεται στην οθόνη μέσω του `Text`.

Αν αλλάξουμε το περιεχόμενο της μεταβλητής:

```swift
struct ContentView: View {
    var name = "Sam"
    var body: some View {
        Text(name)
    }
}
```

τότε αλλάζει και το κείμενο που εμφανίζει το `Text`.

Με αυτόν τον τρόπο μπορούμε να χρησιμοποιούμε μεταβλητές για να ελέγχουμε το περιεχόμενο που εμφανίζεται στη διεπαφή, χωρίς να αλλάζουμε κάθε φορά χειροκίνητα το κείμενο μέσα στο `Text`.

Αν χρειάζεται να χρησιμοποιηθεί αυτή η πληροφορία και αλλού, μπορούμε να περάσουμε τη μεταβλητή εκεί, σαν να δίνουμε το περιεχόμενο του «συρταριού» σε άλλο σημείο.

### Η θέση μεταβλητών και σταθερών στη SwiftUI

Στη SwiftUI, οι μεταβλητές και οι σταθερές συνήθως γράφονται έξω από το `body`:

```swift
struct ContentView: View {
    var name = "FangJunyu"

    var body: some View {
        Text(name)
    }
}
```

Φυσικά, μπορούμε επίσης να τις γράψουμε και μέσα στο `body`:

```swift
struct ContentView: View {
    var body: some View {
        var name = "FangJunyu"
        Text(name)
    }
}
```

Ωστόσο υπάρχει μια σημαντική διαφορά ανάμεσα στις δύο περιπτώσεις:

- Αν οι μεταβλητές και οι σταθερές γραφτούν μέσα στο `body`, τότε κάθε φορά που η view επανυπολογίζεται, αυτές δημιουργούνται ξανά
- Αν γραφτούν έξω από το `body`, υπάρχουν ως properties της δομής της view και ο κώδικας είναι πιο καθαρός

Γι’ αυτό, στην πράξη, οι μεταβλητές και οι σταθερές συνήθως γράφονται έξω από το `body`.

## Τύποι δεδομένων

Οι μεταβλητές μπορούν να αποθηκεύουν πολλά διαφορετικά είδη δεδομένων. Αυτά τα είδη ονομάζονται τύποι δεδομένων.

Για παράδειγμα, προηγουμένως χρησιμοποιήσαμε `Text` για να εμφανίσουμε κείμενο:

```swift
Text("Hello, World")
```

Εδώ το `"Hello, World"` είναι τύπος `String`.

Το `String` χρησιμοποιείται για να αναπαριστά κείμενο και στη Swift πρέπει να βρίσκεται μέσα σε διπλά εισαγωγικά `""`.

Για παράδειγμα:

```swift
var hello = "Hello, World"
```

Εκτός από τα strings, υπάρχουν πολλοί ακόμη τύποι δεδομένων στη Swift.

Στο αρχικό στάδιο, οι τέσσερις πιο συνηθισμένοι είναι:

- `String`
- `Int`
- `Double`
- `Bool`

Με αυτούς τους τέσσερις τύπους μπορούμε να καλύψουμε το μεγαλύτερο μέρος της βασικής επεξεργασίας δεδομένων.

**String**

Το `String` αναπαριστά περιεχόμενο κειμένου, για παράδειγμα:

```swift
var name = "FangJunyu"
var city = "Rizhao"
```

Τα strings χρησιμοποιούνται συνήθως για ονόματα χρηστών, τίτλους ή γενικό κείμενο.

**Int**

Το `Int` σημαίνει ακέραιος αριθμός, δηλαδή αριθμός χωρίς δεκαδικά.

Για παράδειγμα:

```swift
var age = 26
var count = 100
```

Οι ακέραιοι χρησιμοποιούνται συνήθως για ηλικία, ποσότητα ή μετρητές.

**Double**

Το `Double` αναπαριστά αριθμούς με δεκαδικά.

Για παράδειγμα:

```swift
var weight = 74.5
var height = 185.0
```

Στη Swift, οι δεκαδικοί αριθμοί χρησιμοποιούν συνήθως από προεπιλογή τον τύπο `Double`.

Υπάρχει και ένας παρόμοιος τύπος, ο `Float`, αλλά μπορεί να αναπαραστήσει μικρότερο εύρος τιμών, γι’ αυτό στην πράξη χρησιμοποιείται συχνότερα το `Double`.

**Bool**

Ο τύπος `Bool` χρησιμοποιείται για την αναπαράσταση δύο καταστάσεων: `true` και `false`.

Για παράδειγμα:

```swift
var isShowAlert = true
var isLogin = false
```

Το `Bool` χρησιμοποιείται συχνά σε συνθήκες, για παράδειγμα για να αποφασίσουμε αν θα εμφανιστεί κάποιο alert.

Όταν υπάρχουν μόνο δύο πιθανά αποτελέσματα, το `Bool` είναι πολύ κατάλληλο.

### Εμφάνιση στη SwiftUI

Όταν θέλουμε να εμφανίσουμε τιμή τύπου `String`, μπορούμε να τη χρησιμοποιήσουμε απευθείας σε `Text`:

```swift
Text(name)
```

Όμως οι τύποι `Int`, `Double` κτλ. δεν είναι strings, οπότε δεν μπορούν να χρησιμοποιηθούν απευθείας ως μέρος κειμένου.

Αν θέλουμε να εμφανίσουμε αυτές τις τιμές μέσα σε `Text`, μπορούμε να χρησιμοποιήσουμε string interpolation:

```swift
\()
```

Το string interpolation γράφεται μέσα σε string και τυλίγει τη μεταβλητή ή τη σταθερά με `\()`.

Για παράδειγμα:

```swift
var num = 1

Text("\(num)")
Text("DoubleNum: \(num * 2)")
```

Στον παραπάνω κώδικα, τα `""` δηλώνουν string, και το `\()` χρησιμοποιείται για να ενσωματώνει μεταβλητές ή σταθερές μέσα σε αυτό το string.

Για παράδειγμα:

```swift
"DoubleNum: \(num * 2)"
```

Το περιεχόμενο που εμφανίζεται είναι:

```text
DoubleNum: 2
```

Με αυτόν τον τρόπο μπορούμε να εισάγουμε μεταβλητές ή σταθερές μέσα σε string και έπειτα να τις εμφανίσουμε στην οθόνη με `Text`.

Σημείωση: Το `\()` μπορεί να χρησιμοποιηθεί μόνο μέσα σε string `""`.

## Τελεστές

Όταν δηλώνουμε μια μεταβλητή ή μια σταθερά, συναντάμε συχνά γραφή όπως η παρακάτω:

```swift
var num = 1
```

Το `=` ονομάζεται τελεστής ανάθεσης.

Η λειτουργία του είναι να αναθέτει την τιμή που βρίσκεται δεξιά στη μεταβλητή που βρίσκεται αριστερά.

Σε αυτό το παράδειγμα, το `1` ανατίθεται στη μεταβλητή `num`.

Εκτός από τον τελεστή ανάθεσης, υπάρχουν και μερικοί συνηθισμένοι αριθμητικοί τελεστές:

- `+`
- `-`
- `*`
- `/`

Όταν κάνουμε πράξεις με αριθμούς, για παράδειγμα με δεδομένα τύπου `Int` ή `Double`, χρησιμοποιούμε αυτούς τους τελεστές.

Για παράδειγμα:

```swift
var age = 21 + 3 // 24
var value = 30 - 2 // 28
var num = 1 * 3 // 3
var weight = 88.5 / 2 // 44.25
```

Αφού ολοκληρωθεί ο υπολογισμός, το αποτέλεσμα αποθηκεύεται στη μεταβλητή.

## Σύνοψη

Οι μεταβλητές, οι σταθερές και οι τελεστές είναι από τις πιο βασικές έννοιες του προγραμματισμού.

Μέσα από μεταβλητές και σταθερές μπορούμε να αποθηκεύουμε διαφορετικά δεδομένα μέσα στο πρόγραμμα· μέσω των τύπων δεδομένων μπορούμε να ξέρουμε τι είδους είναι αυτά τα δεδομένα· και μέσω των τελεστών μπορούμε να κάνουμε υπολογισμούς και επεξεργασία πάνω τους.

Αυτές οι γνώσεις είναι σαν τα βασικά εργαλεία στον κόσμο του προγραμματισμού. Αν τις κατακτήσεις, θα χτίσεις μια σταθερή βάση για να συνεχίσεις με Swift και SwiftUI.
