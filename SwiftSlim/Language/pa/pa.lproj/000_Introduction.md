# ਭੂਮਿਕਾ

## Swift ਕੀ ਹੈ?

Swift ਸਿੱਖਣ ਤੋਂ ਪਹਿਲਾਂ, ਆਓ ਪਹਿਲਾਂ Swift ਪ੍ਰੋਗ੍ਰਾਮਿੰਗ ਭਾਸ਼ਾ ਬਾਰੇ ਸੰਖੇਪ ਵਿੱਚ ਜਾਣੀਏ। Swift Apple ਦੁਆਰਾ ਵਿਕਸਿਤ ਕੀਤੀ ਇੱਕ ਆਧੁਨਿਕ ਪ੍ਰੋਗ੍ਰਾਮਿੰਗ ਭਾਸ਼ਾ ਹੈ, ਜੋ ਮੁੱਖ ਤੌਰ 'ਤੇ iOS, macOS, iPadOS, watchOS ਆਦਿ Apple ਪਲੇਟਫਾਰਮਾਂ ਉੱਤੇ ਐਪਲੀਕੇਸ਼ਨ ਬਣਾਉਣ ਲਈ ਵਰਤੀ ਜਾਂਦੀ ਹੈ।

![Swift](../../RESOURCE/000_swift.png)

ਹੋਰ ਪ੍ਰੋਗ੍ਰਾਮਿੰਗ ਭਾਸ਼ਾਵਾਂ ਨਾਲ ਤੁਲਨਾ ਕਰਦੇ ਹੋਏ, Swift ਦੀ syntax ਹੋਰ ਸੰਖੇਪ ਹੈ ਅਤੇ ਇਹ safety ਅਤੇ readability 'ਤੇ ਵੱਧ ਜ਼ੋਰ ਦਿੰਦੀ ਹੈ। SwiftUI ਨਾਲ ਮਿਲ ਕੇ ਇਸਨੂੰ ਸਿੱਖਣਾ ਅਤੇ ਸਮਝਣਾ ਹੋਰ ਆਸਾਨ ਹੋ ਜਾਂਦਾ ਹੈ। ਬੇਸ਼ੱਕ, ਮੈਂ ਪਹਿਲਾਂ ਹੋਰ ਪ੍ਰੋਗ੍ਰਾਮਿੰਗ ਭਾਸ਼ਾਵਾਂ ਨਹੀਂ ਸਿੱਖੀਆਂ ਸਨ, ਇਸ ਲਈ ਇਹ ਸ਼ਾਇਦ HTML ਨਾਲੋਂ ਥੋੜ੍ਹੀ ਜ਼ਿਆਦਾ ਮੁਸ਼ਕਲ ਲੱਗ ਸਕਦੀ ਹੈ।

Swift ਸਿਰਫ਼ Apple ਪਲੇਟਫਾਰਮਾਂ ਉੱਤੇ ਐਪਲੀਕੇਸ਼ਨ ਬਣਾਉਣ ਲਈ ਹੀ ਨਹੀਂ ਵਰਤੀ ਜਾਂਦੀ, Swift community ਇਸਨੂੰ [Android](https://www.swift.org/blog/nightly-swift-sdk-for-android/) ਪਲੇਟਫਾਰਮ 'ਤੇ ਲਿਜਾਣ ਲਈ ਵੀ ਕੋਸ਼ਿਸ਼ ਕਰ ਰਹੀ ਹੈ, ਅਤੇ ਇਹ [server](https://github.com/awslabs/swift-aws-lambda-runtime) ਵਰਗੇ ਹੋਰ environment ਵਿੱਚ ਵੀ ਚੱਲ ਸਕਦੀ ਹੈ। ਲੰਬੇ ਸਮੇਂ ਦੇ ਵਿਕਾਸਕ ਨਜ਼ਰੀਏ ਤੋਂ ਵੇਖੀਏ ਤਾਂ Swift ਦੇ ਵਿਕਾਸ ਦੀ ਸੰਭਾਵਨਾ ਵੱਧ ਹੈ, ਅਤੇ ਭਵਿੱਖ ਵਿੱਚ ਇਸਦੇ Flutter, React Native ਵਰਗੀਆਂ cross-platform ਪ੍ਰੋਗ੍ਰਾਮਿੰਗ ਭਾਸ਼ਾਵਾਂ ਵਾਂਗ ਬਣਨ ਦੀ ਸੰਭਾਵਨਾ ਹੈ।

## ਸਿੱਖਣ ਵਾਲੀ ਸਮੱਗਰੀ

ਇਹ ਟਿਊਟੋਰਿਯਲ ਮੁੱਖ ਤੌਰ 'ਤੇ Swift ਅਤੇ SwiftUI ਦੇ ਬੁਨਿਆਦੀ ਗਿਆਨ ਉੱਤੇ ਆਧਾਰਿਤ ਹੈ।

ਅਸੀਂ ਸਧਾਰਣ variables ਅਤੇ data types ਤੋਂ ਸ਼ੁਰੂ ਕਰਾਂਗੇ, SwiftUI ਨਾਲ interface ਬਣਾਉਣਾ ਸਿੱਖਾਂਗੇ, data ਨੂੰ store ਅਤੇ read ਕਰਨ ਦੇ ਤਰੀਕੇ ਨੂੰ ਸਮਝਾਂਗੇ, ਅਤੇ ਆਖ਼ਿਰ ਵਿੱਚ ਸ਼ਾਇਦ SwiftData, iCloud sync ਵਰਗੇ ਉੱਚ ਪੱਧਰੀ ਵਿਸ਼ਿਆਂ ਤੱਕ ਵੀ ਜਾਵਾਂਗੇ।

ਹਰ ਅਧਿਆਇ ਇੱਕ ਅਸਲੀ case ਨੂੰ ਕੇਂਦਰ ਬਣਾਕੇ ਅੱਗੇ ਵਧੇਗਾ, ਤਾਂ ਜੋ practical ਵਰਤੋਂ ਵਿੱਚ Swift ਦਾ ਗਿਆਨ ਸਿੱਖਿਆ ਜਾ ਸਕੇ।

ਕੁਝ ਟਿਊਟੋਰਿਯਲ ਪਹਿਲਾਂ ਬਹੁਤ ਸਾਰੀ syntax ਅਤੇ API ਨੂੰ ਇਕੱਠੇ ਸਮਝਾਉਂਦੇ ਹਨ, ਫਿਰ project practice ਵੱਲ ਜਾਂਦੇ ਹਨ। ਸ਼ੁਰੂਆਤੀ ਸਿਖਿਆਰਥੀਆਂ ਲਈ ਇਹ ਓਹੋ ਜਿਹਾ ਹੈ ਜਿਵੇਂ ਕੋਈ ਬੱਚਾ dictionary ਵਿਚੋਂ ਅੱਖਰ ਸਿੱਖੇ — ਵਰਤੋਂ ਦੇ ਸੰਦਰਭ ਤੋਂ ਬਿਨਾਂ ਬਹੁਤ ਸਾਰੀ ਚੀਜ਼ਾਂ ਯਾਦ ਕਰਨਾ ਆਖ਼ਰਕਾਰ ਸਿਖਿਆਰਥੀ ਦੀ ਦਿਲਚਸਪੀ ਘਟਾ ਦਿੰਦਾ ਹੈ।

ਇਹ ਟਿਊਟੋਰਿਯਲ ਅਸਲੀ development scenarios ਵਿੱਚ ਗਿਆਨ ਸਿੱਖਣ ਦੀ ਕੋਸ਼ਿਸ਼ ਕਰਦਾ ਹੈ, ਜਿੱਥੇ ਲੋੜਾਂ ਦੇ ਆਧਾਰ 'ਤੇ ਸੰਬੰਧਿਤ concepts ਨੂੰ ਪੇਸ਼ ਕੀਤਾ ਜਾਂਦਾ ਹੈ।

ਇਸ ਟਿਊਟੋਰਿਯਲ ਨੂੰ ਪੂਰਾ ਕਰਨ ਤੋਂ ਬਾਅਦ, ਤੁਸੀਂ ਖੁਦ ਇੱਕ ਬੁਨਿਆਦੀ ਐਪਲੀਕੇਸ਼ਨ ਬਣਾਕੇ ਚਲਾ ਸਕੋਗੇ, ਇਸਨੂੰ ਆਪਣੇ iOS / macOS devices 'ਤੇ install ਕਰ ਸਕੋਗੇ, ਅਤੇ ਐਪ ਨੂੰ App Store 'ਤੇ publish ਵੀ ਕਰ ਸਕੋਗੇ।

## ਬਿਲਕੁਲ ਬੁਨਿਆਦ ਤੋਂ ਕਿਵੇਂ ਸਿੱਖੀਏ?

ਮੈਂ ਵੀ ਸ਼ੁਰੂ ਵਿੱਚ ਬਿਲਕੁਲ zero-base ਸੀ। ਪਹਿਲਾਂ ਮੈਂ ਸਿਰਫ਼ Youtube ਉੱਤੇ ਕੁਝ [Swift ਸਿੱਖਲਾਈ ਵੀਡੀਓਜ਼](https://www.youtube.com/watch?v=4SaddkGm5yA&list=PLliocbKHJNwuLHJlVv6q5tKJxJltjBspC&index=3) ਵੇਖੀਆਂ, ਅਤੇ ਮੈਨੂੰ ਲੱਗਿਆ ਕਿ iOS ਐਪ ਬਣਾਉਣਾ ਕੋਈ ਬਹੁਤ ਜਟਿਲ ਕੰਮ ਨਹੀਂ ਹੈ। ਇਸ ਨਾਲ ਮੇਰੀ ਦਿਲਚਸਪੀ ਪੈਦਾ ਹੋਈ। ਉਸ ਤੋਂ ਬਾਅਦ, ਮੈਂ ਹੋਰ Swift ਗਿਆਨ ਸਿੱਖਣ ਦੀ ਕੋਸ਼ਿਸ਼ ਕੀਤੀ। ਉਦਾਹਰਨ ਵਜੋਂ, [Swift Playgrounds](https://apps.apple.com/cn/app/swift-playgrounds/id1496833156?mt=12) ਵਿੱਚ programming games ਅਜ਼ਮਾਉਣਾ, [Swift ਦੀ official tutorial documentation](https://docs.swift.org/swift-book/documentation/the-swift-programming-language/) ਪੜ੍ਹਨਾ, ਅਤੇ [100 Days Of SwiftUI](https://www.hackingwithswift.com/100/swiftui) ਸਿੱਖਣਾ।

ਹੁਣ ਮੁੜ ਵੇਖਣ 'ਤੇ ਲੱਗਦਾ ਹੈ ਕਿ ਅਸਲ ਵਿੱਚ [100 Days Of SwiftUI](https://www.hackingwithswift.com/100/swiftui) ਨੇ ਮੈਨੂੰ ਇੱਕ ਸਿੱਖਣ ਵਾਲਾ framework ਦਿੱਤਾ, ਜਿਸ ਦੀ ਮਦਦ ਨਾਲ ਮੈਂ SwiftUI ਨੂੰ systematic ਢੰਗ ਨਾਲ ਸਿੱਖ ਸਕਿਆ। ਪਰ ਕਿਉਂਕਿ ਇਹ ਟਿਊਟੋਰਿਯਲ ਅੰਗਰੇਜ਼ੀ ਵਿੱਚ ਸੀ, ਅਤੇ ਮੇਰੀ ਆਪਣੀ programming foundation ਵੀ ਕਮਜ਼ੋਰ ਸੀ, ਇਸ ਕਰਕੇ ਸਮਝਣ ਦੀ ਪ੍ਰਕਿਰਿਆ ਵਿੱਚ ਬਹੁਤ ਮੁਸ਼ਕਲਾਂ ਆਈਆਂ। ਇਸ ਲਈ, ਮੈਂ [ChatGPT](https://chatgpt.com/) ਅਤੇ [Claude](https://claude.ai/?redirect=claude.com) ਵਰਗੇ AI tools ਦੀ ਮਦਦ ਨਾਲ concepts ਨੂੰ ਸਮਝਿਆ।

ਸਿੱਖਣ ਦੀ ਪ੍ਰਕਿਰਿਆ ਵਿੱਚ, ਕੁਝ ਐਸੇ concepts ਆਉਣੇ ਲਾਜ਼ਮੀ ਹਨ ਜਿਨ੍ਹਾਂ ਨੂੰ ਸਮਝਣਾ ਔਖਾ ਹੁੰਦਾ ਹੈ। ਮੈਨੂੰ ਯਾਦ ਹੈ ਜਦੋਂ ਮੈਂ UserDefaults ਸਿੱਖ ਰਿਹਾ ਸੀ, ਮੈਨੂੰ ਸਿਰਫ਼ ਇਹ ਪਤਾ ਸੀ ਕਿ data ਕਿਵੇਂ save ਕਰਨਾ ਹੈ, ਪਰ ਇਹ ਨਹੀਂ ਪਤਾ ਸੀ ਕਿ data ਨੂੰ manually read ਵੀ ਕਰਨਾ ਪੈਂਦਾ ਹੈ। ਇਸ ਕਰਕੇ ਮੈਨੂੰ ਸ਼ੱਕ ਹੋਇਆ ਕਿ ਮੇਰੇ code ਵਿੱਚ ਕੋਈ ਸਮੱਸਿਆ ਹੈ। ਇੰਨਾ ਸਧਾਰਣ ਗਿਆਨ-ਬਿੰਦੂ ਵੀ ਮੈਨੂੰ ਕਈ ਦਿਨ ਬਰਬਾਦ ਕਰਵਾ ਗਿਆ।

ਇਸੇ ਤਰ੍ਹਾਂ closure, main thread, generic ਵਰਗੇ concepts, ਜਦੋਂ ਮੈਂ ਸ਼ੁਰੂਆਤੀ ਪੜਾਅ ਵਿੱਚ ਸਿੱਖ ਰਿਹਾ ਸੀ, ਤਾਂ ਮੈਂ ਉਨ੍ਹਾਂ ਨੂੰ ਡੂੰਘਾਈ ਨਾਲ ਨਹੀਂ ਸਮਝਿਆ। ਨਤੀਜੇ ਵਜੋਂ, ਜਦੋਂ ਬਾਅਦ ਵਿੱਚ practical development ਵਿੱਚ ਇਹ ਮੁੜ ਆਏ, ਤਾਂ ਮੈਨੂੰ ਦੁਬਾਰਾ ਸਮਾਂ ਅਤੇ ਉਰਜਾ ਲਗਾਕੇ ਇਹਨਾਂ ਨੂੰ ਸਿੱਖਣਾ ਪਿਆ।

ਇਸ ਲਈ, ਮੇਰੀ ਸਲਾਹ ਇਹ ਹੈ:

**1、ਬੁਨਿਆਦੀ ਗਿਆਨ ਪੂਰਾ ਕਰੋ**

ਚੀਨ ਵਿੱਚ ਇੱਕ ਕਹਾਵਤ ਹੈ: “ਹਰ ਕੰਮ ਦੀ ਸ਼ੁਰੂਆਤ ਔਖੀ ਹੁੰਦੀ ਹੈ।” ਬਿਲਕੁਲ ਸ਼ੁਰੂ ਤੋਂ Swift ਸਿੱਖਣ ਵਾਲਿਆਂ ਲਈ, ਸਿੱਖਣ ਦੇ ਦੌਰਾਨ ਕੁਝ ਐਸੇ concepts ਮਿਲਣੇ ਸਵਭਾਵਿਕ ਹਨ ਜੋ ਸਮਝਣੇ ਔਖੇ ਲੱਗਦੇ ਹਨ — ਪਰ ਅਕਸਰ ਇਹ ਇਸ ਲਈ ਨਹੀਂ ਹੁੰਦਾ ਕਿ ਉਹ concept ਆਪ ਵਿੱਚ ਬਹੁਤ ਜਟਿਲ ਹੈ, ਸਗੋਂ ਇਸ ਲਈ ਕਿ ਇਸ knowledge pyramid ਦੇ ਹੇਠਲੇ ਹਿੱਸੇ ਵਿੱਚ ਕੁਝ ਹੋਰ ਬੁਨਿਆਦੀ concepts ਹੁੰਦੇ ਹਨ, ਜਿਨ੍ਹਾਂ ਨਾਲ ਸਿਖਿਆਰਥੀ ਹਾਲੇ ਤੱਕ ਜਾਣੂ ਨਹੀਂ ਹੋਇਆ ਹੁੰਦਾ।

ਐਸੀ ਸਥਿਤੀ ਵਿੱਚ, [ChatGPT](https://chatgpt.com/) ਅਤੇ [Claude](https://claude.ai/?redirect=claude.com) ਵਰਗੇ AI tools ਦੀ ਮਦਦ ਨਾਲ, ਘਾਟ ਰਹਿ ਗਏ ਬੁਨਿਆਦੀ concepts ਨੂੰ ਸਮੇਂ ਸਿਰ ਪੂਰਾ ਕਰਨ ਬਾਰੇ ਸੋਚਿਆ ਜਾ ਸਕਦਾ ਹੈ।

**2、ਹਰ ਇਕ ਗਿਆਨ-ਬਿੰਦੂ ਨੂੰ ਸਮਝੋ**

ਸਿੱਖਣ ਦੀ ਪ੍ਰਕਿਰਿਆ ਵਿੱਚ ਸਭ ਤੋਂ ਮਹੱਤਵਪੂਰਨ ਗੱਲ ਇਹ ਹੈ ਕਿ ਹਰ ਇੱਕ knowledge point ਨੂੰ ਸਮਝਣ ਦੀ ਕੋਸ਼ਿਸ਼ ਕੀਤੀ ਜਾਵੇ, ਭਾਵੇਂ ਇਸ ਲਈ ਕੁਝ ਦਿਨ ਹੀ ਕਿਉਂ ਨਾ ਲੱਗ ਜਾਣ।

ਜੇ ਤੁਸੀਂ ਨਾ ਸਮਝ ਆਉਣ ਵਾਲੀ ਚੀਜ਼ ਨੂੰ ਛੱਡ ਦਿੰਦੇ ਹੋ, ਤਾਂ ਬਹੁਤ ਸੰਭਾਵਨਾ ਹੈ ਕਿ development ਦੇ ਅਗਲੇ ਪੜਾਅ ਵਿੱਚ ਇਹ ਮੁੜ ਸਾਹਮਣੇ ਆਵੇਗੀ ਅਤੇ ਤੁਹਾਡੀ ਤਰੱਕੀ ਨੂੰ ਰੋਕੇਗੀ। ਆਖ਼ਿਰਕਾਰ, ਤੁਹਾਨੂੰ ਫਿਰ ਵੀ ਇਸਨੂੰ ਸਿੱਖਣ ਲਈ ਸਮਾਂ ਲਗਾਉਣਾ ਹੀ ਪਵੇਗਾ। ਬਾਅਦ ਵਿੱਚ ਸਮੱਸਿਆ ਠੀਕ ਕਰਨ ਨਾਲੋਂ ਵਧੀਆ ਹੈ ਕਿ ਸਿੱਖਣ ਦੇ ਪੜਾਅ ਵਿੱਚ ਹੀ concepts ਨੂੰ ਢੰਗ ਨਾਲ ਸਮਝ ਲਿਆ ਜਾਵੇ।

ਹਰ ਇੱਕ knowledge point ਮੂਲ ਤੌਰ 'ਤੇ ਇੱਕ tool ਹੁੰਦਾ ਹੈ — ਜਿੰਨੇ ਵੱਧ tools ਤੁਹਾਡੇ ਕੋਲ ਹੋਣਗੇ, ਸਮੱਸਿਆ ਹੱਲ ਕਰਨ ਦੀ ਤੁਹਾਡੀ ਸਮਰੱਥਾ ਉਨੀ ਵੱਧ ਹੋਵੇਗੀ।

ਇਹ ਘਰ ਬਣਾਉਣ ਵਰਗਾ ਹੈ। ਜੇ ਤੁਹਾਡੇ ਹੱਥ ਵਿੱਚ ਸਿਰਫ਼ ਇੱਕ tool ਹੋਵੇ, ਤਾਂ ਤੁਸੀਂ ਮੁਸ਼ਕਲ ਨਾਲ ਜ਼ਿਆਦਾਤਰ ਕੰਮ ਕਰ ਲਓਗੇ, ਪਰ ਜਦੋਂ ਗੁਣਵੱਤਾ ਦੀ ਉੱਚ ਮੰਗ ਸਾਹਮਣੇ ਆਵੇਗੀ, ਤਾਂ ਤੁਸੀਂ ਆਪਣੇ ਆਪ ਨੂੰ ਅਸਹਾਇ ਮਹਿਸੂਸ ਕਰੋਗੇ। ਵੱਧ tools ਵਿੱਚ ਦੱਖਲ ਹਾਸਲ ਕਰਕੇ ਹੀ ਹੋਰ ਮਜ਼ਬੂਤ ਘਰ ਬਣਾਇਆ ਜਾ ਸਕਦਾ ਹੈ।

**3、ਨਿਰੰਤਰ ਸਿੱਖਣਾ ਜਾਰੀ ਰੱਖੋ**

ਸਿੱਖਣਾ ਲੰਬੇ ਸਮੇਂ ਦੀ ਲਗਾਤਾਰ ਪ੍ਰਕਿਰਿਆ ਹੈ। ਬਹੁਤ ਘੱਟ ਲੋਕ ਸਚਮੁੱਚ ਕਈ ਮਹੀਨੇ ਕੱਢ ਕੇ ਕਿਸੇ ਟਿਊਟੋਰਿਯਲ ਨੂੰ ਧਿਆਨ ਨਾਲ ਸਿੱਖਦੇ ਹਨ। ਥੋੜ੍ਹੇ ਸਮੇਂ ਦੀ ਤੀਬਰ ਮਿਹਨਤ ਨਾਲੋਂ ਲੰਬੇ ਸਮੇਂ ਦੀ ਨਿਰੰਤਰ ਮਿਹਨਤ ਹੋਰ ਮਹੱਤਵਪੂਰਨ ਹੁੰਦੀ ਹੈ।

ਭਾਵੇਂ ਤੁਸੀਂ ਹਰ ਰੋਜ਼ ਸਿਰਫ਼ ਇੱਕ ਪ੍ਰਤੀਸ਼ਤ ਹੀ ਪੂਰਾ ਕਰੋ, ਤਾਂ ਵੀ ਸੌ ਦਿਨਾਂ ਵਿੱਚ ਪੂਰਾ ਸਿੱਖਣ ਲਕਸ਼ ਪੂਰਾ ਹੋ ਸਕਦਾ ਹੈ।

## ਸਿੱਖਿਆਕ ਦ੍ਰਿਸ਼ਟੀਕੋਣ

ਇਹ ਕੋਰਸ Swift ਅਤੇ SwiftUI ਦਾ ਬਹੁਤ ਵੱਧ ਗਿਆਨ ਨਹੀਂ ਪੜ੍ਹਾਏਗਾ, ਸਗੋਂ ਸ਼ੁਰੂਆਤੀ ਸਿਖਿਆਰਥੀਆਂ ਨੂੰ Swift ਦੀ ਪ੍ਰੋਗ੍ਰਾਮਿੰਗ ਰਾਹ 'ਤੇ ਲੈ ਜਾਣ ਦੀ ਕੋਸ਼ਿਸ਼ ਕਰੇਗਾ। ਇਹ ਬਿਲਕੁਲ building blocks ਜੋੜਨ ਵਰਗਾ ਹੈ — ਟਿਊਟੋਰਿਯਲ ਤੁਹਾਨੂੰ ਜੋੜਨ ਦਾ ਤਰੀਕਾ ਦੱਸੇਗਾ, ਬਾਕੀ ਸਮਾਂ ਤੁਹਾਡਾ ਹੋਵੇਗਾ ਕਿ ਤੁਸੀਂ ਆਪਣੇ blocks ਨਾਲ ਆਪਣੀ ਰਚਨਾ ਤਿਆਰ ਕਰੋ।

ਇਹ ਪ੍ਰੋਜੈਕਟ [GitHub](https://github.com/fangjunyu1/SwiftSlim) 'ਤੇ open source ਕੀਤਾ ਗਿਆ ਹੈ, ਇਸਨੂੰ ਮੁਫ਼ਤ download ਅਤੇ ਵਰਤਿਆ ਜਾ ਸਕਦਾ ਹੈ। ਨਾਲ ਹੀ, ਇੱਕ App Store version ਵੀ ਤਜ਼ਰਬੇ ਲਈ ਉਪਲਬਧ ਹੈ।

ਅਸੀਂ ਉਹਨਾਂ ਹੋਰ ਲੋਕਾਂ ਦੀ ਮਦਦ ਕਰਨਾ ਚਾਹੁੰਦੇ ਹਾਂ ਜਿਨ੍ਹਾਂ ਕੋਲ ਤਕਨੀਕੀ ਪਿਛੋਕੜ ਨਹੀਂ ਹੈ ਪਰ ਜੋ ਆਪਣੇ career path ਨੂੰ ਬਦਲਣਾ ਚਾਹੁੰਦੇ ਹਨ, ਤਾਂ ਜੋ ਉਹ programming ਅਤੇ app development ਸਿੱਖ ਕੇ ਨਵੀਆਂ ਸੰਭਾਵਨਾਵਾਂ ਹਾਸਲ ਕਰ ਸਕਣ।
