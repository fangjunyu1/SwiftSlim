# সঞ্চয় বাক্স

এই পাঠে, আমরা একটি সহজ "সঞ্চয় বাক্স" প্রজেক্ট তৈরি করব, যেটির কার্যকারিতা সহজ, সম্পূর্ণ ইন্টারেকশন লজিক সহ, এবং নতুনদের জন্য শেখার শুরু করার জন্য অত্যন্ত উপযুক্ত।

এই প্রজেক্টের মাধ্যমে, আমরা TextField (ইনপুট বক্স) এবং border (বর্ডার) শিখব, এবং ডেটা বাইন্ডিং ($) সম্পর্কে জানব।

চূড়ান্ত লক্ষ্য হলো এমন একটি "সঞ্চয় বাক্স" প্রজেক্ট তৈরি করা যেখানে পরিমাণ ইনপুট করা যাবে, বাটন ক্লিক করে সংরক্ষণ করা যাবে এবং মোট পরিমাণ জমা হতে থাকবে।

## মোট পরিমাণ প্রদর্শন করা

প্রথমে, আমাদের ভিউতে "সঞ্চয় বাক্স"-এর মোট পরিমাণ প্রদর্শন করতে হবে।

ContentView-এ, "সঞ্চয় বাক্স"-এর মোট পরিমাণ সংরক্ষণ করার জন্য একটি ভেরিয়েবল ঘোষণা করুন।

```swift
@State private var amount = 0
```

এরপর, এই পরিমাণ প্রদর্শন করতে Text ব্যবহার করুন:

```swift
Text("\(amount)")
	.font(.largeTitle)
	.fontWeight(.bold)
```

এখানে, `.font` এবং `.fontWeight` ব্যবহার করে ফন্টের আকার এবং মোটামুটি ওজন নির্ধারণ করা হয়েছে।

```swift
struct ContentView: View {
    
    @State private var amount = 0
    
    var body: some View {
        VStack {
			Text("\(amount)")
			.font(.largeTitle)
			.fontWeight(.bold)
		}
    }
}
```

যখন `amount` পরিবর্তিত হয়, Text-এ প্রদর্শিত মোট পরিমাণও স্বয়ংক্রিয়ভাবে রিফ্রেশ হবে।

## পরিমাণ ইনপুট করা

"সঞ্চয় বাক্স" মোট পরিমাণ প্রদর্শনের পাশাপাশি, ব্যবহারকারীদের জমা বা উত্তোলনের পরিমাণ "ইনপুট" করতে দেওয়ার প্রয়োজন, এর জন্য SwiftUI-এর ভিউ কন্ট্রোল TextField ব্যবহার করতে হবে।

### TextField

TextField হলো SwiftUI-এ কন্টেন্ট ইনপুট করার জন্য একটি ভিউ কন্ট্রোল, সাধারণত একক লাইনের ইনপুটের জন্য ব্যবহৃত হয়।

TextField মূলত দুটি বাইন্ডিং ফরম্যাট প্রদান করে:

**১. স্ট্রিং টাইপের সাথে বাইন্ডিং (String)**

```swift
TextField("Placeholder", text: $text)
```

নাম, শিরোনাম ইত্যাদি টেক্সট ইনপুট কন্টেন্টের জন্য উপযুক্ত।

**২. সংখ্যা টাইপের সাথে বাইন্ডিং (Int/Double)**

```swift
TextField("Amount", value: $number, format: .number)
```

বয়স, বেতন, পরিমাণ ইত্যাদি সংখ্যা ইনপুট কন্টেন্টের জন্য উপযুক্ত।

`format: .number` মানে এই ইনপুট বক্সটি "সংখ্যা ফরম্যাট" অনুযায়ী পার্স এবং প্রদর্শন করবে।

#### টেক্সট প্লেসহোল্ডার

TextField-এর প্রথম প্যারামিটার হলো প্লেসহোল্ডার (Placeholder), যা ব্যবহারকারীকে ইনপুট কন্টেন্টের জন্য প্রম্পট করে:

```swift
TextField("input your name", text: $text)
```

যখন ইনপুট বক্স খালি থাকে, তখন একটি ধূসর প্রম্পট টেক্সট প্রদর্শিত হয়।

![textfield](../../RESOURCE/014_textfield.png)

#### ডেটা বাইন্ডিং

TextField ইনপুট কন্টেন্ট স্থায়ীভাবে সংরক্ষণের দায়িত্ব নেয় না, ডেটা পরিচালনার জন্য বাইন্ডিং ভেরিয়েবলের প্রয়োজন।

```swift
@State private var text = ""
TextField("Placeholder", text: $text)
```

ব্যবহারকারীর ইনপুট করা কন্টেন্ট সরাসরি `text` ভেরিয়েবলে সংরক্ষিত হবে, TextField শুধুমাত্র ইনপুট ইন্টারফেসের দায়িত্ব নেয়।

`$` বাইন্ডিং (Binding) বোঝায়:

```swift
$text
```

এটি একটি সাধারণ ভেরিয়েবল নয়, বরং এটি একটি Binding টাইপ।

এর কাজ হলো ভিউ এবং ডেটার মধ্যে সংযোগ স্থাপন করা:

যখন ব্যবহারকারী TextField-এ কন্টেন্ট ইনপুট করেন, `text` স্বয়ংক্রিয়ভাবে আপডেট হয়; যখন `text` পরিবর্তিত হয়, TextField-ও সিঙ্ক্রোনাইজ হয়ে আপডেট হয়।

উদাহরণ:

```swift
struct ContentView: View {
    @State private var text = ""
    
    var body: some View {
        VStack {
			Text("text:\(text)")
        	TextField("input your name", text: $text)
            	.frame(width: 150)
		}
    }
}
```

TextField-এ কন্টেন্ট ইনপুট করলে, উপরের Text রিয়েল-টাইমে সিঙ্ক করে প্রদর্শন করবে।

![textField](../../RESOURCE/014_textfield1.png)

### সংখ্যা টাইপের সাথে বাইন্ডিং

যখন সংখ্যা ইনপুট করার প্রয়োজন হয়, তখন সংশ্লিষ্ট টাইপের ভেরিয়েবলের সাথে বাইন্ড করতে হবে:

```swift
@State private var number = 0

TextField("Amount", value: $number, format: .number)
```

যখন ব্যবহারকারী পরিমাণ ইনপুট করেন, TextField ইনপুট কন্টেন্ট পার্স করবে এবং স্বয়ংক্রিয়ভাবে `number`-এর মান আপডেট করবে।

## ইনপুট বক্স যোগ করা

TextField-এর মৌলিক ব্যবহার বোঝার পর, আমরা এটি "সঞ্চয় বাক্স" প্রজেক্টে প্রয়োগ করব।

"সঞ্চয় বাক্স"-এ "পরিমাণ" ইনপুট করতে হবে, তাই আমরা "সংখ্যা টাইপের সাথে বাইন্ডিং (Int/Double)" পদ্ধতি ব্যবহার করব, এবং ব্যবহারকারীর ইনপুট করা পরিমাণ সংরক্ষণ করার জন্য একটি নতুন `number` ভেরিয়েবল যোগ করব:

```swift
struct ContentView: View {
    
    @State private var amount = 0
    @State private var number = 0
    
    var body: some View {
        VStack {
			Text("\(amount)")
            	.font(.largeTitle)
            	.fontWeight(.bold)
        	TextField("Amount", value: $number, format: .number)
		}
    }
}
```

এখন, TextField ডিফল্টরূপে `number`-এর মান (ডিফল্ট ০) প্রদর্শন করে।

যখন ইনপুট কন্টেন্ট মুছে ফেলা হয়, তখন প্লেসহোল্ডার "Amount" প্রদর্শিত হবে।

এটি আবারো প্রমাণ করে যে TextField বাইন্ডিং ভেরিয়েবলের মান প্রদর্শন করে, নিজে কন্টেন্ট সংরক্ষণ করে না।

### ইনপুট বক্সের আকারের সমস্যা

বর্তমান ভিউতে, আমরা দেখব যে TextField-এ কোনো বর্ডার নেই, শুধু একটি ০ প্রদর্শিত হয়, কন্টেন্টটি বাম-সারিবদ্ধ দেখায়।

আমরা ব্যাকগ্রাউন্ড রঙ যোগ করে এটি যাচাই করতে পারি:

### নির্দিষ্ট প্রস্থ নির্ধারণ করা

যদি আমরা ইনপুট বক্সটি আরও কম্প্যাক্ট করতে চাই, তাহলে `frame` ব্যবহার করে এর প্রস্থ সীমিত করা যায়:

```swift

```

### বর্ডার এবং ইনার প্যাডিং যোগ করা

যেহেতু TextField-এ ডিফল্টরূপে কোনো বর্ডার নেই, আমরা ম্যানুয়ালি স্টাইল যোগ করতে পারি:

```swift
TextField("Amount", value: $number, format: .number)
    .frame(width: 100)
    .padding(5)
    .border(Color.black, width: 1)
```

এখন, এই ইনপুট বক্সটি একটি "স্ট্যান্ডার্ড ইনপুট বক্স"-এর মতো দেখাচ্ছে।

## বর্ডার (Border)

SwiftUI-এ, `border` ভিউতে বর্ডার যোগ করার জন্য ব্যবহৃত হয়।

মৌলিক ব্যবহার:

```swift
border(Color.black, width: 1)
```

এখানে, `Color.black` বর্ডারের রঙ বোঝায়, এবং `width` বর্ডারের পুরুত্ব বোঝায়।

TextField-এ, আমরা ব্যবহার করি:

```swift
TextField("Amount", value: $number, format: .number)
    .frame(width: 100)
    .padding(5)
    .border(Color.black, width: 1)
```

এটি ইনপুট বক্সে কালো রঙের, ১ pt প্রস্থের বর্ডার যোগ করে।

অন্যান্য রঙও চেষ্টা করা যায়, যেমন `Color.blue`, `Color.green`, অথবা বর্ডার ঘন করতে `width: 2` ব্যবহার করা যায়।

## সঞ্চয় বাটন

এখন, আমাদের কাছে মোট পরিমাণ প্রদর্শনের জন্য একটি `amount` ভেরিয়েবল, পরিমাণ ইনপুট করার জন্য একটি TextField, এবং ইনপুট বক্সের সাথে বাইন্ড করা `number` ভেরিয়েবল রয়েছে।

বর্তমান কোড নিম্নরূপ:

```swift
struct ContentView: View {
    
    @State private var amount = 0
    @State private var number = 0
    
    var body: some View {
        VStack {
			Text("\(amount)")
            	.font(.largeTitle)
            	.fontWeight(.bold)
        	TextField("Amount", value: $number, format: .number)
            	.frame(width: 100)
            	.padding(5)
            	.border(Color.black, width: 1)
		}
    }
}
```

### বাটন যোগ করা

এরপর, আমাদের একটি বাটন যোগ করতে হবে, সঞ্চয় অপারেশন ট্রিগার করার জন্য।

```swift
Button("Save") {
    
}
```

### সঞ্চয় লজিক বাস্তবায়ন

আমরা চাই যে ব্যবহারকারী পরিমাণ ইনপুট করলে, বাটন ক্লিক করলে পরিমাণ স্বয়ংক্রিয়ভাবে মোট পরিমাণে জমা হবে।

```swift
Button("Save") {
    amount += number
}
```

এখানে "যৌগিক অ্যাসাইনমেন্ট অপারেটর" ব্যবহার করা হয়েছে, ব্যবহারকারীর ইনপুট করা `number` পরিমাণকে `amount` মোট পরিমাণে যোগ করতে।

### ইনপুট পরিমাণ রিসেট করা

এখন একটি সমস্যা আছে, প্রতিবার বাটন ক্লিক করলে ইনপুট বক্সের পরিমাণ মুছে যায় না।

যখন ব্যবহারকারী ১০ ইনপুট করেন এবং সেভ ক্লিক করেন, ইনপুট বক্সে তখনও ১০ সংরক্ষিত থাকে, যা পরবর্তী ইনপুটের সঞ্চয় পরিমাণকে প্রভাবিত করবে।

এর কারণ হলো TextField `number`-এর সাথে বাইন্ড করা, আমরা বাটন ক্লিক করার সময় শুধুমাত্র `amount` পরিবর্তন করেছি, `number` পরিবর্তিত হয়নি, তাই ইনপুট বক্সে এখনও পুরনো মান প্রদর্শিত হচ্ছে।

আমাদের একটি ক্লিয়ার লজিক যোগ করতে হবে, বাটন ক্লিক করার পর প্রথমে সঞ্চয়ের পরিমাণ যোগ করবে, তারপর ইনপুট করা পরিমাণ রিসেট করবে:

```swift
Button("Save") {
    amount += number
	number = 0
}
```

এখন, ব্যবহারকারী পরিমাণ ইনপুট করে বাটন ক্লিক করলে, ইনপুট পরিমাণ মোট পরিমাণে যোগ হয়। `number` ০-তে রিসেট হয়, যাতে ব্যবহারকারী পরবর্তীবার ইনপুট করতে পারেন।

এভাবে আমরা "সঞ্চয় বাক্স"-এর সমস্ত সঞ্চয় লজিক বাস্তবায়ন করেছি।

## ভিউ অপ্টিমাইজ করা

ফিচার সম্পন্ন হওয়ার পর, আমরা ইন্টারফেসে সহজ অপ্টিমাইজেশন করতে পারি, ব্যাকগ্রাউন্ড ছবি এবং বাটন স্টাইল যোগ করে।

বাটন স্টাইল যোগ করুন:

```swift
.buttonStyle(.borderedProminent)
```

ব্যাকগ্রাউন্ড ছবি যোগ করুন:

```swift
.background {
    Image("1")
}
```

এখন, আমরা সম্পূর্ণ "সঞ্চয় বাক্স"-এর ডেভেলপমেন্ট কাজ সম্পন্ন করেছি।

## সম্পূর্ণ কোড

```swift
struct ContentView: View {
    
    @State private var amount = 0
    @State private var number = 0
    
    var body: some View {
        VStack {
            Text("\(amount)")
                .font(.largeTitle)
                .fontWeight(.bold)
            TextField("Amount", value: $number, format: .number)
                .frame(width: 100)
                .padding(5)
                .border(Color.black, width: 1)
            Button("Save") {
                amount += number
                number = 0
            }
            .buttonStyle(.borderedProminent)
        }
        .background {
            Image("1")
        }
    }
}
```

অথবা "সঞ্চয় বাক্স" সিমুলেটর বা রিয়েল ডিভাইসে চালু করে, আমাদের ডেভেলপ করা "সঞ্চয় বাক্স" অ্যাপ অনুভব করতে পারি।

### ডিবাগ আউটপুট

লজিক সঠিক কিনা যাচাই করার জন্য, আমরা বাটনে ডিবাগ লজিকও যোগ করতে পারি, ইনপুট করা পরিমাণ সঠিক কিনা যাচাই করতে:

```swift
Button("Save") {
    print("---Saving---")
    amount += number
    number = 0
    print("amount:\(amount)")
    print("number:\(number)")
}
```

প্রতিবার অর্থ সঞ্চয় করে বাটন ক্লিক করলে, আমরা সংশ্লিষ্ট ডিবাগ আউটপুট দেখতে পাব, কোড লজিকে কোনো সমস্যা আছে কিনা তা পরীক্ষা করতে।

```
---Saving---
amount:11
number:0
```
